　　“是啊。”叶少阳转过头，说道：“不可伤人！”
<br/>　　一个士兵嬉笑道：“不伤人，夫人这细皮嫩肉的，怎能随便伤了……”
<br/>　　“砰！”话没说完，人已经倒着飞出去，躺在地上鬼号起来，双眼瞪的老大，不可思议地望着侯爷夫人——她还笑吟吟地站在窗前，这兵士想了半天，才想起方才好像是这女人踢了自己一脚……边上那几个兵士也都愣了一会，然后尝试着靠过去。
<br/>　　砰砰砰！
<br/>　　几人相继被撞飞。
<br/>　　“她会法术，这下我可看清楚了，她会法术！”一个兵士叫嚷起来。
<br/>　　都统大惊，急忙让手下去找白虎堂。
<br/>　　这时候隔壁房间也传来一阵打斗声，几个捉拿侯爷“同党”的士兵被撞飞出来，倒在地上之后，一个个表情抽搐，连同这边房间的几个，都相继变成了邪物。
<br/>　　“被鬼占了身子了！”
<br/>　　这都统大叫一声，命令手下赶紧去报告白虎堂。
<br/>　　这时候那些完成蜕变的邪物已经开始戾气发作，见人就扑上去，抓人是没指望了，都统只好招呼剩余手下后退，守在客栈门外。
<br/>　　一个个拿出了随身的长矛，对着大门站成一排，都统一个劲叫掌柜的把别的门都关好。这时候里头的客人还不知道怎么回事，就听见楼上一阵响动。
<br/>　　蛤蟆精、蜘蛛精一类的邪物顺着楼梯呼啦一下就来下来了，本来他们是互相打的，从楼上打到楼下，结果看到楼下有这么多人，不少邪物就开始攻击人类，一时间鬼哭狼嚎的。
<br/>　　还有一些试图冲向守在门外的士兵，都统一声令下，所有士兵手拿长枪一起捅过去，他们的兵器上都是白虎堂的高人开光过的，上面刻有简单的道纹，算是普通人能用的低级法器。
<br/>　　因为门很小，长矛又很多，形成了一条枪阵，一时间将攻过来的邪物挡了回去。
<br/>　　但是客栈里就像是丧尸蔓延似的，那些住户和客栈的工人，除了被杀的，活着的也都变成了邪物，后来没人类可杀了，一边自相残杀，一边朝出口涌过来。
<br/>　　这些士兵眼看着挡不住了，都统只能命令撤退，回衙门找人去。
<br/>　　这些邪物冲到路上，见人就啃，当场就咬伤了不少人。
<br/>　　叶少阳一行人集中在一间房里，房门大开，瓜瓜一个人守着，只要有邪物上来，都被他撞飞出去，后来客栈里邪物都跑出去了，也没有新的过来了。
<br/>　　叶少阳挑开窗户朝街上看。
<br/>　　那些邪物的涌入，在街上造成了混乱，不少人受伤，但这些“居民”显然不是第一次遇到这种情况，就近奔逃到建筑物里，把门窗关好。
<br/>　　那些邪物就想撞门和窗子进去，结果这些门窗上都贴着对联，邪物一撞上去，对联上的字迹立刻亮起灵光，将邪物弹开。
<br/>　　家家户户的情况都一样，叶少阳看在眼里，猜测这些对联应该都是用朱砂笔画的，说是对联，其实就是大号的灵符，心中感慨，圣灵会为了保证他们在这个世界的统治，还真是够费心的。
<br/>　　这些邪物冲不到路边的建筑里去，就只能顺着街道一路狂奔，就在叶少阳站着看的工夫，街道对面有一队士兵列队奔来。
<br/>　　这些士兵都穿着厚厚的白色盔甲，头戴红缨，在几个道士和尚的率领下，从四面八方赶来，到了一定距离就站住，然后拉弓射箭。
<br/>　　箭射进那些邪物身上，伤口处立刻冒出白汽，邪物立刻就叫起来，双手去拔箭，但手抓在上面也是一股子白汽冒出来。有修为深一些的还能撑着把箭拔出，差一点的就在地上哀嚎着翻滚起来，更多的箭像雨点一样飞来，很多邪物就这样被射死了。
<br/>　　一轮箭雨飞过，这些邪物非死即伤，而且都被赶到了比较集中的地方，那些士兵就拔出刻下符文的长枪佩剑，列着方阵，从四面合围，不断压迫着这些邪物的空间，凡是反击的，都寡不敌众，被活活砍死，而那些和尚道士则盯着其中几个修为深厚些的，各执法器，包围进攻，不一会工夫也都消灭了。
<br/>　　几十个邪物，就这样被军队给碾压了，地上到处都是邪物留下的残肢断臂，以及各种颜色的血。
<br/>　　法师们用石灰、糯米一类的法药泡水，在地上清洗着，将那些残肢断臂融化，用不了一时半会，地上什么痕迹都没了，仿佛什么都没发生过。
<br/>　　队伍里一个士兵吹起了一个牛角形状的东西，发出苍凉而绵长的乐声，跟着队伍列着方阵离去。又过了一会，有些建筑打开了窗户，里面的人们四下张望，见一切都结束了，便各自出门，来到街上，该干啥干啥。一场突如其来的变故，就这么被镇压了下去。
<br/>　　“咱们可能是被遗忘了，那个都统什么的，大概以为我们被杀死了。”腾永清在叶少阳身边说道。
<br/>　　“厉害啊！这圣灵会倒是有手段的。”叶少阳有些失望，他本来这件事会酿成一场大乱，没想到这么快就被镇压了。
<br/>　　不过转念一想，这个结果，倒也是可以预料的，毕竟这里人人都是邪物变的，一旦有一个现出真身，到处咬人，假如不管不顾，很快就会一传十十传百，整个世界就会沦陷。
<br/>　　圣灵会当然不会让这种事发生。
<br/>　　叶少阳笑道：“我之前想靠这个制造混乱，原来也没那么容易。”
<br/>　　碧清道：“还是敌我实力悬殊太大，我们如果出手，肯定不一样的。”
<br/>　　瓜瓜道：“老大我们现在怎么办？”
<br/>　　“能怎么办，先换个地方住下来，总不能让他们抓到大牢里去。”
<br/>　　一行人收拾了出门去，为了避免显眼，连马车也不要了，跟着人群混到城外去（叶少阳觉得，城外怎么也比城内要安全得多），瓜瓜提议可以到附近的县城找地方住，叶少阳提醒，这里又不是人间，除了几座城市是真的，其余地方都是假的，根本就没什么县城。
<br/>　　碧清提议，这京城里有这么多房子，大多数大概都是摆设，不可能有这么多人住，如果找到空房子，倒是可以偷偷进去住。于是叶少阳把这个任务交给瓜瓜。
