　　中级功法区域，此处摆放的都是天墉城，或者其他门派的中级功法，剑法居多。
<br/>　　“叮，《疾风剑》收录中，价值1000掠夺积分。”
<br/>　　“叮，《晦明剑》收录中，价值1500掠夺积分。”
<br/>　　“叮，《玄真剑》收录中，价值3000掠夺积分......”
<br/>　　当逛完中级功法区域，慕清霄算是满载而归，直接就获得了五万掠夺积分的奖励，虽不是太多，但也算不得少。
<br/>　　而且，系统收录的功法，他都能修炼，哪怕自己不用，给自己的妻子用也好啊。
<br/>　　其中，有几部价值颇高的中级剑法，倒是令慕清霄记了下来，价值就越高，威力越大，威力越小的，价值自然就越低。
<br/>　　逛完第二层，慕清霄就向第三层走去，刚踏进第三层，其中没有一丝声音，也没有任何一名弟子，安静无比。
<br/>　　但下一秒，他的视线就被坐在中央的一名老者所吸引。
<br/>　　老者穿着一袭蓝色长袍，满头白发，下巴上白花花的胡子直接拖到胸口，从他雄浑的气息上来看，绝对是练虚境。
<br/>　　听说妙法堂长老，凝虚真人是一名练虚境七重的强者，估计就是眼前的这位了吧。
<br/>　　没想到，第三重居然直接由凝虚真人把守，可见天墉城对此地的重视。
<br/>　　慕清霄打量凝虚真人的同时，后者也睁开双眼打量着他，浑浊的双眼闪烁着精芒，似乎在看什么宝物一般。
<br/>　　只见凝虚真人站了起来，面露微笑的走上前来，道：“想必，阁下就是刚受封的第十一代凝丹长老，九霄真人吧，年轻有为，前途无量啊。”
<br/>　　“真人不敢当，前辈就是凝虚真人吧。”
<br/>　　正所谓伸手不打笑脸人，凝虚真人如此客气，慕清霄也当以礼待之，微微拱手。
<br/>　　“不错，正是老夫，听闻凝丹长但丹道上的成就极高，年纪轻轻就快迈入练虚境，是来阅览剑技的？”
<br/>　　“在下对天墉城的剑技颇感兴趣。”
<br/>　　闻言，凝虚真人捋了捋白花花的胡子，笑道：“天墉城乃是天下第一大派，这里的剑技自然最强，相信不会让你失望的。”
<br/>　　“希望如此。”
<br/>　　说着，慕清霄又道：“劳烦凝虚真人替在下登记。”
<br/>　　“其中的剑技你可以随意阅览，登记就不必了，不过是走个形势，我们作为一堂长老，没有如此多的约束。”
<br/>　　说着，凝虚真人就回到座位上，闭目养神起来。
<br/>　　微微点头，慕清霄就向对面的书架走去，第三层只摆放着一个书架，书架上空荡荡的，只有寥寥几部剑法。
<br/>　　像高级功法或者剑法，都是一个门派立命之根本，不像初级与中级功法，哪怕外传也没有多大的损失。
<br/>　　但高级功法或剑法不同，每一部都拥有着恐怖的威力，只有本门精英真传弟子，或者长老能够修炼。
<br/>　　高级功法一律不得外传，否则门规处置。
<br/>　　脚步轻缓的走到书架前，慕清霄随手拿起其中一本剑法，脑海中就立刻受到系统的提示声。
<br/>　　“叮，《空明剑》收录中，价值50000掠夺积分。”
<br/>　　《空明剑》天墉城强力剑技之一，紫胤真人剑法之所成，空明幻虚剑初级形态。
<br/>　　此剑技幻化出无数剑影，从天纵贯而下，造成两次伤害，第二次伤害较大，修习者需要有一定修为。
<br/>　　望着手中的剑技，慕清霄满意的点了点头，终于找到了不错的剑技，正巧自己现在没有剑技在身。
<br/>　　凭他对剑的领悟，配上魂剑威能，更能发挥出剑技的威力。
<br/>　　系统已经收录《空明剑》，慕清霄也就没有阅览的必要，等回到住处在慢慢阅览也不迟，于是便将秘籍放回原处。
<br/>　　随后，再次拿起另一本剑法秘籍。
<br/>　　“叮，《太虚剑》收录中，价值50000掠夺积分。”
<br/>　　再次获得五万掠夺积分，慕清霄星眸中闪过一丝喜意，掠夺积分自然是越多越好，就好比普通人，谁会嫌钱多？
<br/>　　这也是同样的道理。
<br/>　　《太虚剑》天墉城强力剑技之一，同样是紫胤真人剑法之所成。
<br/>　　此剑技将剑气全部汇于一体，凝成一道巨大剑柱纵贯对方，随即四散开来，化为无数虚幻剑影横扫对手，两次攻击所造成的威力极其恐怖，修为较低者足以致命。
<br/>　　慕清霄微微一笑，看都没看，直接将《太虚剑》放回远处，也正是因此，引起了凝虚真人的注意。
<br/>　　“凝丹长老，你确定是来阅览剑技的，你可不要骗老夫？”
<br/>　　嘴上虽这么说，但凝虚真人早就在心中将慕清霄骂了个狗血淋头，有你这么阅览剑技的吗？拿起来看都不看一眼？
<br/>　　慕清霄一怔，倒是将凝虚真人给忘了，微微一笑，道：“这两部剑技，掌门已经传授于我，所以不必再看。”
<br/>　　随便扯了个谎话敷衍了一下，其实掌门根本没传他剑法，凝虚真人不信也得信，他总不能去质问掌门吧，除非他作死。
<br/>　　眼角抽了抽，看到他那满不在乎的模样，凝虚真人深吸了口气，微微闭上双眼，眼不看为净。
<br/>　　“叮，《乱剑诀》收录中，价值50000掠夺积分。”
<br/>　　《乱剑诀》天墉城强力剑技之一，素有夺命残身剑法之说，威力何其霸道！
<br/>　　此技凝聚周身真气，变换出无数剑雨，纵穿对手，威力极强，修微弱者瞬间致残，即使侥幸从剑雨中生还也将四肢僵硬，动弹不得，坐以待毙。
<br/>　　将三套剑技收入囊中，慕清霄非常满意，有了这些剑技，与敌人对战时，哪怕不变成黑凤也能应战。
<br/>　　毕竟，战技对修士的重要性，如同武器一样，如果手无寸铁的修士，与手握武器的修士对战，占据上风的肯定是手中有武器的。
<br/>　　战技也是同样的道理。
<br/>　　看了看书架下面空荡荡的，除了三套剑法，并无其他功法，慕清霄略有失望，这可都是掠夺积分啊。
<br/>　　瞥了眼闭目养神的凝虚真人，慕清霄走上前来，道：“前辈，在下想去第四层看看。”
<br/>　　闻言，凝虚真人睁开双眼，盯着他愣愣出神。
<br/>　　“凝丹长老，你可知道贪多嚼不烂的道理？这些剑法可都是紫胤真人剑法所成，哪怕其中任何一本，没有三五年都无法参透。”
