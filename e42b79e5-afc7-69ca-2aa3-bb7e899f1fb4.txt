　　“色不异空，空不异色，色即是空，空即是色，受想行识，亦复如是。”
<br/>　　准提圣人一出手，便是非同凡响，虚空波动，一切皆空，万法虚无，众人只感觉一股神秘的力量透过虚空，影响着自己的思想，让自己放弃一切抵抗，顺从佛的安排。观战的众人都是如此，更何况万劫魔尊了。
<br/>　　准提圣人出手便是为了对付万劫魔尊，所以大部分法力都是对着他去的。众人猜测万劫魔尊现在一定是竭力抵抗着这股神秘的力量，但当然人们仔细看去，却是目瞪口呆，因为万劫魔尊仿佛根本不受影响，他嘴上挂着冷笑，手中魔刀向着准提圣人一道砍去。
<br/>　　“空你妹啊！万事皆空，你还在这儿干什么？”
<br/>　　万劫魔尊心中对此不屑一顾，佛门讲究四大皆空，一切皆空，既然这一切都是空的，那么他怎么还会在这里。
<br/>　　这就是佛家与道家的区别，道家讲究修真，去伪求真，将假的修成真的，而佛家却讲究修假，修空，他们相信万法皆空，相信来世，今世修善果，来世成佛。
<br/>　　一道巨大的黑色刀芒，仿佛划破天际，跨过虚空，斩向准提圣人。
<br/>　　准提圣人的道法被万劫魔尊破了，顿时出现一丝停顿，只见一道宏大的刀芒向他斩来，他心念一动，顿时在身前形成一道坚固的防御，之后一掌轰出，将刀芒轰碎。
<br/>　　“阿弥陀佛，施主罪孽深重，贫道要除魔卫道。”
<br/>　　接引圣人此刻也站了出来，对着万劫魔尊想要出手，他伸手对着万劫魔尊便是轻轻一点，顿时，有一道灵光闪现，紧接着，无数光汇聚而来，无量光辉，向着万劫魔尊射去，光芒普照的地方，万物寂灭。
<br/>　　“轰！”
<br/>　　就在此时，同样一道青色的造化之光从远处破空而来，对上接引的寂灭之光，两者相撞，发生剧烈的爆炸声，湮灭于虚空。
<br/>　　“接引道友，你身为圣人，又何必以多欺少呢？不如我们来论道一番，如何？”
<br/>　　紧接着，一道悦耳动听的声音传入到众人的耳中，像是惊起万丈滔天巨浪，是女娲圣人出手了。
<br/>　　“女娲师妹，这是为何？”
<br/>　　接引看向女娲，不明白女娲与眼前这位万劫魔尊到底是何关系，为何会出手帮他，大有女娲不说出一个道理来，他就要出手的意思。
<br/>　　“本宫曾欠下万劫魔尊一份因果，还请师兄不要插手为好。”女娲淡淡道。
<br/>　　“原来如此，若是贫道偏要出手呢？”
<br/>　　接引继续说道，关乎到魔祖的传承，接引可不是那么容易放弃的。
<br/>　　“那本宫便要前来试试师兄的高招了。”女娲面色不变，淡淡道。
<br/>　　“哦，那么，师妹，我们去混沌一战，如何？”
<br/>　　女娲毫不犹豫的便转身，向着混沌飞去，紧接着，接引圣人也向着混沌飞去。
<br/>　　洪荒众人面面相觑，一头雾水，怎么回事？就连女娲圣人都出手了，眼前这人到底是何人，竟会认识女娲圣人？
<br/>　　不久，顿时之处就传来剧烈的震动声，惊天动地，混沌暴动，毁天灭地，众人只感觉到一个个神通玄光爆发，世界开始震动。
<br/>　　显然，女娲已经与准提圣人交手了。只是不知道孰强孰弱？
<br/>　　“原来你的后手便是女娲师妹啊，怪不得有恃无恐。”准提圣人恍然大悟，之后继续冷笑道：
<br/>　　“现在女娲师妹走了，我看你还有什么招数。”
<br/>　　“是吗？”
<br/>　　万劫魔尊冷笑一声，毫不在意，明亮的眼神冷静的如平静无波的湖水。一柄黑色的魔道在他身前横着，周身魔气纵横，给人的感觉，就仿佛一头平静的魔神，好似亘古以来就屹立在这里一般。
<br/>　　一道磅礴无边的压抑气息传来，准提圣人深深的吸了一口气，眼前这人眼神太平静了，好似还有什么他不知道的底牌，让人拿捏不准。他手中的佛珠不由自主的转动起来，发出一阵阵梵音，就好似可以涅灭一切世界。
<br/>　　万劫魔尊抡起手中的魔刀，便向着准提圣人砍去，他可没有被动挨打的兴趣，管他是什么人，圣人也好，神人也罢，直接动手才符合他的性格。
<br/>　　“轰……”
<br/>　　一道噬天裂地般扭曲的黑色刀芒劈断了准提前方的空间，空间破碎，形成风暴，在准提圣人面前爆发起来，席卷向他。
<br/>　　没有华丽的神通，没有复杂的手印，有的只是简简单单的一刀。
<br/>　　准提圣人冷哼一声，手中念珠化成一个圆圈，这个圆圈有着无数个小世界组成，每一个小世界都环绕着强悍的释道道韵。这些世界连在一起，形成浩瀚虚空中最坚硬的阻拦。
<br/>　　滚滚的释道道韵和低沉的梵音在这一方天地环绕，这一刻只要被这梵音与道韵扫过的修士，都好像有一种道韵灌体的明悟，恨不得立即坐在地上感悟这无上道韵。只要能够感受到这无上道韵，便能够立地成佛，成就大造化一般。
<br/>　　万劫魔尊面无表情，这道韵与梵音冲击在他周身魔域之外，就好似微风拂过，留不下任何的东西。
<br/>　　论心境，万劫魔尊为元始的恶尸，早已经是混元中期境界了，比起只是混元初期的准提圣人，高了不知多少，他那点心境神通又怎么影响到万劫魔尊的心境呢，更何况，万劫魔尊修炼的是魔祖的《万劫魔典》，这部功法本就是佛家功法的克星，当然，佛家功法也是这本功法的克星，只是分孰强孰弱的问题罢了。
<br/>　　万劫魔尊如今的一身实力，经过几次大劫的修炼，早已经超出了准圣圆满，介于混元与准圣圆满之间，乃是半步混元，虽然一时战胜不了混元，但混元想要打败他，也是不可能。
<br/>　　“轰！”
<br/>　　那狂暴的道韵撞击在一起，掀起无量风暴，空间坍塌，地水火风涌动，一切都陷入了混沌之初。
<br/>　　准提圣人的脸色突然变化，他发现自己小看了眼前这人，眼前这人可以说已经半只脚踏入了混元境界，这已经非常可怕了，要知道，就连吞噬了无数功德的冥河也没有达到这个境界。他连忙收拾起心中那些杂念，开始认真对待起眼前此人。
<br/>　　“万道森罗，混洞为墟，吞天噬地，以葬神魔！”
<br/>　　万劫魔尊抓到一点先机，自然是不会放过，他继续强攻。以万劫魔典为根本，施展万道森罗中的混洞大道神通。
<br/>　　他面前出现一个巨大的混洞，缓缓旋转，形成一股逆天的吞噬力，吞噬之中，散发着无尽魔气，能够影响人的心神，更能吞噬人的灵魂，天地间一切能量都被掠夺过来，空间变得空洞起来。
<br/>　　一个巨大的黑洞，散发着混沌般的魔气，以一种无可匹敌的魔力，吞向准提圣人。
<br/>　　准提圣人顿时脸色大变，他从这个混洞之中感受到深深的威胁，同时，他也确定了，眼前这人的确是得到了魔祖的传承，因为这万道森罗神通，便是魔祖常用的手段。
<br/>　　“菩提本无树，明镜亦非台，
<br/>　　本来无一物，何处惹尘埃？”
