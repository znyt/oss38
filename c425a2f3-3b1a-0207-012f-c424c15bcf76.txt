有很多名门淑媛围在夏小蝶的身边，各种艳羡奉承，夏小蝶那张漂亮的小脸开心得意的能滴出蜜来了。
<br/>
<br/>        这时夏小蝶看到了夏夕绾，她迅速上前，“夕绾，你来了？刚才我还好担心你不来，不能亲眼见证我这么幸福的时刻。”
<br/>
<br/>        那些名门淑媛也走了过来，
<br/>
<br/>        小蝶，你心太善了，夏夕绾以前可是苏少的未婚妻，你邀请她过来就不怕她嫉妒到砸场子？
<br/>
<br/>        她夏夕绾已经嫁给幽兰苑那个鬼夫了，那个鬼夫也不知道怎么折磨她的，说不定哪一天两眼一闭就让她成为寡妇了，这真是人同命不同，乡下来的土包子永远比不上真正的千金大小姐，我们小蝶已经是苏少夫人了。
<br/>
<br/>        还是苏少的眼光好，早就抛弃了那个土包子，苏少跟我们小蝶就是一对金童玉女。
<br/>
<br/>        听着这一搭一唱，夏小蝶嗔怪的看了那些人一眼，“你们不要这样说夕绾了，其实夕绾够可怜了。”
<br/>
<br/>        夏夕绾就看着这些人演戏，她没有生气，只是觉得很好笑，趋炎附势是小人的本能，夏小蝶如今成了四大豪门苏希的未婚妻，这场盛世婚礼又彰显了苏希对她的宠爱，所以她有骄傲的资本。
<br/>
<br/>        夏小蝶这么急切让她来参加订婚party，就是给她难堪的。
<br/>
<br/>        这时李玉兰也走了过来，最近李玉兰和夏振国夫妻感情不和，很是心烦，不过她的女儿跟苏希订婚可给她带来了无上的荣光，所以李玉兰今天也是容光焕发，一身定制旗袍外面披着海蓝色的皮草，雍容华贵里不失女人的风韵。
<br/>
<br/>        “夕绾，你来了，今天小蝶订婚来的可都是海城的名媛们，你不是想融入进去，认识大家么？小蝶，你带着夕绾多认识一些人，免得又有人偷偷的去买高仿衣服穿，还一脸可怜相的说我们不带她玩。”
<br/>
<br/>        李玉兰心机深，说话也是一针见血，当即将那天夏夕绾在夏小蝶生日派对上的话拿出来大做文章。
<br/>
<br/>        今时不同往日，那时的夏小蝶还只是夏家千金，现在的夏小蝶已经是苏希的未婚妻了，李玉兰是仗着苏家给的气焰狠狠的踩夏夕绾。
<br/>
<br/>        果然那些富太太们都离夏夕绾远远的了，
<br/>
<br/>        夏夫人，我们的女儿可都是名媛千金，那些从乡下回来的土包子就不要来巴结了，我们是不会带她玩的。
<br/>
<br/>        夏夕绾被孤立了，这场订婚party上所有的人都对她心怀不善。
<br/>
<br/>        夏夕绾淡淡的莞尔，这些情况跟她所料不差，既然她敢来，就不会在乎这些。
<br/>
<br/>        毕竟，真正的好戏还没有开演。
<br/>
<br/>        有些人只有爬的越高，才会摔的越惨。

<br/>        “苏少，恭喜恭喜！”

<br/>        这时苏希来了。
<br/>
<br/>        夏夕绾抬眸，看向苏希。

<br/>        苏希今天一身黑色西装，里面白色衬衫打着领带，本就是俊逸温润的男人，今天更加好看。

<br/>        夏小蝶迅速走过去挽住了苏希的健臂，宣誓主权，她嗓音娇甜道，“苏希哥哥，夕绾来了，她来祝我们幸福了。”

<br/>        苏希早就注意到了这里的动静，自然将刚才所有人对夏夕绾的恶意都收在了眼底，他看着夏夕绾，勾了一下薄唇，“你喜欢的男人呢，他怎么没有过来保护你？”
