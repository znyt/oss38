        “为什么？”连星儿倒是没这种感觉。
<br/>
<br/>        顾千凝瘪了瘪嘴，愁眉苦脸地回答道：“因为，我喜欢洛羽啊！所以，不想看到自己的爱豆，跟别的女人接吻。”
<br/>
<br/>        “演那场戏的时候，不是借位错开了吗？又不是真的在吻。”连星儿忍俊不禁道。
<br/>
<br/>        顾千凝仍旧不满地说道：“但是，他的手是搂着那个女配的腰的吧！还把那女配搂得那么紧。”
<br/>
<br/>        “你这是吃醋呢？”连星儿笑着说。
<br/>
<br/>        顾千凝不解地看着连星儿：“难道你就不吃醋吗？”
<br/>
<br/>        连星儿微微摇了摇头。
<br/>
<br/>        顾千凝唉声叹气道：“其实我不仅仅只是吃醋，而且还是羡慕啊！心想着，要是那个女配换成是我该多好！”
<br/>
<br/>        “你又不会演戏。”
<br/>
<br/>        “对哦！我要是去报名表演系，将来说不定能和洛羽对戏。”顾千凝突然喜上眉梢，下一秒，又像一只泄了气的气球，“可惜，我颜值不够。”
<br/>
<br/>        说到这里，顾千凝猛然抬眸看着连星儿，眨了眨眼睛。
<br/>
<br/>        连星儿被她盯得浑身不自在，下意识地双手交叉摩挲着自己的双臂，讷讷地问道：“你盯着我干嘛？怪渗人的！”
<br/>
<br/>        “我觉得你就有当演员的资本！长得漂亮不说，身材也不错。”顾千凝咧着嘴，上上下下地将连星儿打量了一番。
<br/>
<br/>        连星儿只笑不语。
<br/>
<br/>        她又不是没想过这事，只是她的爹地大人不允许，这事她也就没得商量的余地了。
<br/>
<br/>        对戏结束后，洛羽看向了连星儿那边，见连星儿也正望着自己，脸上便挂起了一抹淡淡的微笑。
<br/>
<br/>        顾千凝见状后，激动不已地摇着连星儿的手，在不知情的情况下，自作多情了：“星儿，你刚刚看到了吗？洛羽对我笑了诶！”
<br/>
<br/>        “那说明他很宠粉丝的啊！”连星儿搪塞道。
<br/>
<br/>        顾千凝立马就把洛羽借位吻女配的事情给抛到了九霄云外。
<br/>
<br/>        就在两人有说有笑的时候，剧组的导演突然向她俩走了过来。
<br/>
<br/>        导演是个光头，脖子又短又粗，大圆脸，蒜头鼻上架着一副黑框眼镜，却看起来仍旧很凶巴巴的样子。
<br/>
<br/>        看到这个导演过来时，连星儿和顾千凝有些面面相觑。
<br/>
<br/>        直到导演在她俩跟前停下脚步，抬起手中的折扇，指向连星儿。
<br/>
<br/>        “你。”
<br/>
<br/>        “我？”连星儿下意识地指了指自己的鼻子。
<br/>
<br/>        就连那边的洛羽和女演员以及一旁的工作人员，也纷纷向连星儿看了过来。
<br/>
<br/>        导演上上下下将连星儿打量了一番后，一副很满意的样子，点头道：“就由你来出演男主心里那个死去的白月光。”
<br/>
<br/>        “啊？”连星儿一脸懵。
<br/>
<br/>        而这个时候，顾千凝表现得比连星儿更激动。
<br/>
<br/>        “星儿，你是不是以后就要当明星呢？”
<br/>
<br/>        “……”连星儿有些无语。
<br/>
<br/>        直到站在那边的洛羽冲她眨眼睛，她才明白，原来这是洛羽的意思。
<br/>
<br/>        可是，充当男主心里那个死去的白月光是几个意思？
<br/>
<br/>        如果不是洛羽私底下发短信说要她点头答应这件事情，不然，她才不会去演一个“死人”。
<br/>
<br/>        以前，连星儿不理解白月光是什么意思。
<br/>
<br/>        直到她看了剧本后，也算是一知半解了。
<br/>
<br/>        这个“白月光”完全就是活在男主的回忆里。
<br/>
<br/>        一袭长发披肩，白色蕾丝绑着公主辫，穿着白色轻纱连衣裙，肤若凝脂，优雅地坐在三角钢琴前。
<br/>
<br/>        阳光从落地窗外照耀进来，洒在女孩精致的脸蛋上，美得如梦如幻。
<br/>
<br/>        连星儿并不需要任何对戏，只需要坐在三角钢琴前假装弹奏就好。
<br/>
<br/>        可是，假装她不会，真弹倒是能露那么一手。
<br/>
<br/>        洛羽本来还担心她会怯场，却没料到，此刻，连星儿成为了众目癸癸的焦点。
<br/>
<br/>        在场无论是工作人员还是围观的学生，一个个都十分惊艳地看着连星儿。
<br/>
<br/>        就连这个“白月光”的出场其实不过十秒的戏份，导演却给连星儿录了一整段，直到钢琴独奏结束，连“咔”都省了。
<br/>
<br/>        现场直播的视频，很快就在男生之间传开。
<br/>
<br/>        男生寝室里，明司寒看到连星儿的这段视频的时候，脸色不大好，连忙打了通电话给自己的二哥明希哲。
<br/>
<br/>        “谁要她演你的白月光的？”明司寒没好气地质问。
<br/>
<br/>        明希哲先是一脸懵，随即马上就反应了过来，忍俊不禁道：“这是导演自己挑的！没办法！导演挑了很多个女演员，都没挑到合适的。然后洛羽就把连星儿的照片拿给导演看，导演一眼就看中了。”
<br/>
<br/>        “不能换？”明司寒凝眉道。
<br/>
<br/>        明希哲笑道：“都拍好了！而且还是一条就过。导演十分满意。哪怕给导演施压，导演估计也不会换。”
