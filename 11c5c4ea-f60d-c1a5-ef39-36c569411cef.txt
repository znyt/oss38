<br/>　　五行刀圣长啸一声，手持地元神刀，向着楚风眠追逐而来，又是一刀向着楚风眠斩杀而去。
<br/>　　而这时那少帝，也是同样出手，手心一点，他面前的空间，便是开始崩灭，一指崩灭，崩灭天地，施展出魔族之中极为恐怖的杀招。
<br/>　　两位半步天人境的强者，同时出手。
<br/>　　樊老则是手中凝聚剑锋，一剑斩杀而去，但是以一敌二，更是面对这少帝这样一位半步天人境巅峰的强者，他打出的剑锋，都是被轰然打破。
<br/>　　瞬息之间，樊老的身形便是被逼退，身上的气息都是变的混乱了起来。

<br/>　　看着樊老的遁光离开，五行刀圣的脸上冷笑一声，转头看向楚风眠。
<br/>　　他的一双充满杀意的眼睛，死死的盯着楚风眠，脸色语法的阴狠起来，冰冷的说道。
<br/>　　“小子，这一次我看你到底还能不能逃的出去！”

<br/>　　五行刀圣的身影，一步上前，瞬息之间来到了楚风眠的面前，一刀猛然斩落而下。
<br/>　　只见楚风眠的身躯之上的衣衫，都是开裂了起来，十二万九千六百枚血肉晶体，都不由自主的凝聚了起来，准备抵挡着这一道刀锋的落下。

<br/>　　楚风眠身上的生命力，猛然凝聚起来，突然之间在楚风眠的掌心之中，一段朽木，突然出现。

<br/>　　这一道轰然落下，斩杀在了楚风眠的身躯之上，只见楚风眠的身形，从天空之上当即跌落了下去。

<br/>　　这样的轰击之下，哪怕是半步天人境的强者，都只有死路一条了。
