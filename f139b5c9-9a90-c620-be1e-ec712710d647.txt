　　萧壮和萧子木仍旧昏迷在床上，萧逸在一旁看着。
<br/>　　作为一个炼药师，熟知的炼药知识，除了炼制丹药外，还有对各种材料的认知，以及对人体所起的作用。
<br/>　　毕竟炼丹，是为了让武者服用的，自然要考虑人体与丹药间的各种关系。
<br/>　　因此，炼药师本身就有着高超的医术。
<br/>　　如果只是普通人的感冒发烧之类的小病，炼药师往往一颗回元丹、强身丹，以助固本培元、强身健体，便能治愈。
<br/>　　而武者的大病，一般来源于战斗造成的伤势，修炼中出现的反噬等等，这则需要更高级的丹药去治疗。
<br/>　　萧逸替两人检查了一番，眉头紧皱。
<br/>　　“武者战斗，多有受伤的情况，丹田被伤、经脉被损，都是家常便饭，二品丹药中有数种丹药都能起疗效。”
<br/>　　“可丹田完全被毁，却从未有过能治愈的例子，起码紫云城内无人成功过。”
<br/>　　萧逸边看着大长老的信，边思索着。
<br/>　　萧家中也有数位族人，也是因战斗而丹田被毁，但连大长老这个三品炼药师，也无法帮助他们治疗。
<br/>　　萧逸继续将信看下去。
<br/>　　“丹田完全被毁，之所以无法治疗，最大的原因并非丹药无效，而是身体无法吸收药力。”
<br/>　　“丹田起吸收外在力量之功，丹田被毁，无论是天地灵气抑或丹药之力，自然无法被身体吸收，更无法起效。”
<br/>　　“丹田不修复，就无法吸收药力；而没有药力，丹田又不可能恢复。彼此成了一个死循环，是一个无法解开的死结。”
<br/>　　“死结吗？”萧逸眉头皱的更紧，自语道，“大长老既然送信给我，必有解结之法。”
<br/>　　果然，待得萧逸将信读完，紧皱的眉头稍微松了一些。
<br/>　　“萧壮、萧子木二人可惜是凡境武者，若是后天境武者，他们除了丹田能吸收丹药力量外，还能多出主脉来吸收药力。”
<br/>　　“若你能想到法子助他二人突破后天境，激发主脉的力量，或许可以反其道而行之，不借助丹田，直接将丹药之力送进主脉并发挥药效。”
<br/>　　“当然，这同样是一个死结。没有丹田，无法凝聚气丹，他们无法炼化主脉，便不能成为后天境。而不突破后天境，他们又无法恢复丹田。”
<br/>　　“两个死结，我只能为你解开一个。剩下一个，便看你自己的本事了。”
<br/>　　将信读完，后面是一些大长老勉励的话语。
<br/>　　萧逸将信收起来，忽然笑了，“考验我的本事？呵，恐怕不是关于炼药的本事，而是...”
<br/>　　“大长老阿大长老，你这所谓的考验，对我而言是最简单不过了。”萧逸轻轻一笑。
<br/>　　萧逸拿出了乾坤袋，从中挑选出了材料，不过他并没有炼丹。
<br/>　　材料中，几乎全是药性温和，起护脉养血的功效。唯有一样，刚一拿出来，顿时整个房间都被一股妖异的红芒所覆盖。
<br/>　　正是妖血炎心果。
<br/>　　“妖血炎心果，三品巅峰灵果，药力猛得很，也不知萧壮和萧子木的身体受不受得住，我还是小心些为好。”
<br/>　　想罢，萧逸走到萧壮身前，并将妖血炎心果的果肉一点点放入萧壮口中。
<br/>　　果肉放进去，几乎是一瞬间，便化为一道妖异红光，进入萧壮体内。
<br/>　　妖血炎心果，有着焚筋炼脉的神效。这灵果对于后天境武者，乃是不可多得的神果，武者服用，能快速炼化主脉，提升修为。
<br/>　　既然萧壮和萧子木无法靠自己的力量去炼化主脉，萧逸便借这妖血炎心果，直接为他们炼化主脉。
<br/>　　“额。”昏迷中的萧壮忽然面露痛苦之色，浑身发烫，双脸红热，满头大汗。
<br/>　　萧逸皱起了眉头，有些担心。
<br/>　　幸好，这种情况只持续了十数秒。之后，萧壮的脸色恢复了正常。
<br/>　　萧逸继续给他一点点地喂服着妖血炎心果。
<br/>　　当半颗妖血炎心果都被萧壮吞服后，忽然，一道澎湃的气息从他身上爆发。
<br/>　　同一时间，萧壮的身体上不断有一层黑乎乎的粘稠液体排出。
<br/>　　“气息涌动，这是突破的征兆；杂质被排出，证明一条主脉已被炼化；萧壮马上就能突破到后天一重。”萧逸顿时一喜。
<br/>　　当萧壮的气息停止涌动后，萧逸感受了一番，果然，萧壮的气息时后天一重。
<br/>　　“境界已突破，主脉也已成功炼化，之后，只需为他炼制修复丹田的丹药即可，药力会通过主脉生效。”
<br/>　　萧逸喃喃自语着，暗暗点头。
<br/>　　之后，萧逸如法炮制，将剩下的半颗妖血炎心果给萧子木服下，萧子木也成功突破到凡境一重，并炼化出一条主脉。
<br/>　　“呼...”萧逸松了口气，心头放下一颗大石，转过身，离开了房间。
<br/>　　房门外，四长老和六长老竟在等待。
<br/>　　“你们怎么在这？”萧逸淡淡地问道。
<br/>　　“自然是给你把门，免得别人打扰了你替萧壮和萧子木二人疗伤。”四长老打了个哈哈，笑着说道。
<br/>　　萧逸玩味一笑，“两个凡境子弟，竟劳烦两位长老亲自把门守护？呵呵，两位长老有心了。”
<br/>　　说罢，萧逸转身离去，临走时，语气恢复了淡漠，“让他们休息一番，明日我为他们炼制好丹药即可。”
<br/>　　待得萧逸完全离去，六长老才愣了愣，道，“我怎么感觉这小子话里有刺，难道他发现了？”
<br/>　　“你说呢？”四长老一副在看白痴的眼神看着六长老。
<br/>　　下一秒，四长老露出了会心的笑意，“萧逸真的很好，他的绝世天赋自不必多说，智慧也过人，竟一眼就看穿了我们的意图，起码比你这个笨蛋要聪明许多。”
<br/>　　“他真的看明白了？”六长老自语道，“小小年纪，竟有这般过人的洞察力。”
<br/>　　“那妖血炎心果，对后天武者的诱惑力太强了，顷刻间就能炼化主脉，令武者修为大增，没人能抵抗这种诱惑。”
<br/>　　“那慕容千军就是最好的例子，之前明知那里有银背妖蟒守护，危险得很，却还是宁愿冒险等待采摘时机，甚至不惜为此算计我们萧家的族人，不择手段也要得到。”
<br/>　　“但萧逸几乎没有任何考虑，立刻就愿意拿出一颗来给萧子木和萧壮，证明他将情义看得比其他东西看得更为重要。”
<br/>　　四长老一边说着，脸上的满意之色也越发浓郁。
<br/>　　六长老疑惑道，“如果刚才他不拿出来，我们会硬抢吗？”
<br/>　　“当然不会。”四长老道，“那本来就是打算给他的，他才是天赋最高之人，妖血炎心果给他最合适。”
<br/>　　四长老继续道，“大长老给他这个考验，也只是顺手而为，就算他不给，也没人会逼他。但，现在我们起码知道了，我们的这位少家主，不会是那种寡情薄意之人，日后成了家主，也会对族人们很好。还有他刚才明明看穿了我们，却没有拆穿，摆明了是给我们个台阶下。”
<br/>　　六长老点点头，道，“也是，幸好当初五长老没让他儿子萧若寒抢去了少家主之位。哼，萧若寒那小子，我一看就知道是那种心狠手辣之辈，要是家族落到他们手里，也不知会成什么样子。”
<br/>　　六长老边说着，忽然想起了什么，顿时大怒，“四长老，你刚才说谁是笨蛋？老子警告你，以后再敢诋毁我，老子跟你没完。”
<br/>　　.....
<br/>　　萧逸回到了自己的房中，对于可以恢复萧壮和萧子木两人丹田之事，心头仿佛落下了一块大石，他实在不想欠其他人人情。
<br/>　　他再次拿出了妖血炎心果，现在他也准备吞服了。
<br/>　　经历过五天前的密林战斗，他充分了解到这个世界的残酷与危险，实力才是一切的保障。
<br/>　　他倒是没有顾虑妖血炎心果的药力很猛，毕竟他的实力和修为摆在这，一口便把一颗吞了。
<br/>　　一股妖异的力量涌入体内，而后不经过丹田，直接环绕到第三条主脉上。
<br/>　　同时，一股被火焰焚烧的疼痛感觉让他咬了咬牙。
<br/>　　不过这感觉没有持续多久，不多时，第三条经脉的杂质便完全被炼化了。然而，这时妖异力量还未完全消失。
<br/>　　在萧逸的控制下，将其环绕到第四条主脉上，将第四条主脉也一并炼化。
<br/>　　一颗妖血炎心果，足以炼化两条主脉。
<br/>(本章完)
