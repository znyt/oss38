　　小九跟着进来，冲他们点了点头。
<br/>　　林三生和瓜瓜都松了口气。
<br/>　　“老大……”瓜瓜冲到叶少阳身边，脑袋埋在他腰间。叶少阳揉了揉他的脑袋。
<br/>　　“你接着往下说吧，之前你还没说完呢。”
<br/>　　“我？”
<br/>　　瓜瓜挠着头回想着自己之前说到哪儿了，想了一会才想到，说道：“那个，老大，军师他们说的没错，我之前那个不愿意说出来的秘密就是……其实，你未必是天选之子。风大伯他们说的那些，我不知道，我的任务就是跟风大伯类似：先辅佐你，让你觉醒……这个，我还是从头说。
<br/>　　老大，你的确是叶法善天师的传人，这一点没错，但是真相是——叶法善将他的元神和金身各自赋予了一缕神念，在酆都大帝帮忙下，一起投入了轮回中，他们各自轮回，生出完整的三魂七魄，各自为人，是的，就是有两个人。
<br/>　　其中金身所化，就是老大你，在两抹神念从产生到轮回的过程中，你相对来说是在明处的，另一道元神所化，却是在暗，叶法善这么做，是构成了一道双保险，就算这件事被无极鬼王注意到，也只会注意到你，从而让真正的天选之子逃过追捕……而我的任务，就是赶在天劫之乱发生之前，潜到你身边，保护你，并且引导你觉醒……
<br/>　　你虽然是假的天选之子，但也是叶法善金身所化，若你不死，叶法善则期待你能变得更强，辅佐真的天子，荡平三界要挟，度过天劫之乱。”
<br/>　　此言一出，连道风都有些震惊，毕竟他只知道酆都大帝这边的计划，对于叶法善当初做了什么，只能靠猜，他也是真没想到叶法善居然来了一手双保险。
<br/>　　“金身？什么金身？”
<br/>　　“他斩尸证道之后，凡身被舍在了茅山，火化之后，被他炼成了金身，这个我不是道士，我也不是很懂，他也就是这么告诉我的。”
<br/>　　叶少阳问道风：“你有金身吗？”
<br/>　　“大罗真仙，他是大罗真仙！”
<br/>　　在叶少阳询问下，道风又解释了什么是大罗真仙：道家的斩三尸证道，有很多修行宗法，但归纳起来有两大类，大罗真仙和太乙散仙，因为凡是证道的都远离了三界，跟人类再没什么来往，所以人间法术界对这两个名词不甚了然，甚至曲解了意思。
<br/>　　大罗真仙与太乙散仙，是道门斩三尸之后的两重境界，绝大部分斩三尸的，都是太乙仙，通过长时间的修炼（可能有几百上千年），有机缘的可以晋升为大罗仙，不过其中的难度不亚于普通法师斩三尸证道。
<br/>　　“但因为修行法门各有万千，也有极少数的道士，证道之后，初始就是大罗仙，这样的古往今来也没几个，吕洞宾，张果，陈抟，这些都是。大罗仙，能以凡身修炼成金身，实际就是身外化身，只不过属性远比一般身外化身强得多。
<br/>　　“那你……”
<br/>　　“我又不是按照道门法术证的道，跟这些不一样，但只要重塑灵身，应该比大罗仙不差。”一时间大伙都用惊羡的目光望着他。
<br/>　　“对了，李浩然就是大罗仙。”
<br/>　　叶少阳沉吟了一下，对瓜瓜说道：“也就是说，我仍然是叶法善的隔代传人，但不是正牌的天选之子？”
<br/>　　瓜瓜点点头。
<br/>　　“那转世鬼童又是谁？”
<br/>　　“这……我是听说，当初你跟那一位刚进入轮回，就被无极鬼王盯上了，用上了所有力量在人间搜寻你们，都没有结果，于是他自己切割出了一缕神魂，也按照你们的轨迹，通过轮回井进入人间，长时间地寻找你们，后来真的找到了，但是，他却不能杀了你们。”
<br/>　　“为什么？”不止叶少阳一个人问道。
<br/>　　“当时离天劫之乱还早，杀了你们，你们的精魄仍然会聚魂重生，最终还是能通过轮回来到人间……既然杀不死，无极鬼王就想了个办法，用了某种秘法，将神魂灌入天选之子体内，然后裂魂为二，一起送去轮回，酆都大帝虽然知晓这一切，但经过这次移魂和裂魂，就算是他，也已经分辨不出哪个是转世鬼童，哪个是天选之子。”
<br/>　　叶少阳瞠目结舌，心中默默算了一下，道：“这样就是三个人，其中两人，一个是转世鬼童，一个是天选之子？”
<br/>　　“不，是四个。”瓜瓜咬着嘴唇道，“无极鬼王模仿了叶法善的鱼目混珠之法，让左君重生轮回，来到人间，一来是掩人耳目，等天劫之乱开始，左君会最先醒来，然后找到转世鬼童，帮助他觉醒，再保护他，直到魂桥搭建，将无极鬼王牵引到人间来……”
<br/>　　此言一出，大家面面相觑。
<br/>　　小九沉吟道：“怪不得从来都听说鬼王有左右手，但一向我们我只见过右君，从没见过左君，原来在人间……”
<br/>　　林三生道：“我听所，右君，是鬼王拆了自己一根灵骨，修炼而成，左君却是更加神秘，从没听说过有关他的来历和背景。”
<br/>　　叶少阳不关心这些，心中飞快地思索着，喃喃道：“所以，一共有四个人，分别是：天选之子、转世鬼童、沉睡的左君，还有叶法善金身所化的后人？为什么连大帝也无法分辨这四人？”
<br/>　　“因为鬼王用了移魂术，将你们四个拆魂换命，又修改了你们四人的命格，让你们看上去没任何区别，你们各自体内的特殊气息，都隐藏在元神神念之中，就算是酆都大帝，想要弄清楚你们谁跟谁，也得炼化了元神才知道，但他不敢这么做，因为元神一旦毁了，就无法再凝聚，如果选对了，找到转世鬼童固然好，但如果不小心弄死了天选之子，那叶法善千年前那一场谋划，都白忙活了。”
<br/>　　叶少阳愕然。
<br/>　　林三生想了下说道：“那同样的，鬼王也不敢有什么动作，不然万一杀错了，自己的计划也就付诸东流了。”
<br/>　　叶少阳道：“这有什么，把四个人一起杀了，大家都一样，谁也不亏不赚，岂不是好？”
<br/>(本章完)
