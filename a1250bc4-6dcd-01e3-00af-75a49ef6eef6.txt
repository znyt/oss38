        &lt;h3&gt;第三卷第二百四十八章  我想与你共度余生&lt;/h3&gt;
<br/>
<br/>        宁家为陈子昂公司员工举办的接风宴因为宁雅是宁家人的原因，变成了对凌辰的感谢宴。
<br/>
<br/>        凌辰心里不爽，但是又不敢说的太过，毕竟宁少天还是未来的岳父，宁雅也不想将关系闹得太僵，她是代表陈子昂回来的，不，应该是她需要更加的努力起来。
<br/>
<br/>        一餐饭吃的心思各异，宁少天坐了一会就推说有事情先行离开，留下凌辰和宁雅、宁玥、董亚子四个人。
<br/>
<br/>        宁少天一走，凌辰立马精神了，平时溜情话的本领在酒精的发酵下，顺顺利利的脱口而出：
<br/>
<br/>        “小雅，你知不知道，没有你的日子是多么的难熬，我的心就像是在火上烤，在冰水里泡。”
<br/>
<br/>        宁玥坐在凌辰的身边，眼睛瞪得大大的，桃花眼泪水涟涟，凌辰怎么可以这样对他，前几天拉着她的手登五台山，今天就向宁雅示爱。
<br/>
<br/>        “凌总这撩妹的本事是越来越溜了，而且还不看看场合和时间。”宁雅冷冷的笑着。
<br/>
<br/>        “此生非你不撩，非你不娶。”凌辰已经豁出去了，管他呢，机会可不是经常有，况且身边还坐着对他虎视眈眈的宁玥。
<br/>
<br/>        这一次送宁玥到杭州，原本是要去北京找宁雅的，既然有机会他就要好好把握住。
<br/>
<br/>        况且，今年他们会更加忙碌，刘诚之去了甸城，就报了个平安，别的都没有说，也不知道情况怎么样。
<br/>
<br/>        董亚子已经是忍不住要笑了：“宁雅姐姐，我告诉你，凌辰哥哥在我们哪里可是天天都念叨着你，我追了他好久，差点将董家都给他了，可是他都不为所动。”
<br/>
<br/>        “你追他？”宁雅笑着问，这个董亚子也是可爱的很，连这样的话都说出来了。
<br/>
<br/>        “是啊，记得有一次你给他打电话，我接的，他和我爸爸正在谈事情呢。”
<br/>
<br/>        “你怎么知道是我？”宁雅心里一动，原来那个电话是董亚子接的。
<br/>
<br/>        “电话名称是，我的小雅啊。不是女朋友谁会有这么亲密的称呼？”
<br/>
<br/>        “亚子，你害得我好苦，原来小雅不理我是你的原因，罚你三杯酒，给小雅道歉。”
<br/>
<br/>        董亚子委委屈屈：“好吧，小雅姐姐，我错了。”
<br/>
<br/>        “关亚子什么事情？自己的问题自己不知道？”宁雅为董亚子申辩。
<br/>
<br/>        “我确实不知道你怎么不理我，这么长时间我一直在煎熬，不敢给你打电话，不敢给你发信息。”只要宁雅搭理凌辰，凌辰就会有办法赢回宁雅的心。
<br/>
<br/>        “说来说去，你还是不够信任我。”胡婷接电话的事情宁雅已经知道了，就是接了胡婷的电话，凌辰才发生改变的。
<br/>
<br/>        “小雅，这么说一切都是误会，误会，你还是原来的小雅，我也是原来的凌辰，我们都不曾改变过，我们一直都在。”此时的凌辰不再是哪个腹黑高冷的人，而是温柔多情的。
<br/>
<br/>        “是啊，我一直都在啊，从来都不曾远离。”
<br/>
<br/>        宁玥看着凌辰和宁雅在饭桌上毫无顾忌目无旁人的秀恩爱，心里受到了一万点的伤害，她终于是坐不住了“哇”的一声哭出来，站起来跑出去了。
<br/>
<br/>        “宁玥。”宁雅看着跑出去的宁玥，准备去追，被凌辰拉住了手。
<br/>
<br/>        董亚子看着跑出去的宁玥，站起来追了出去。
<br/>
<br/>        “你干嘛拉着我？”
<br/>
<br/>        “我不想错过，我也不想别人窥觊我。小雅，我知道此时说出来有些不合时宜，但是我还是想对你说，我不想错过你。”
<br/>
<br/>        “你想干什么？”宁雅被凌辰拉着手。
<br/>
<br/>        “我想和你共度余生！”
<br/>
<br/>        “噗嗤，在这里共度余生？”宁雅的父亲走了，妹妹生气的跑了，凌辰此时却说要和她共度余生？
<br/>
<br/>        “我，不是，我......”凌辰突然清醒过来了，这个时候说这种话确实不合时宜，但是他控制不住自己。
<br/>
<br/>        “凌辰，我还有很多事情要做，请你放开我。”宁雅冷静下来了，明天她还要面临一场争斗，宁玥跑了，回去会怎么样对父亲说？
<br/>
<br/>        她没有时间和凌辰卿卿我我，她要想好明天怎么做。
<br/>
<br/>        凌辰看着宁雅，她的眼里是一种前所未有的坚定和倔强，她的身上散发着自信的光芒，似乎这才是她自己。
<br/>
<br/>        “好。”凌辰放开宁雅：
<br/>
<br/>        “小雅，我想对你说，无论你今后干什么我都会站在你身边陪你一起走过，此生此世不离不弃！”
<br/>
<br/>        “凌辰，谢谢你，我不知道我今后的路走的顺不顺利，既然选择了，我就会坚持下去，我要做我自己，努力、上进、坚强的自己，所以祝福我吧！”宁雅看着凌辰，在这一段时间里，她明白了一个道理，女人，其实要比男人更努力，只有你足够优秀你才能配得上优秀的他。
<br/>
<br/>        无论是恋爱期间，还是婚姻，人们讲究的门当户对不是一点道理都没有，如果对方才识渊博，学富五车，而你只是一个小学水平，你又怎么能够站在他的面前和他引经论据，畅谈未来？如果对方是一个山野村夫，而你腹有诗书，他怎么能够和你晨起描眉、晚来对吟？同理，在强大的实力面前，你必须有足够的资本和他并肩而立！王子和灰姑娘的故事不是没有，前提是灰姑娘必须足够优秀，就比喻陈子昂，强大如陈子昂也逃不过世俗的桎梏。
<br/>
<br/>        宁雅需要成长，她需要靠自己的努力来完成她人生的蜕变。
<br/>
<br/>        这一段时间是她人生的转折点，她需要把握住，餐桌上的酒还在散发着淡淡的酒香，菜微凉，可是一个人都没有了。
<br/>
<br/>        凌辰看着宁雅走出房间，他默默的和她并肩而行，似乎他只能陪着她默默前行。
<br/>
<br/>        “凌辰，宁玥喜欢你，大家都能看出来，我相信你也能感觉到，如果你喜欢她就去追求她，不喜欢就不要给她希望，更不要因为她是我的妹妹而觉得需要去照顾她的情绪，爱情不是施舍，更不是模棱两可。”宁雅停下脚步，看着凌辰的眼睛说。
<br/>
<br/>        “我想和你共度余生，我只想牵你的手去踏浪逐云追月看斜阳。”凌辰锲而不舍。
<br/>
<br/>        “我就是被你的情话给迷惑了。”宁雅叹了口气。
<br/>
<br/>        “我的情话此生只对你一人说。”
<br/>
<br/>        “好吧，我，信你！”心已动又怎么能拒绝？
<br/>
<br/>        “明天你将要和你妹妹谈判，我想你肯定已经做好了准备。”凌辰握住了宁雅的手。
<br/>
<br/>        “陈子昂安排我来的时候，我已经做好了准备。凭心做事，不偏不倚。”
