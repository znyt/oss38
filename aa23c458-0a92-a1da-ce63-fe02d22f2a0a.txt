陆寒霆来了，所有人像打了鸡血一样都冲上前，将陆寒霆众星拱月的围住了，
<br/>
<br/>        听说陆总要来参加妍妍小姐的生日party，没想到真的来了，陆总，幸会幸会。
<br/>
<br/>        这些年陆总十分的低调神秘，从来没有公开出席过什么场合，这还是第一次，足以可见陆总对妍妍小姐的喜爱。
<br/>
<br/>        妍妍小姐不但是海城第一名媛，还是医学天才少女，跟陆总真是非常的般配，天生一对啊。
<br/>
<br/>        在所有的恭迎和赞美声中，夏妍妍抬眸看了身边的陆寒霆一眼，他身高腿长的伫立在璀璨的灯火中央，灯光仿佛给他英俊的五官镀上了一层金黄而炫目的光芒，他瑰丽迷人的让人不敢直视。
<br/>
<br/>        夏妍妍一直梦想着有一天能够光明正大的站在他的身边，享受着所有人的羡慕嫉妒恭迎，还有无上的荣光。
<br/>
<br/>        这个梦想终于实现了。
<br/>
<br/>        夏振国满面春风，他也是做梦都没想到自己会成为这个权贵男人的国丈，他会看上自己的女儿。
<br/>
<br/>        夏振国谄媚的笑道，“陆总，谢谢你来参加妍妍的生日party，我自荐一下，我是妍妍的爸爸，叫夏振国，这是妍妍的妈妈…”
<br/>
<br/>        夏振国去拉身边的李玉兰，但是没有拉到，回头一看，本该是全场最得意的李玉兰已经脸色煞白。
<br/>
<br/>        李玉兰震惊的看着陆寒霆，瞳仁不停的收缩放大，她露出了不可置信的表情，“你…你…”
<br/>
<br/>        李玉兰前段时间元气大伤，休养了整整两个月才放出来的，但是她记性不差的，她迅速认出了陆寒霆。
<br/>
<br/>        陆寒霆曾经去过夏家，还救下了夏夕绾。
<br/>
<br/>        当时她以为陆寒霆是夏夕绾包养的小白脸，那天他气场冷厉狷狂，轻而易举的就将她高薪聘请来的几个黑衣保镖给吓跑了，他绝对是她活久见系列里见过最嚣张的小白脸，没有之一。
<br/>
<br/>        所以李玉兰对他印象十分的深刻，她万万没有想到，他…他竟然就是陆氏总裁陆寒霆！
<br/>
<br/>        那他和夏夕绾是什么关系？
<br/>
<br/>        李玉兰的脑袋一下子就当机了，无法思考了。
<br/>
<br/>        李玉兰的失态让夏妍妍和夏振国相当的不悦，夏振国迅速低声训斥道，“李玉兰，你在干什么，快见过陆总。”
<br/>
<br/>        夏妍妍看着陆寒霆，娇柔的解释道，“陆总，我妈见到你可能是太开心了，希望你不要介意。”
<br/>
<br/>        陆寒霆自从进来还没有说话，现在他深邃的狭眸淡淡的落在了李玉兰的身上，然后把薄唇勾出了一道似笑非笑的弧线，“我跟伯母曾经见过一面。”
<br/>
<br/>        什么？
<br/>
<br/>        夏振国差点跳了起来，他低声询问道，“李玉兰，你什么时候跟陆总见面的，你怎么也不告诉我？”
<br/>
<br/>        李玉兰出了一身的冷汗，她结巴道，“我…我…”
<br/>
<br/>        夏妍妍觉得李玉兰有点丢人，在陆寒霆面前丢了她的面子，她迅速转移了一个话题，告状小能手夏妍妍迅速上线了，“陆总，告诉你一个很好笑的笑话哦，这位原来不是绾绾的老公，而是绾绾的管家，但是这位管家说绾绾的老公要在帝豪酒店的顶楼为绾绾庆生，这是不是很好笑呢？”
