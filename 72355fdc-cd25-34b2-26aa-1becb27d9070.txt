　　一行人逐渐的接近乱葬海的区域。
<br/>　　所遇到的虚空兽，也是越来越多。
<br/>　　一路上的无数虚空兽，都是楚风眠自告奋勇，全部斩杀。
<br/>　　这让司空轩的心中，都是冷笑连连。
<br/>　　这些虚空兽本身就难以对付，虽然说斩杀也可以得到不小的好处。
<br/>　　可是他们这一次的目的，可是要前往颜古龙帝的洞府，跟颜古龙帝洞府中的无尽机缘相比，这等虚空兽又算得了什么。
<br/>　　本来他们都是一个个的节约灵力，准备是在颜古龙帝的洞府中大展拳脚的。
<br/>　　碰到楚风眠这样一个愿意去主动斩杀虚空兽的，他们自当都各种奉承，让楚风眠不断的去斩杀虚空兽，一路上的虚空兽，几乎全部都被楚风眠斩杀。
<br/>　　楚风眠脸色显得十分平静，不过暗地里，却是装作吞服了不少的丹药，来恢复灵力。
<br/>　　但是这灵力的消耗，哪里是丹药可以很快补充好的，显然对楚风眠也是有着极大的负担。
<br/>　　在他们看来，楚风眠只是在外强中干，强撑面子罢了，不过这一切，唯有楚风眠自己的心中清楚。
<br/>　　他拥有神脉，灵力无穷无尽，让他斩杀一万头虚空兽，楚风眠都不会任何的劳累。
<br/>　　更何况斩杀虚空兽，也足以给楚风眠带来巨大的好处，吞噬虚空兽的灵力，凝聚远古战龙精血，楚风眠已经是凝聚了一百多滴远古战龙精血了。
<br/>　　既然这些人相要算计楚风眠，楚风眠倒是正好将计就计。
<br/>　　故意示弱，等到进入到颜古龙帝的洞府之中去，在展现真正实力。
<br/>　　“前方又是一群虚空白虎，大约两三百只，不知道楚兄能不能对付的了？”
<br/>　　司空轩看着前往出现的一群虚空白虎，悄然之间对着楚风眠开口道。
<br/>　　“若是楚兄一个人难以对付，那我们还是绕路走吧，这虚空白虎，可不好对付。”
<br/>　　“楚兄已经斩杀了不少虚空兽了，这虚空白虎不好对付，我们还是绕路吧。”
<br/>　　麟木在一旁也是故意开口道，说是在劝解，但是其中的意思，却是在激楚风眠。
<br/>　　“哼，几头区区虚空白虎罢了，我这就去将他们，全部斩杀！”
<br/>　　楚风眠脸上故意露出几分怒色，直接是身形一动，冲了上去。
<br/>　　这让这几人的脸上都是露出大喜。
<br/>　　“这虚空白虎，每一只可都有着巅峰圣者的实力，斩杀起来麻烦无比，这小子居然自己一个人进去斩杀，真是愚蠢。”
<br/>　　罗辰看着楚风眠，冷声的嘲笑道。
<br/>　　“将灵力用在这里地方，等到了颜古龙帝的洞府，看他哪里还有力量。”
<br/>　　“不错，一个愚蠢的小子，带他来倒是省了不少麻烦，反正有人帮我们对付虚空兽，何乐而不为。”
<br/>　　麟木也是冷笑连连道。
<br/>　　“他还以为我们是在奉承他，实际上不过就是利用他罢了，等到了颜古龙帝的洞府中，他没有任何用处，就直接将他踢走。”
<br/>　　“话不能这么说，毕竟他还是有点作用了，我们可以让他去对付其他妖兽嘛。”
<br/>　　那云公子目光闪烁，也是大笑道。
<br/>　　“颜古龙帝的洞府中，也有着无数陷阱，可以将他当做挡箭牌去探探正好。”
<br/>　　“白萱，你从哪里找来的这样一个傻子。”
<br/>　　司空轩看着白萱，冷笑道。
<br/>　　“我看你这个帮手，快要撑不下去了，你要不要求求我们，帮你去救他？”
<br/>　　“哼。”
<br/>　　听着司空轩的话，白萱脸色冰冷到极点，一句话不发。
<br/>　　不过她的心中，也是担心无比，这几人，显然都是在对付楚风眠。
<br/>　　她也悄悄传音过楚风眠，但是楚风眠却是告诉她不用任何动作，这让她也没有其他办法，只能是在一旁等待着。
<br/>　　交谈之际，楚风眠已经是直接冲入到了虚空白虎之中。
<br/>　　“这些人都以为我是在硬撑要面子，实际上根本想不到，我会有着神脉，我的灵力根本就是无穷无尽，杀再多的虚空兽都无妨。”
<br/>　　“没有这些人来争抢，这些虚空白虎的力量，也全部都归我了！”
<br/>　　楚风眠心中兴奋无比，凝聚灵力，猛然一掌对着其中一尊虚空白虎轰击了过去。
<br/>　　“砰！”
<br/>　　这一只虚空白虎，顿时在这一掌之下，被直接贯穿头颅，其中的无数灵力，被楚风眠一下子吞噬，这一切做起来，行云流水。
<br/>　　“好，这一只虚空白虎的灵力，足以凝聚十分之一滴远古战龙精血，十只虚空白虎，足以助我凝聚一滴完整的远古战龙精血。”
<br/>　　“将这些虚空白虎全部斩杀，至少可以凝聚出十几滴远古战龙精血！”
<br/>　　楚风眠心中计算这，也是狂喜。
<br/>　　这虚空海果然是好地方，这人人畏惧觉得麻烦的虚空兽，在楚风眠面前，都是一个个补品。
<br/>　　寻常武者，就算是实力在强大，也没有人敢来虚空海中打死捕杀虚空兽。
<br/>　　因为对付一只虚空兽，本身就需要消耗不少的灵力，就算是斩杀，也是杀敌一千自损八百。
<br/>　　不过楚风眠可不同，有着神脉，他消耗的灵力都可以随时恢复，所以这一只只虚空兽，在楚风眠的面前，都如同一枚枚丹药，补品。
<br/>　　“统统斩杀！”
<br/>　　楚风眠连连催动力量，不断的灭杀这周围的虚空白虎，汲取这他们的力量。
<br/>　　在斩杀的时候，楚风眠故意是放慢了速度，造就一种已经是外强中干，没有多少实力的样子，只是在苦苦支撑。
<br/>　　楚风眠还故意的吞服了不少贵重的丹药，看起来是在恢复灵力，但是实际上却只是被楚风眠炼化，凝聚远古战龙精血所用了。
<br/>　　“这小子果然是在硬撑，就连上元回灵丹都拿了出来。”
<br/>　　司空轩看着楚风眠吞食的丹药，笑着说道。
<br/>　　“这可真是大手笔。”
<br/>　　“吞服如此贵重的丹药，就为了对付几只虚空白虎，看来这小子的手中，也有着不小的财富。”
<br/>　　罗辰看着楚风眠，心中暗自想到。
<br/>　　“等到进入了颜古龙帝的洞府，一定要找个机会杀了他，夺取了他的财富，还有他的血脉。”
<br/>　　接近一个时辰的时间过去，楚风眠才缓缓的飞了回来。
