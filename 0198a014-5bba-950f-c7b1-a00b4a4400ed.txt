　　山背大兄这次也可以算是孤注一掷了，不管成功还是失败，他所面临的都将是失去一切。
<br/>　　不过也正是因为这一点，也使得某些人对他的印象起了一些变化。
<br/>　　能够做到一切，足以说明山背大兄的心里还是以倭国为重的，为了这个国家他可以放弃一切，而相比较来说，秦吉胜的做法就有些落了下层，自私、独裁成了他留给所有人的印象。
<br/>　　不过，这些事情随着山背大兄交出兵权的那一刻，都显的不是那么重要了。
<br/>　　……
<br/>　　钦陵的十万铁骑驻扎在倭人百万大军的对面，彼此间有着十里的缓冲距离，不过这段距离对于靺鞨骑兵来说，几乎可以算做没有。
<br/>　　此时的钦陵并不知道倭人已经统一了思想，当然，就算是知道了他也不会在乎。
<br/>　　十万大军枕戈待旦，为了就是毕其功于一役，现在两军对峙，正好到了决战的一刻。
<br/>　　“大都督，倭人集百万大军与我们对峙，您可有什么指示？”黑齿常之趁着夜色来到了中军，见到了正在拿着一本兵书品读的钦陵。
<br/>　　“兵来将挡、水来土掩！常之，本都督相信你的能力。”钦陵放下手里的书，言语间微有调侃之间。
<br/>　　黑齿常之倒也没中钦陵客气，呵呵一笑浑不在意的抱了抱拳：“大都督，如此信任末将，末将不胜荣幸！”
<br/>　　“哈哈……”钦陵直到此时才露出一个大大的笑容：“常之啊，你这家伙忒不地道，明明自己已经有了应对的办法却要跑来故意为难我这个大都督，实在不当人子。”
<br/>　　“大都督到了这个时候您还有心思开玩笑，末将是不是应该很高兴？”黑齿常之再次被调侃之下倒也没有生气，只是无奈的苦笑了一下。
<br/>　　“这有什么，正面一战不正是我们需要的么？只要打过这一仗接下来我们就可以长驱直入，将整个倭国彻底从地图上抹去。”钦陵指了指一旁的座位让黑齿常之坐下。
<br/>　　仗打到现在这个样子，双方全都失去了改变战术的机会，而这样的对峙对大唐又是有利的，钦陵自然是放松的很。
<br/>　　不过黑齿常之却不这样认为，坐下之后看着钦陵皱着眉头说道：“大都督，不知道你发现没有，此前我们所有的战斗其实都没有什么实质性的缴获，而所谓的缴获也都是一些倭人的破烂。”
<br/>　　“是么？这个我倒是没太注意到。”钦陵微微愣了一下，然后有些严肃的说道：“你不会是怀疑倭人在丢了半个国家之后，还在保留实力吧？”
<br/>　　“我听说咱们大唐一直在对倭国输出武器装备，可是你看，仗打到现在却没有任何一件有大唐生产标识的武器装备出现，这难道还不足以说明问题么？”黑齿常之也是在无意间才发现的这一点，而发现之后，便立刻来了钦陵之里，没有多耽搁一刻钟。
<br/>　　钦陵听到这里大致上也猜到了他想要说什么，双眉也不禁微微一皱：“你的意思是倭人把自己的主力部队全都藏起来了，根本没有拿出来与我们交战，为的就是使我们轻敌，然后在最后拿出来与我们决战？”
<br/>　　“是的，我就是这个意思。”黑齿常之点了点头：“倭人打了这么多年仗，不可能一点战略战术都不懂，仗打成现在这个样子并不像传说中的他们，所以我觉得他们是在故意保存实力，给我们一个虚弱的假像，若是我们真的大意了，很有可能会被他们一个反击彻底打垮。”
<br/>　　“嘶……”钦陵倒吸了一口冷气。
<br/>　　黑齿常之所说的一切并不是不可能，钦陵认为如果自己在明知自己不是敌人敌手的情况下，最可能采用的战术应该也是这样。
<br/>　　先示敌以弱，然后积蓄力量，等到了战争的最后，当敌人以为胜券在握的时候，再全力一击，将敌人彻底打败。
<br/>　　只不过这一招需要在前期负出极大的代价，并不是什么人都能用得出来的。
<br/>　　想到这里，钦陵皱眉对黑齿常之问道：“常之，你的这个推测有多大的把握？”
<br/>　　“大都督即然已经相信了末将的判断，何必还要再问。”黑齿常之默默与钦陵对视着，神情前所未有之严肃。
<br/>　　大唐此前向倭国输送的武器可不少，就算是倭人打了这么多年的战争，也不可能消耗的一点不剩，而且战场之上什么事情都有可能发生，任何一点大意都会让己方万劫不复。
<br/>　　所以既然想到了倭人在示弱，那么就只能当成倭人在开始的时候就把所有的来自大唐的精良武器全都藏了起来，现在全部拿出来使用。
<br/>　　虽然这些武器并不如钦陵他们现在装备的武器先进，可是如果数量太大，依旧会给他们造成伤害，很有可能会让这次跨海东征彻底失败，这是钦陵和黑齿常之绝不想看到的。
<br/>　　钦陵默默的思考着一切，他与黑齿常之不同，作为此次跨海东征的行军大总管，他必须考虑全局，不会因为一个小小的可能就放弃眼前已经取得的大好成果。
<br/>　　他不能畏缩不前，这样会引起军心动荡；他又不能贸然出击，因为他的命令关系着十万大军的生存与死亡；当然，他更不能下令后撤，眼下这个情况如果撤回去，不说有没有面皮，就是他自己都有些惋惜现在取得的战果。
<br/>　　想到这里，钦陵不由一阵头大，看着黑齿常之说道：“先准备吧，必须按照倭人有隐藏实力来作打算。”
<br/>　　“问题是我们不知道他们有多强大的实力，如果只有十到二十万主力军，我们还可以应付，如果是更多那就不好办了。”黑齿常之苦笑着摇摇头。
<br/>　　“那你觉得我们应该怎么办？”钦陵继续追问，他不相信黑齿常之会在没有任何解决办法的情况下来找自己。
<br/>　　而事实证明，钦陵的猜测是对的，就在他问完之后，黑齿常之有了那么一丝的犹豫，然后才一字一顿的说道：“末将以为，最好的办法就是减兵增灶。”
