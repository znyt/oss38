　　第549章
<br/>　　入夜，琅琊城城门将闭未闭之时，一行三人出现在城门口。
<br/>　　李婉儿手撑竹杖，仰望着面前的城门，面露憧憬神色。
<br/>　　“此地便是琅琊城，诗书礼乐之乡，道德沐化所在。城中有万卷书院，孟夫子更被誉为当世第一读书人。”
<br/>　　怕李莫愁不解，她如此向李莫愁介绍。但李莫愁的人，况且李莫愁当年就是被个读书人忽悠了，半世情伤至此。所以她对读书人的感情很复杂。
<br/>　　呃……主要是不知道该用什么复杂的方式弄死他们。
<br/>　　“好啦，好啦，走了一路，我们也进城打尖住店，好好歇歇。再不进去，城门就要关上了。”李莫愁没好气道。
<br/>　　那穿着宽大黑袍的老者，已经拄着木杖一步步跨入了琅琊城，将李婉儿二人甩在身后。
<br/>　　几乎是他们刚入城不久。城门守军就准备将关城门，因为伏魔大会的事，琅琊城三教九流来了许多人。城内的治安顿时变得紧张起来，早早关城门，他们也能省些麻烦。
<br/>　　啪！
<br/>　　就在城门将关上时，一只大手扣在城门上。守军一楞，看到城门外杵着一张大脸。
<br/>　　方口阔目，皮肤粗糙，尤其是一双眼睛，直勾勾盯着守军。
<br/>　　住在琅琊城，也不可能没有见过世面。来自异域的海外商人，放浪形骸的学子，一等一的青楼歌姬，他们不知道见过多少。时间长了，也就见怪不怪。
<br/>　　但这个的眼神看着还是令人有些怕，守军愣了愣，也没有阻拦，稍微放开些城门，放这伙人入城。
<br/>　　一行七人，为首是个黑袍汉子，在他身后是个裹着斗篷的女人，皮肤有些黑，但眼睛很亮。
<br/>　　其他五人也是奇形怪状，凶神恶煞，似乎与他多说两句，对方就会张开巨口吞掉你的脑袋一般。
<br/>　　李婉儿三人入了城，一时竟找不到住店的客栈。近日琅琊城人满为患，大小客栈都已住满。不过一路行来风餐露宿，这对李婉儿来说，也不算什么。三人就在长街屋檐下露宿，李莫愁从街边店铺买来烧饼和肉食。
<br/>　　李婉儿分一些与那黑袍老人吃，老人咀嚼着烧饼，缓缓露出个微笑。
<br/>　　这还是一路上，黑袍来人第一次露出这种表情。倒也不是他有多冷漠，喜欢假装高人。
<br/>　　实在是他年纪太大了，很难再对某些事情提起精神。若是年轻时，说不得也会对李莫愁或李婉儿这样的美貌女子展现一下男人魅力。可这般年纪，年与日驰，意与岁去，面对花花世界，不仅无力也无心了。
<br/>　　李婉儿和李莫愁坐在一个角落里，李莫愁啃着半个烧饼。
<br/>　　“刚刚我买东西时听人说，明日万卷山上会有一场伏魔大会。要诛杀的魔头就是那位。”
<br/>　　“呀。”李婉儿瞬间意识到李莫愁口中言的是谁：“他不是在西边嘛，怎么来了东海？”
<br/>　　“这……我也猜不透呐。”李莫愁叹了一句：“你说，他会不会是专门寻你来的？”
<br/>　　李婉儿一时愣住，想来想去，也只有这种推测最有可能。不然无法解释，程大雷要千里迢迢从凉州来东海。
<br/>　　“何苦如此，何苦如此。”
<br/>　　李莫愁：“我劝你一句，易得无价宝，难得有情郎，既然他心诚至此，你便不该装作瞧不着。这天下大事，本也不是你我女人能参与的。”
<br/>　　李婉儿沉默下来，她现在蹲坐在大街上，忍着秋夜长风，啃着硬邦邦的烧饼，心里却忽然温暖起来。
<br/>　　而此时此刻，程大雷待在书院特意安排的暖房内，拥着暖被正睡得踏实。
<br/>　　孟玄青坐在离二人稍远一些的距离，此刻也竖起耳朵，对这种事他也是颇为好奇的。哪怕他真是世外高人，可谁说世外高人就不能有八卦之心。
<br/>　　“明日伏魔大会，东海怕有很多高手。若他此刻在琅琊城，那情形便就危险了。”李莫愁。
<br/>　　“是呀。”李婉儿一惊：“我们该及早将这个消息告诉他，叫他早早离开东海。”
<br/>　　“唉，我们现在也不知道他在何地，或许他距离此地十万八千里，也许他就在琅琊城……”
<br/>　　“若他真在琅琊城，那该如何是好？”李婉儿忧心忡忡道。
<br/>　　孟玄青支棱着耳朵听着，伏魔大会的事情他一路走来也听说了。伏魔大会要针对的是谁，他自然也很清楚。此刻听着听着，忽然心中咯噔一下，目光落在了李婉儿二人身上。
<br/>　　“不知二位从何处而来？”
<br/>　　“凉州啊，怎么了？”李婉儿下意思道。
<br/>　　李莫愁阻拦不及，李婉儿已经说出了口。她目光看向孟玄青，将老人已低下头去，嘴里嘀嘀咕咕，念叨着什么。
<br/>　　“凉州，凉州么……”孟玄青脸上忽然浮现诡异的冷笑。
<br/>　　……
<br/>　　夜深，白云绕山峰，海浪层层阵阵，不绝于耳。
<br/>　　钱观鱼心情有些憋屈，白日在程大雷处吃了瘪，后又被书院罚到后山看守藏书。
<br/>　　这未必没有夫子的一番良苦用心，少年得志，难免傲气有余，耐性不足。书院有座藏书楼，藏书万卷，帝国多少读书人连藏书楼的大门都摸不到。而钱观鱼却可以住在此地，可以随意阅览前辈高人留下的智慧。
<br/>　　若能沉下心来读上几年书，磨掉身上的浮夸之气，日后的帝国未必没有他一席之地。他毕竟还年轻，未来可期。
<br/>　　但很显然，钱观鱼现在还无法理解这一点，他百无聊赖的转悠在一栋栋书架中，心中想着怎么找夫子求情，免了这场责罚。
<br/>　　正这时候，无意往窗外看了一眼，就见几道黑影沿着侧峰上山，速度很快，矫健如猿猴。
<br/>　　看这架势，是冲着藏经楼来的。
<br/>　　楼中藏书万卷，有想着来此地读书的，也有想着来此地偷书的，这种事，以前也并非没有出现过。
<br/>　　哇呀呀。
<br/>　　钱观鱼大喜过望，如果能抓住这波偷书贼，岂不是为书院立了一件大功，自己既可以免了现在的责罚，也能将丢掉的面子捡回来。
<br/>　　“呔，什么人！”
<br/>　　他突然从二楼跳了出来。
<br/>(本章完)
