　　“道风，你居然敢拿我师姐做人质！”摇光仙子冷冰冰说道。
<br/>　　“人质就是人质，都是一样。”
<br/>　　既然对方可以拿叶少阳来威胁自己，自己当然也可以用黎山老母来威胁她。道风也不想跟她多说，重复了一遍自己的要求：“关闭阵法。”
<br/>　　“已经没有关系了。叶少阳已经获救了。”
<br/>　　道风内心震了一下。
<br/>　　小九站在悬崖边上，望着下方，怔怔地说道：“少阳获救了，只是……”
<br/>　　只是什么？
<br/>　　道风自然不会怀疑小九的话，立刻松开了黎山老母，飞奔到悬崖边上，往下方看去，在谷口的位置，找到了叶少阳等人，毫不犹豫地纵身飞了下去。
<br/>　　风之谷和捉鬼联盟的成员们紧随其后。
<br/>　　“下去！拦住他们！”摇光仙子喝道。既然叶少阳脱困，再守着阵眼下去也没意思了。之前钟馗搭救叶少阳的过程，摇光仙子在山顶都看得一清二楚，小九等人也是看见，一方想要下去帮忙，一方就死死缠住，最后谁也没能下到山谷里，眼睁睁地看着下方的一切发生。
<br/>　　“师姐，怎么样？”摇光仙子走到黎山老母面前，问道。
<br/>　　黎山老母站起来，喃喃道：“轻敌了。”
<br/>　　摇光仙子笑了一笑，道：“机会还是有的，战斗，才刚开始呢。”
<br/>　　“那你得先拦住道风。”黎山老母说完，也纵身朝悬崖下面飞了下去。
<br/>　　芮冷玉坐在叶少阳身边，抓着他一只手，放在自己膝盖上，这个时候，任何言语上的安慰都是多余的。芮冷玉因为实力有限，在之前一系列的战斗和救援行动中都没什么存在感，但她也拼尽了全力。现在的她，对外界的一切也都不在乎了，她在乎的只有叶少阳的精神状况，也只能默默地陪着他，给他一丝安慰。
<br/>　　小九也拼尽了全力。没有完全恢复修为的她，对阵摇光仙子等人很有点吃力，却是舍生忘死地拖住了好几个强者……下山之后，她立刻来到叶少阳身边，但是见到芮冷玉跟他在一起之后，自己默默地站在了他们身后，望着叶少阳的背影，没有再上前一步。
<br/>　　所有捉鬼联盟的成员，围着橙子站在一起，小白和美华互相抱着哭，其余人的眼底也尽是悲伤。
<br/>　　捉鬼联盟第一个牺牲的人，没想到……居然是橙子。
<br/>　　萧逸云抱着橙子的尸体，仰天长啸。
<br/>　　身后几个天子殿的人，望着飞身下来的摇光仙子等人，眼睛中的悲伤，完全化成了仇恨之火，二话不说就要冲上去给橙子报仇。捉鬼联盟的成员也是一触即发。
<br/>　　突然之间，一道强烈的光芒，从遥远的地方照了过来，仿佛阳光穿过云层。而青冥界是没有太阳的，因此这怪异的一幕，引起了所有人的注意，一起转头看去。
<br/>　　光芒落在不远的树林里，紧接着一闪而现，一道人影，缓缓走了过来，还是那个鬼卒。
<br/>　　几个天子殿的人一看到他，立刻放弃了进攻的念头，一个个跪在地上，有人忍不住哽咽起来。
<br/>　　“府君大人，少奶奶被人杀了！”
<br/>　　“求府君大人为少奶奶报仇！”
<br/>　　崔府君！！
<br/>　　摇光仙子等人当场惊呆了，谁也没有想到，崔府君居然会亲自驾临空界……
<br/>　　空界联盟的几大宗师猛然想到了之前钟馗说的话，那个死去的姑娘，是他的儿媳妇……顿时都感到意一丝凉气蔓延上全身。
<br/>　　“见过府君。”黎山老母已经恢复了精气神，朝崔府君拱了拱手，其余人也跟着一起行礼。崔府君走过来，看也没看他们一眼。
<br/>　　钟馗挠着后脑勺，有些失意地叹声说道：“老崔，对不住。”
<br/>　　“多谢师兄。”崔府君对他拱了拱手。
<br/>　　钟馗环视了一眼众人，目光最后落在黎山老母等人脸上，目光变冷，点头说道：“很好的。”随即竟然一只手打开了虚空裂缝，钻了进去。
<br/>　　他之前并没撒谎，他之所以来，只是为了帮崔府君救人，因此除了那些送上门挡路的，没有主动进攻任何人。
<br/>　　地位越高，越是不能率性而为，以免造成无法解决的麻烦。钟馗虽然一向不服管束，也不敢在这种事情上太过恣意放肆，能帮崔府君来这里走一趟，已经是所能做的最多了。
<br/>　　以他跟崔府君之类的关系，也根本不用说太多。
<br/>　　萧逸云怀里抱着橙子，见到崔府君之后，立刻崩溃了，眼中止不住地往下流泪。
<br/>　　“大人，求为橙子报仇！”
<br/>　　崔府君低头望着橙子的脸，脸上没有任何表情，淡淡说道：“没到那个地步。”
<br/>　　一句话，瞬间改变了整个氛围，所有人目光都落在了崔府君脸上，原本灰暗绝望的眼神瞬间明亮起来。连叶少阳也猛然抬头，朝崔府君望去。
<br/>　　元神俱灭，代表的是生灵最终极的死亡，虽然磨灭的元神，在几百年之后会重新汇聚成精魄，再被动吸收几百年的天地灵气，才可以成为鬼魂，但那时候的鬼魂，已经完全崭新的，并没有任何往昔的记忆。
<br/>　　这种消亡，本来就是无可挽回的天地大道，因此不管是叶少阳还是萧逸云，都觉得橙子是真正的死了，再也回不来了。
<br/>　　他们却忘了，眼前这个人是崔府君，拥有通判生死、逆转阴阳的不可思议的法力和权力。只要他没判一个人的死刑，就没人敢说这人死了。
<br/>　　“府君大人……”叶少阳急切地说道。
<br/>　　“你闯的祸还不够吗？”崔府君淡淡地瞥了他一眼，也没有多说什么，对萧逸云道：“回去。”
<br/>　　“大人……”
<br/>　　“回去，带着她一起回去。”崔府君的口吻不容置疑。
<br/>　　萧逸云总算是吃了一颗定心丸，既然崔府君让他带橙子的尸体回去，那就一定有办法救活她。当下看了叶少阳一眼，点点头，怀抱着橙子，往山谷外飞奔。
<br/>　　天子殿其余人也都跟着他走了。
<br/>　　一个金甲鬼武士犹豫了一下，对崔府君拜了一拜，恨恨地盯着摇光仙子等人说道：“大人，少奶奶就是被他们逼死的，求大人做主！”说完也匆忙告退。
