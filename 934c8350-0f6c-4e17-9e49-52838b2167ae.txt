　　江飞燕差点脱口喊出‘叶帅’，不过转念又想起之前，张长林说过的话，便没敢暴露他的身份。
<br/>　　不过饶是如此，她这番举动，落在众人的眼中，还是引起了巨大轰动。
<br/>　　不但众人全懵了，江飞龙懵了，就连杨之凡也已经彻底懵了！
<br/>　　心中不由得狂吼起来，
<br/>　　“这家伙到底是什么来头？不但唐家太子爷对他恭敬有加，怎么连江飞龙的姐姐，也对他如此敬畏？”
<br/>　　“根据我之前调查的资料显示，这家伙明明没有什么背景啊！怎么会这样？”
<br/>　　杨之凡越想越想不通，心中不由得升起一股强大的危机感。
<br/>　　之前，他一直都是整个生物学院，最闪耀的一颗星，可是如今叶尘横空出世，一下子把他的风头，全都给抢跑了。
<br/>　　叶尘淡淡一笑，
<br/>　　“江警官，没想到这么快咱们又见面了！”
<br/>　　江飞燕见叶尘记得自己，顿时心头一喜，旋即又想起了自己弟弟的那番话，当即道：
<br/>　　“不知道我弟弟怎么得罪了叶先生，还请叶先生千万海涵！”
<br/>　　说完这话，江飞燕转过头，狠狠瞪了一眼江飞龙，
<br/>　　“还愣着做什么？还不过来向叶先生磕头道歉！”
<br/>　　江飞龙顿时两眼一瞪，简直怀疑自己的耳朵出了问题。
<br/>　　虽然此时此刻，他也看出来了，这个比他还要年轻的少年，绝对不是简单人物。
<br/>　　可是，他纵然是某个超级大家族的公子哥，也不值得让他江飞龙磕头道歉吧？
<br/>　　他可是堂堂江家的嫡子，将来江家家主的继承人！
<br/>　　“姐，你没有搞错吧？你让我给他磕头？这不可能！”
<br/>　　当着这多人的面，向一个比他年轻的少年磕头，江飞龙说什么也做不出来。
<br/>　　江飞燕顿时急得不行，真想立刻把叶尘的身份，告诉自己这个傻弟弟。
<br/>　　就在僵持之时，叶尘忽然开口了，
<br/>　　“罢了，既然他是你的弟弟，那我就看在你的面子上，不和他一般见识了！”
<br/>　　江飞燕顿时大喜，
<br/>　　“多谢叶先生！”
<br/>　　叶尘说完这话，便想要转身离开，没想到唐清山快步追了上来，
<br/>　　“叶先生，我有些事情想和你聊一聊，是关于我妹妹清雅的！”
<br/>　　叶尘眉头微微一挑，点了点头，答应了下来。
<br/>　　很快，
<br/>　　叶尘随着唐清山，重新返回酒店之中，
<br/>　　不过，
<br/>　　在路过那王少和方少两人跟前的时候，看到两人眼底闪过一抹不屑之色。
<br/>　　叶尘见状，顿时脸色一寒，
<br/>　　他本不想杀人，可是有些人，你若是不给他一个教训，他们便以为你是软柿子！
<br/>　　“既然是这样的话，那便死去吧！”
<br/>　　一念及此，在迈入酒店的那一刹那，叶尘直接两道神念攻击了过去！
<br/>　　噗！
<br/>　　噗！
<br/>　　那方少和王少两人，顿时如遭雷劈，全身猛然一震，继而便软绵绵的倒了下去，
<br/>　　“方少！”
<br/>　　“王少！”
<br/>　　两位大少的手下们见状，当即纷纷冲了上前来，扶住两人，可是任凭他们怎么呼喊，两人都毫无反应。
<br/>　　“死……死了？”
<br/>　　“怎么会这样！”
<br/>　　众人发现两人已经气绝，顿时一个个毛骨悚然，全都想不明白，到底发生了什么事情。
<br/>　　而另外一边，原本还一脸不服气的江飞龙，看到这一幕，也吓得浑身发抖，忍不住一边抓住江飞燕的胳膊，
<br/>　　“姐，这……这怎么回事？难道是他……”
<br/>　　江飞燕同样倒吸了一口凉气，道：
<br/>　　“应该就是他！果然是神仙手段啊！”
<br/>　　江飞龙顿时急了，
<br/>　　“姐，既然知道是他干的，那你还等什么？还不赶快把他抓起来，否则他对我下手该怎么办？”
<br/>　　江飞燕顿时眉头一皱，
<br/>　　“我怎么有你这么一个蠢货弟弟！就算明知道人是他杀的又如何？你能找到证据吗？”
<br/>　　“更何况，以他的身份和能耐，就算是把整个天海市掀个底朝天，国家也不会动他分毫！”
<br/>　　江飞龙顿时两眼一瞪，
<br/>　　“怎么会这样？他……他到底是什么身份？”
<br/>　　江飞燕再次摇了摇头，
<br/>　　“事到如今，你怎么还想不到？当真是蠢死了！”
<br/>　　说到这里，江飞燕一脸的怒其不争，接着又道：
<br/>　　“他姓叶，有来自云州，能让唐清山如此恭敬对待，还能是谁？”
<br/>　　经江飞燕这么一提醒，江飞龙顿时恍然大悟，继而一脸的惨白之色，结结巴巴道：
<br/>　　“姐，你的意思，他、他是云州那位叶……”
<br/>　　江飞龙总算猜出了叶尘的真实身份，顿时两腿一软，险些栽倒在地上，多亏江飞燕一把将他拉住，这才没有当众出手。
<br/>　　但是此刻，江飞龙哪里还会计较，会不会出丑的问题。
<br/>　　一想到自己，竟然把那位华夏仙人给得罪了，江飞龙就恨不得去撞墙。
<br/>　　想当初，仙人一怒，天海四大家族，尽数被灭！
<br/>　　然后方才有他们三大家族，如今异军突起，引领风骚。
<br/>　　江飞龙丝毫不怀疑，今日若非江飞燕来恰好在场，恐怕此时也已经和另外两位大手一般，如今早已经横尸街头！
<br/>　　一想到这里，江飞龙就出了一身的冷汗，顿时急的不行，
<br/>　　“姐！姐！你说叶先生他……他不会秋后算账吧？他会不会把我也杀了，把我们江家给灭了？要不你替我去向求求情吧？”
<br/>　　江飞燕幽幽叹了一口气道：
<br/>　　“这个我也说不了！不过，以他的身份，既然方才没有杀你，应该就代表不会继续追究了！”
<br/>　　江飞龙却还是不放心，
<br/>　　“那万一他要是……姐！我是你亲弟弟，你可以一定要帮帮我啊！”
<br/>　　“唉！”
<br/>　　江飞燕重重叹了一口气，道：
<br/>　　“早知现在，何必当初！罢了，你现在速速给我回家，好好闭门思过，我这里等着，等他出来之后，我会再替你求求情，希望可以有用吧！”
<br/>　　……
<br/>　　却说叶尘随着唐清山进入酒店之后，来到一处豪华包厢之中。
<br/>　　“叶先生，这三个不知死活的家伙，竟然敢冒犯到您的头上，要不要我动用唐家的力量，直接将那三大家族给灭了？”
<br/>　　叶尘轻轻摇了摇头，
<br/>　　“不必了！不过就是几个狂妄小辈而已，没必要和他们置气！”
<br/>　　说到这里，叶尘微微一顿，
<br/>　　“你之前说关于清雅的事情，到底是何事？”
<br/>　　唐清山听到叶尘提及这个，脸色顿时变得有些凝重。
