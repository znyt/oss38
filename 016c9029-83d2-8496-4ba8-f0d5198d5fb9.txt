　　第507章
<br/>　　“啊呀呀！”
<br/>　　李行哉长叹一声，赶忙将戚继光扶起，上上下下，仔仔细细打量戚继光一遍。
<br/>　　此刻李行哉惊讶大过欢喜，心道：这小子脑袋不太清楚吧，投奔我，我有什么好投奔的。
<br/>　　“壮士快快请起。”李行哉将戚继光扶起，长叹道：“现在我无兵无将无钱无地盘，壮士一身武艺，是不是该好好考虑清楚，莫因本王耽搁了本王前程。”
<br/>　　“仁义自可招兵，有兵自然有粮，逍遥王仁义为怀，为保百姓不顾生死，刚才某已看得清楚。若王上看不起在下的武艺，某自可离去，不让王上为难。”
<br/>　　终于有个人来投靠自己，李行哉怎么可能往外赶。他道：“此地不是说话的地方，壮士请先随我回府。”
<br/>　　“王上请。”
<br/>　　李行哉没有回府，只见李行哉拖着瘸腿，一步一步将海盗的兵器都收起来。
<br/>　　“这是做何？”
<br/>　　“来来来，先把他们的兵器都拿了，这可都能换钱，等我先把欠粮铺的债还了。”
<br/>　　“呃……”戚继光有些无语，这位逍遥王……的确惨点儿。
<br/>　　好不容易回到城主府，里外也没人伺候，只能李行哉自己烧水，给戚继光倒茶。戚继光是做也不是，站也不是。让一位王爷亲自烧水倒茶，这怕是从未有过的事。
<br/>　　“不知壮士从何处来的？”李行哉好不容易收拾好，摘掉帽子抓虱子。
<br/>　　“启禀王上……”戚继光腾地一下站起来。
<br/>　　“坐下说话，坐下说话。”李行哉摆摆手：“这里只有咱们两个人，以后就以兄弟相称，日后咱们混好了，一起吃酒一起吃肉。”
<br/>　　戚继光有些无语，这逍遥王的王爷派头是一点都没看见，怎么感觉像个山贼。
<br/>　　“在下是扬州人士，家中有几艘大船，幼时也随高人学过武艺，兵马韬略，后来家道中落，便想靠一身武艺，奔一番前程，不给祖宗蒙羞。”
<br/>　　“蛮好，蛮好。”李行哉有些不以为然，心道这戚继光年纪不大，口气倒是不小，还一身武艺，我怎么没看出来。
<br/>　　他是不知道这戚继光的本事，如果换到蛤蟆城，程大雷非得将这戚继光奉为上宾不可。戚继光强不在武艺，也不是用兵的谋略，而在练兵，有此一人，何愁百万精兵。
<br/>　　“你可能不知落凤城现在的情况，现在城内无人，多是一些老弱妇孺。”
<br/>　　“依在下看，此事不难，连年战乱，帝国多有流民，我们广开贤路，聚集天下英雄当也不难。”
<br/>　　李行哉顿了顿：“还有山贼海盗作乱，依你见识，该如何是好？”
<br/>　　“杀了便是。”戚继光平静道。
<br/>　　李行哉眯起眼睛，上三眼下三眼打量戚继光，忽地哈哈一笑，倒是吓得戚继光不轻。这王爷，该不会脑袋有些不清楚吧。
<br/>　　“好好好，我得将军如旱苗得雨，猛虎插翅，日后你就是我落凤城的兵马大元帅。”李行哉豪气纵横：“城中的一百老兵，归你指挥。”
<br/>　　……
<br/>　　差不多李行哉遇见李行哉的时候，程大雷也见到了史文恭。这员将持一杆方天画戟，特意前来投奔程大雷。
<br/>　　本来程大雷不会专程见他，不过最近委实没太多事，也看看这史文恭的人物相貌。所以，刘悲引着史文恭进入城主府，来到程大雷面前。
<br/>　　程大雷上上下下将史文恭打量一遍，三十左右年纪，着一件灰布长袍，身长八尺有余。方脸阔面，虎背熊腰，端地是一俊品人物。
<br/>　　不过，程大雷手下不缺这样的大将呐。武有秦蛮、关张赵，就算高飞豹高飞虎实力也是今非昔比，就连那赤发灵官单雄信还在程大雷这里歇着呢。
<br/>　　史文恭双手抱拳：“闻听程当家在江湖上的英雄名号，某家特意前来投奔，愿为大当家效犬马之劳。”
<br/>　　“好，好。”程大雷拍拍手，确实显得有气无力，道：“既然到了，便是兄弟。现在军中无官职空缺，你先到军中任一员教头，日后若有机会，再听差派。”
<br/>　　史文恭听到这话，重重冷哼一声，道：“某学的文武艺，要与程当家卖命，区区一个教头之位，怕不能给程当家出力。”
<br/>　　“喔。”程大雷顿时来了精神：“如此，你是不服了？”
<br/>　　“史某情实的不服。”
<br/>　　“不服的好，不服的好。”程大雷倒也没生气：“在我山寨之中，便该有如此战意，你既然不服差遣，我便给你个机会。”
<br/>　　“我山寨有五虎将，秦、关、张、赵、高，我若让他们与你对手，算是欺负你。如此我随便点一人，你若能胜过他，你想要带军，我给你个将军的位置。”
<br/>　　“好！”史文恭干脆答应：“便是城中五虎，我也不怕。”
<br/>　　“你若胜了自然一切都好。”程大雷话锋一转：“不过，你初来乍到，便不听差遣，若是输了，我可有军令伺候。”
<br/>　　“史某若是输了，愿听大当家责罚。”
<br/>　　“好，去军营。”
<br/>　　程大雷与刘悲引着史文恭离了将军府，直奔军营校场。史文恭也悄悄打量着蛤蟆城的景象，只见士兵各个精神，城中商铺兴旺，虽然只是一座小城，却有些富贵逼人的繁华气派。
<br/>　　军营内听得这件事，一帮莽汉都激动起来，将军中的擂台围得水泄不通。
<br/>　　史文恭翻身上了擂台，将方天画戟立在一旁，抱拳道：“诸位，史某初来贵地，愿为程当家效犬马之劳。不过，史某也不愿被人轻视，以为我无本事。诸位若有看得上史某的，可上台切磋武艺，也显一显史某的武艺。”
<br/>　　关鱼眼睛眯成一条线，道：“这人倒是个直性汉子。”
<br/>　　程大雷目光转了一圈，落在单雄信身上，道：“单兄弟，你到山寨后，我还未愈你安排差事，若能胜得此人，我便给你个位置。”
<br/>　　单雄信早已迫不及待，史文恭想要显一显本事，他何尝不想在人前露脸。
<br/>　　程大雷的声音刚落地，他便窜上擂台，拿起金顶枣阳槊，口中喝了一声，如同狮子啸山林。
<br/>　　“请啦。”
<br/>(本章完)
