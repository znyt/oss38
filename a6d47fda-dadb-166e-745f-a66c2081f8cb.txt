　　第397章
<br/>　　“已经派了人手打探凉州城的情报，但我们的人没办法潜入城主府。大当家，不如让白元飞试试。”
<br/>　　白元飞，绰号飞天蛤蟆，一身轻功，也就稍弱于程大雷。但他本不会什么轻功，外界传言飞天蛤蟆自幼被白猿收养，无名无姓，无父无母，等到十几岁才下山。传言只是传言，事实当然不会如此。
<br/>　　事实上，飞天蛤蟆少小贫贱，在山中采药为生。值钱的草药都长在悬崖上，想要活下去，就得拿自己的命冒险，于是生得长手长脚，在悬崖峭壁如履平地。后来，他这种天赋，被某个世家看重，训练成为死士。在一次任务失败时，被程大雷所救，于是才成了程大雷的手下。
<br/>　　程大雷点点头：“这件事你安排就行，不需要向我汇报，只是要注意安全。”
<br/>　　阿喜眼中一亮，强压眼中的激动，重重的点了点头。
<br/>　　“明白。”
<br/>　　目前，山寨的情报工作由阿喜来做，实话说，做得不错。阿喜曾经给人做账房先生，心是细的，如今做情报工作，也发挥了这方面优点。琴川关周围的城池，都被他埋入暗线，甚至也派了间谍打入某些大人物身边。如今源源不断的情报汇到程大雷面前，凉州一地的风吹草动，都逃不过程大雷的耳目。
<br/>　　既然如此，程大雷索性全权放手，自己也不去裹乱。
<br/>　　阿喜明白这是怎样的信任，自己从长安逃到这里，刚来不久就能被赋予这样的重任，其实十分难得。自己必须将事情做到最好，才能不辜负这信任。
<br/>　　离开将军府后，阿喜没有丝毫耽搁，便找到了飞天蛤蟆白元飞。
<br/>　　在琴川关，喜字队并不是一个受人欢迎的存在，因为他们总是鬼鬼祟祟，神秘兮兮，你若问他们什么，常常一问三不知，若是问得急了，回答一定是：机密，无可奉告。
<br/>　　大家不仅有些郁闷，哪里这么多机密啊。其实整个琴川关，本也没有太多秘密。不过阿喜觉得，做事么，就是要专业一点，情报部门，没有秘密，也得搞得像有秘密的样子。
<br/>　　白元飞就不是很能接受这一点，每次见面，都要搞得像做坏事一样，非得安排在暗房，没有窗户，外面的光线透不进来，只有一盏昏暗的油灯。阿喜的脸隐在黑暗中，像某位幕后黑手一样。
<br/>　　“这次又有什么事？”白元飞没好气道。
<br/>　　“这次任务非常重要，大当家亲自吩咐下来的，整个山寨除了你，没有人能完成。”
<br/>　　“好啦，好啦。你每次都是这么说的，你有什么事，就能不能好好说话？”
<br/>　　阿喜揉了揉脸：“不好意思，职业习惯。”
<br/>　　“究竟什么事吧？”
<br/>　　“李行哉去了凉州城后，凉州城一直没有动静，大当家有些放心不下，所以让你去看一看。”
<br/>　　“好吧，目前什么情况。”白元飞认真起来。
<br/>　　“现在凉州城对进城的人盘查得很严，进出都很困难，目前李行哉在城主府，想要打探到他们确切情报，你怕是要潜入城主府。”
<br/>　　白元飞点点头，事情比自己想得要困难，城主府防卫森严，想要悄悄潜进去，并不是一件很简单的事情。
<br/>　　当然，正因为不容易，方才需要用到自己。如果是随便就能打探到的情报，也不需自己出马。
<br/>　　这种自信，白元飞还是有的。
<br/>　　当天，白元飞就出发了。他骑了匹马，在靠近凉州城时，把马藏在山坳里，换过衣服后，就开始等着。
<br/>　　一直等到夜深，白元飞才开始行动。他着一身夜行服，口鼻都藏着，只有一双眼睛露在外面。
<br/>　　夜里的凉州城城门紧闭，周围黑漆漆的一点动静也无，在城墙上，有一队队卫兵来回巡逻。
<br/>　　白元飞将身子贴在城墙上，黑衣与墙壁融为一体，手指勾着墙缝，身子一点一点向上爬。为了今天的任务，白元飞拿出了看家本事。他也是初来乍到，要做出点事给其他人瞧瞧。
<br/>　　当然，他的本事无人欣赏。
<br/>　　头顶突然传来脚步声，两名巡逻的卫兵从城墙上经过，此刻竟然停了下来。
<br/>　　“这天寒地冻，非得让我们弟兄受这罪。”
<br/>　　“嗨，如果不是你得罪了队长，我们怎么会被派这差事来。”
<br/>　　二人说着话，却不知白元飞就趴在城墙外，用手指搭着城头垛子。只要他们扫一眼，就能看见白元飞，但白元飞的心情很平静。他不愧有一个出色小偷的素质。
<br/>　　现在他只期盼着两名卫兵可以早些离开，自己就能成功潜入凉州城，任务也就成功了一半。
<br/>　　但这次，白元飞没有幸运女神的眷恋，两名卫兵嘴里骂个不停，有一人走到城垛前，下意识向城下往了一眼。
<br/>　　然后他就撞见了白元飞一双明亮的眼睛，很明显是吓了一跳。深更半夜，有人趴在城墙上，也说不好是人是鬼。
<br/>　　但留给他吃惊的时间不多，白元飞突然起身，手中的匕首毫不犹豫的刺入他的喉咙，随即如一头暴怒的豹子，扑向另外一名卫兵，匕首捅入对方的心脏。
<br/>　　两个人临死前，连一道呼声都来不及发出，就失去了生命。白元飞脚下多了两具安静的尸体。
<br/>　　这个时候，已经可以宣告白元飞行动失败，最正确的选择是立刻离开，寻找下一次潜入的机会。当然，下一次凉州城的防卫会更加森严。
<br/>　　但白元飞丢不起这个人，初来乍到，他也一直在寻找露脸的机会。好不容易碰到一次，难道就这样灰溜溜回去，琴川关的人会怎么看自己，还不觉得从长安城过来的人都是废物。
<br/>　　白元飞决定冒险，他将两名卫兵从城头丢下去，见此地的动静没有惊动其他人，便顺着城墙爬入凉州城。
<br/>　　当然，他也明白，很快就会有人发现两名卫兵被杀死，然后，就可以推断出有敌人潜入，紧跟着就是覆盖整个凉州城的大搜捕。
<br/>　　所以，留给白元飞的时间并不多。
<br/>　　他顺着屋脊前进，足尖轻轻一点地，就是数米距离，像极了山中矫健的白猿。
<br/>　　
<br/>(本章完)
