　　俗话说得好，谋事在人成事在天，小鬼子的如意算盘打得好，但能不能实现还要看王玄策的选择和李承乾的决心。
<br/>　　一心想要摆脱大唐控制的倭人并不知道，大唐其实一直在做的就是压榨他们的战争潜力，至他们到底是不是真的在打仗……，好吧，其实李承乾还是比较关心的。
<br/>　　但作为一条狗，总是想要摆脱主人的控制这是不对的，秦吉胜并不知道，当他们真的想要摆脱大唐的那一天，就是他们灭望的一天。
<br/>　　李承乾已经为此做了很长时间的准备，登州和海参崴都驻扎有大量的部队，只等一声令下便会开进倭岛。
<br/>　　反抗？面对坚船利炮，一切反抗都是多余的。
<br/>　　不过为了表示此行目的的单纯，秦吉胜还是带着人看了几场球赛，顺便在长安城转了几圈，欣赏了一下长安城的夜景。
<br/>　　原本可以容纳两百万人口的长安因为一个球赛，硬生生的塞进了近四百万人，这使得长安城到处都是拥堵不堪，一次出行之后秦吉胜便放弃了再次出行的打算。
<br/>　　原因？原因就是他们个子太小，出行太受气，就算是一个十四、五岁的孩子都可以轻轻松松的把他们队伍挤散，一个晚上下来啥都没看到不说，人还被挤丢了十来个。
<br/>　　……
<br/>　　缩在驿馆中的秦吉胜一等便是两个多月，王玄策像是消失了一样再也没有出现，而大唐其他的一些官员虽然会在看在钱的份上见他们，但显然都不是能做主的人。
<br/>　　面对他们送上来的钱往往都是答应的挺好，可转过身便再也没了动静，就好像当初什么事情都没有发生过一样。
<br/>　　到了后来秦吉胜也看明白了，其实在大唐根本就没人在乎他们，倭人的死活与唐人没有一丝一毫的关系。
<br/>　　有了这个念头之后，秦吉胜心中暗自发狠做了一个任谁都想不到的决定。
<br/>　　这一日，乔装改扮之后的秦吉胜的儿子，也就是前文提到的那个青年带着几个人离开了驿馆，走向长安城外的马球场，看上去就像是要去看马球比赛一般无二。
<br/>　　不过当他们出城之后，立刻在人群中钻来钻去的穿插起来，不多时已经离开了球场，来到了通往长安城外的火车站，接着买票，上车，一路无话到了咸阳。
<br/>　　“怎么样，发现有人跟踪没有？”离开咸阳车站，青年带着几人进入城中，找了一条小巷子钻一了进去。
<br/>　　跟着青年出来的一个护卫在巷口微微探头，扫了眼身后，回过头说道：“没有，没有发现熟面孔。”
<br/>　　“再看看！”青年一脸的阴霾，在小巷两侧的暗影中显的有些阴森。
<br/>　　“噫！”刚刚那个护卫答应着，再次探出了半个脑袋。
<br/>　　这一次他看的时间比较长，隔了好长时间才缩回头说道：“殿下，外面一个人都没有。”
<br/>　　“嗯，看来唐皇还真是相信我们，竟然没有派人跟着我们。”青年哼了一声，露出一抹不屑的表情。
<br/>　　“殿下，接下来我们做什么？”另外一个一直护在青年身边的倭人见没了危险，略略松了一口气上前问道。
<br/>　　“你们出去一个人，去打听一下那个所谓大唐科学院在什么地方，然后再回来这里找汇合。”青年沉思了一会儿抬起头吩咐了一句，待看到护卫中有一人离开之后，才继续说道：“我们这次的任务就是抓人，一会儿等知道大唐科学院的位置，我们就赶过去，不管里面出来的是什么人，找机会把人抓到，明白吗？”
<br/>　　“明白了！”几个倭人护卫点点头，抓人这种事情他们干过不少，并不算是陌生，区别只是这一次他们是在大唐的土地上干这种事情。
<br/>　　又等了一会刚刚出去那人回来了，一脸兴奋的说道：“殿下，打听到了，就在城外不远，而且位置很显眼。”
<br/>　　“那还等什么，马上出发。”青年似乎也有些心急，急吼吼的便带队离开了巷子。
<br/>　　他们的身高与唐人有着太大的区别，只要不是傻子一眼就能看出他们倭人的身份，所以他们必须加快速度，尽量不让人发现他们的身份。
<br/>　　……
<br/>　　可能是因为长安正在召开马球赛的关系，咸阳城中并没有多少人。
<br/>　　青年带着几个护卫躲躲闪闪，在不引人注意的情况下离开了咸阳，向着传说中的大唐科学院走去。
<br/>　　不过当他们兜兜转转之下，远远的看到科学院大门时，几乎所有人都傻了。
<br/>　　在青年和几个护卫的印象中，大唐科学院应该就是一个院子，里面有几间房子，门口站着两人守卫的那种。
<br/>　　可是出现在他们眼前的是什么？那明明就是一座没有城墙的城池好不好，回身看看不远处的咸阳城，相比较之下青年发现，似乎咸阳城还没有那个大唐科学院大。
<br/>　　几个来自于倭国的土鳖中觉得咸阳城就已经够大了，骤然见到一个比城市还要大的研究机构，以他们有限的理解能力有种跟不上节奏的感觉。
<br/>　　这特么好尴尬啊，如此大的科学院到底要抓什么人？谁知道抓来的人是干什么的？万一是个扫地的，岂不是白抓了？
<br/>　　几个护卫看着眼前的一切也有些懵、、逼，舔了舔干裂的嘴唇：“殿下，这，这可怎么办？”
<br/>　　“我特么怎么知道！”青年愣愣的回答道。
<br/>　　此时的他已经完全没了主意，这个地方实在是太大了，如果没有一个目标的话，根本没有办法抓人，如果盲目的抓了，只会打草惊蛇。
<br/>　　可如果不抓，那么他们一路冒着被发现的风险，就等于白来了。
<br/>　　怎么办？到底应该怎么办？青年急的是抓耳挠腮。
<br/>　　想到来此之前老爹交待必须要打听到大唐最新装备的制造技术，青年只觉得自己快要被逼疯了。
<br/>　　就在青年抓破脑袋想不到办法的时候，一个护卫抽了抽鼻子说道：“殿下，要不试一下吧，我们可以先抓一个人，如果打听不到什么就马上回去，如果能打听到，我们就再待一天。”
