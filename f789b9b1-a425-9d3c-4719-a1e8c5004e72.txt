　　他们对人间法师的印象一直都是弱小两个字，就算上一次叶少阳在众目睽睽之下战胜黄观主，也被认为是用了计谋的侥幸取胜。
<br/>　　况且黄观主只是众阁派四座山峰其中一座的观主，并不是空界第一流的强者。甚至连他们这对梨山嫡传的夫妻都不如。
<br/>　　“给你个单打独斗的机会。别说我们以多欺少。”
<br/>　　道姑一剑刺来，口中一道咒语念过，剑锋抖动之间，飞出无数灵光，宛如万箭齐发，朝叶少阳射来。
<br/>　　她的夫君也是师兄，就提剑站在一边看着，对妻子能够取胜一点都不担心。
<br/>　　“今非昔比了。”
<br/>　　叶少阳挺剑而上，用七星龙泉剑的剑诀相斗片刻，找了个机会，从腰带里摸出五张暗金神符，贴在剑锋上。来之前他就知道会有一场恶战，因此做了足够充分的准备，光是暗金神符就画了十几张放在身上。
<br/>　　“天地执法，七星长鸣！”叶少阳咬破舌尖，喷了一口血在剑锋上，将七星龙泉剑的灵力激发到极致，然后用这极致的力量一瞬间激活了五张暗金神符，长剑朝着道姑挥去。
<br/>　　没有什么花样，也没有平时那些奇思妙想的阴招，因为没有足够的时间，叶少阳这一刻化身成了道风，开大硬上。
<br/>　　五道暗金神符，一瞬间释放出了恐怖的灵力波动，瞬间冲开了道姑布置的剑阵，道姑表情呆滞，想躲开已经来不及，眼看着将要被灵力波动击中的瞬间，她的夫君猛然出手，斜刺里对叶少阳刺出一剑，想要逼他收招。
<br/>　　叶少**本没看他。在决定使出最强手段、一击制敌的时候，他已经注意到了身边还有一个潜在的威胁。但是没有办法，自己全神贯注控制神符之际，根本不可能再有余力应付来自其他方向的攻击，否则用常规手段以一敌二，叶少阳十分清楚，自己绝不是面前这对夫妻的对手。
<br/>　　所以，自己只能拼着接下这一击，受伤是肯定的，只希望别被打死。
<br/>　　然而这一幕并没有发生。斜刺里一剑飞来，荡开了道士手中的剑，虽然余势不减，不过总算是为叶少阳争取到一点时间。
<br/>　　轰……
<br/>　　五道暗金神符的灵力，一瞬间轰碎了道姑情急中布置起来的结界，重重地打在她身上，整个人倒飞出去，砸在地上，大口吐血起来，挣扎着刚坐起来，叶少阳的七星龙泉剑，已经顶在了她的眉心处，只要轻轻一挺，刺进她的脑门，立刻就能让她元神俱灭。
<br/>　　道姑一动不敢动，眼中闪烁着愤怒和不甘，更多的还是感到羞愧，自己，居然输给了一个自己最最看不上的人间法师。
<br/>　　被自己轻视的人打败。这种感觉，是最让人难以接受的。
<br/>　　叶少阳却不管她在想什么，听见身后有打斗声，转头看去，居然是四宝和吴嘉伟，两个人围着那道士正在厮打。不用说，刚才那横空飞来救了自己的那一剑，是吴嘉伟使出的。
<br/>　　“别打了。”叶少阳一声喊，三人一起转头，四宝和吴嘉伟立刻明白，退到一边。
<br/>　　那道士看到叶少阳用剑指着自己娘子，顿时就慌了，想要开口呵斥，又觉得没有必要，而且叶少阳似乎也没有动手杀人的意图，于是强迫自己冷静，问道：“要怎么样？”
<br/>　　“人间法师。”
<br/>　　叶少阳笑了笑，收起剑，朝山谷里走去。
<br/>　　四宝和吴嘉伟愣了一下，看了看那道士，立刻去追叶少阳。
<br/>　　那道士冲到娘子身边，检查了一番，受伤是挺重的，不过元神和肉身都没有大伤，回去就能养好，一颗心这才完全放下了，神情复杂地望着叶少阳的背影，叹了口气。
<br/>　　虽然他还有与叶少阳一战的心思，不过那也只有等以后了——叶少阳最后时刻收剑，意思很明显：饶你媳妇不死，别再纠缠。
<br/>　　之所以没有明说，是给他一个台阶下，也相信他是一个君子。
<br/>　　“叶少阳！”
<br/>　　叶少阳站住，转过头看他。
<br/>　　“你记住了，我叫陈留王。将来我与你仍有一战！”
<br/>　　叶少阳笑了笑，“奉陪。”
<br/>　　陈留王抱起娘子，飞身往梨山去了。
<br/>　　“你们怎么来了？”叶少阳问四宝和吴嘉伟。
<br/>　　“来帮你。来都来了，没什么好说的。”吴嘉伟道。
<br/>　　四宝打了叶少阳一拳，“就说你不要装逼，看看，我们不来，你差点就挂了。”
<br/>　　叶少阳没理他，他跟四宝之间，患难与共那么多次，真的没什么好说的。
<br/>　　望着吴嘉伟，咧嘴一笑，如果说上次西域那件事，让他接受吴嘉伟成为了捉鬼联盟的成员，那么就在这一刻，他们也成为了真正的兄弟。
<br/>　　“随我杀敌！”
<br/>　　叶少阳说完，飞身而起，朝山谷深处冲了过去，一路上遇到不少捉对厮杀的，各自都有对手，也都没管他们。只有十二门徒中的一人被一个样子像是牛魔王的家伙杀死，这家伙头上生角，身形如牛，不过身上穿着道袍，已经被撑开了。
<br/>　　一看就是修道的妖，在打斗中现出了本尊，道风门下十二门徒个个修为极深，是从风之谷十万大军中选出来的强者，不然也不可能支撑住这些宗师大能的狂攻。
<br/>　　这个什么牛妖能杀死一个门徒，修为自然不用说。这家伙也被打出了真火，杀了风之谷门徒之后，立刻瞄上了叶少阳三人，扑了过来。
<br/>　　叶少阳三人联手，费了老大一番力气，总算将其打伤，反正暂时也失去了行动力，于是没杀他，丢下他继续往山谷深处走去。
<br/>　　北斗七星阵早就已经寂灭了，战场也已经转移到了山谷腹地，毕竟这里空间更为广阔，不过战场的核心，就是几个人而已：
<br/>　　在那尊铁塔的最高处，站着几道人影，叶少阳扫了一眼，是杨宫梓和建文帝，正在围攻李浩然，双方打得十分激烈，叶少阳看了一会，三人不分胜负，心中不由感到震惊。杨宫梓的实力不用说了，就算是建文帝，实力也在自己之上，两个这种级别的强者围攻李浩然一个，居然也只是平手……
