　　大概安排好事宜，执事们忙碌于给重伤的族人疗伤。萧逸和长老们则直接去了家族大堂。
<br/>　　刚进大堂，已经看到大长老再次吐了一口鲜血，咳嗽起来。
<br/>　　萧逸一惊，连忙上前为其把脉，道，“大长老怎会伤得如此严重，刚才的战斗明明是游刃有余。”
<br/>　　三长老沉声道，“之前二长老未来帮忙前，大长老一人力敌那慕容山和另外两个陌生武者，自然受了不轻的伤势。”
<br/>　　“什么？一人抵挡三个先天九重？”萧逸吃了一惊。
<br/>　　大长老甩开萧逸的手，道，“我的伤势无碍，倒是矿脉那边如何了？”
<br/>　　萧逸大概解释了一遍。
<br/>　　大长老闻言，欣慰地点了点头，道，“好，很好，萧逸你果然没有让我等失望。倒是五长老他，哎...”
<br/>　　七八九长老在一旁，纷纷羞愧地低下了脑袋。
<br/>　　二长老冷声道，“哼，幸好你们三人不像五长老那般丧心病狂。”
<br/>　　三位长老连忙躬身道，“我等是萧家子弟，自然不能做那种损害家族，愧对祖宗之事。”
<br/>　　“好了。”大长老摆摆手，道，“你们三人的秉性我是清楚的，之前也是五长老拉拢的你们，今日你们死战慕容家长老，可见对家族之忠心。”
<br/>　　“大长老明鉴。”三人再次躬身。
<br/>　　这时，大长老面露无比欣慰之色地看着萧逸，道，“萧逸，幸得有你，萧家这次的危机才能渡过。”
<br/>　　“慕容家处心积虑多时，忽然袭击我萧家，又找了那么多帮手，却因为你的回归而彻底坏了他们的阴谋。”
<br/>　　“小小年纪，便有着先天实力，加之聪慧过人，心智成熟，很好。萧家下一代有你，我们这些老家伙就算死了，也能放心了。”
<br/>　　萧逸顿时一惊，怎么感觉大长老似乎在交待遗言似的。
<br/>　　这时，大长老沉声道，“吩咐下去，所有族人即日撤离紫云城，不得有误。三长老，你保护萧逸和其他年轻子弟先走一步，务必保他们平安。”
<br/>　　“慢，到底怎么回事？”萧逸惊疑问道。
<br/>　　大长老叹了口气，道，“你可知道慕容墨？”
<br/>　　萧逸点点头，道，“知道，不就是慕容家久未谋面的大长老吗。”
<br/>　　大长老道，“不错，但不是久未谋面，而是根本就是多年前就离开了紫云城，最近才忽然联系了慕容山。”
<br/>　　“今日慕容家那些生面孔的武者，还有那两个先天九重武者，就是慕容墨派遣来的。”大长老咬牙说道，显然恨透了那些入侵萧家的人。
<br/>　　萧逸一惊，道，“先天九重武者何等尊贵，哪怕是紫云城，也不过寥寥几个，而且全是三大家族的最重要人物。”
<br/>　　“那慕容墨竟然能派遣两人来攻打萧家，难不成....”
<br/>　　萧逸想到了一个可怕的念头。
<br/>　　大长老看了萧逸一眼，脸色难看地说道，“不错，慕容墨是洞玄境武者。离开紫云城多年，这次回归，他突破了。”
<br/>　　“洞玄境？”
<br/>　　此言一出，整个萧家大堂所有长老脸色大变。
<br/>　　洞玄境武者代表什么，没人不清楚，这个境界的武者有多可怕，也都人人知晓。
<br/>　　洞玄境武者，在紫云城属于传说，从来不曾出现过。
<br/>　　他们已经拥有飞天遁地的本事，拥有莫测的能力。
<br/>　　仅凭一人，就能横扫紫云城三大家族，而且是不费吹灰之力。
<br/>　　洞玄境，强大得令人发指。
<br/>　　萧逸惊疑道，“大长老，若他是洞玄境武者，为何不亲自来对付我们萧家？”
<br/>　　这是萧逸想不明白的问题，以慕容墨的实力，萧家今日必败。
<br/>　　大长老摇摇头，道，“不清楚，但之前与慕容山战斗时，从他只言片语中猜测，慕容墨似乎是在闭关。或许，他认为就算自己不出手，单凭三大先天九重武者，足以灭我们萧家吧。”
<br/>　　大长老继续道，“总之，慕容墨突破到洞玄境已是事实。这次慕容家伤亡惨重，他一定会亲自来袭。我们要趁他未出关前，马上离开紫云城，举族迁移。”
<br/>　　“是。”众位长老应答一声。
<br/>　　他们已经不抱任何希望了，洞玄境武者的强大，远不是他们能抵抗的。
<br/>　　只有举族迁移，才能保住家族薪火，不至于家族全灭。
<br/>　　大长老有条不紊地下达着命令，“家族年轻子弟先走，执事们沿途保护，我们老一辈的则垫后，防止有追兵。”
<br/>　　“其他族人分开撤离，能走一个是一个。”
<br/>　　“将家族雕像以及武技书籍拿走，其余财物一概扔下，所有人轻装上路，务求在慕容家反应过来前尽早离开。”
<br/>　　“送信去矿脉，让四长老立刻带领族人撤离，不必回萧家。”
<br/>　　众长老一个个接令，快速安排下去。
<br/>　　萧逸呆愣在原地，万万没想到，偌大个萧家，竟被逼到放弃紫云城多年家业，举族逃离的地步。
<br/>　　而这一切，仅仅是因为一个人。
<br/>　　只是，这个人是洞玄境武者。
<br/>　　“洞玄境吗？”萧逸心中暗暗发誓，“终有一天我也会达到的，到时候，我会带着萧家强势回归。”
<br/>　　萧逸虽然孤傲，甚至不想走。
<br/>　　但他不是傻子，更何况现在事关全族人的安危，他根本无法选择。
<br/>　　以卵击石之事，他不会做。
<br/>　　就算结合整个萧家的力量，也只是脆弱的卵；慕容墨一人，就已是坚硬的石头。
<br/>　　不多时，所有事情都迅速处理好，族人们也开始了分散撤离紫云城。
<br/>　　“少家主，走了。”一个执事语气萧瑟地说道。
<br/>　　萧逸笑笑，道，“你们先走，我待会儿跟上。”
<br/>　　“嗯。”执事点点头，带领一批族人先离开。
<br/>　　偌大个萧家，只余下长老们，萧逸以及依依，寥寥几人。
<br/>　　众人都在久久凝视着府邸，很是不舍，他们在这里出生、长大，这里承载着他们所有的回忆。
<br/>　　忽然，三长老重重地对着府邸跪了下来，面露无比愧疚，“家主离开前，吩咐我一定要打理好家族。是我没用，竟让萧家多年基业，拱手放弃。我愧对萧家祖先，愧对家主阿。”
<br/>　　三长老平日坚强，今日却无法抑制地落下了眼泪。
<br/>　　大长老哀叹一声，拉起三长老，道，“起来吧，就算今日家主在，也无法扭转局势。”
<br/>　　萧逸认真地说道，“萧重叔叔不必难过，我们还会回来的，而且，这个日子不会太久，相信我。”
<br/>　　三长老抬头看了萧逸一眼，面露希冀，笑容中夹杂着泪水，道，“我当然信，以你的天赋，他日一定会带领家族取回这一切。”
<br/>　　“好了。”二长老忽然出言道，“这些伤感的缅怀，交给我们这些老家伙行了。萧逸，你快带着依依离开吧。”
<br/>　　“呵。”萧逸忽然笑笑，道，“一起走吧。”
<br/>　　“一起？”二长老怒瞪萧逸，道，“你发什么疯，我们几个长老要留下来阻挡慕容家追兵的。”
<br/>　　“如今所有族人都撤离了，就剩你和依依，快跟上他们。”
<br/>　　萧逸摇摇头，淡然道，“要么一起走，要么我也不走。”
<br/>　　“你...臭小子，这个时候你胡闹什么。”二长老作势就要一巴掌扇向萧逸。
<br/>　　大长老这时对三长老打了个眼色，三长老会意，点了点头。
<br/>　　萧逸瞥了他们一眼，淡淡道，“三长老，我知道你想做什么。你也知道我的性子，就算你将我打晕，强行送走，等我醒来，我会立刻回来替你们报仇。”
<br/>　　“混账。”二长老雷霆大怒，“整个萧家，谁都可以不走，唯独你，一定要走。你若死了，萧家还谈什么希望，日后谈何回来报仇。”
<br/>　　三长老也语重心长地说道，“逸儿，听话，走吧。你是家族第一天才，日后必能突破洞玄。你又是家主唯一的儿子，只有你，日后才能给予萧家希望。”
<br/>　　萧逸冷笑一声，“希望？若你们这些老家伙都死了，萧家才叫真的没有希望。”
<br/>　　“炎龙大陆，天大地大，但若没有你们保护，单靠一群后天执事，带着这么大一群族人，哪里能成为安生之所？”
<br/>　　“这个世界有多残酷，你们比我更清楚。萧家就算能安然撤离，但偌大个家族，身怀武技和重宝，若没有先天武者庇护，离死也不远了。”
<br/>　　萧逸一连窜话语掷地有声，让长老们愣了愣。
<br/>　　“还是那句话，要么一起走，要么我也不走。”萧逸语气强硬。
<br/>　　“你...”大长老愤愤道，“臭小子，原来你刚才特地留下，是想逼我们一起走。”
<br/>　　萧逸双手一摊，道，“你们可以这么认为。”
<br/>　　“臭小子。”大长老笑骂一声，道，“时间急迫，这次便算你有理。反正日后萧家的命运都掌握在你手中，既然你执意要一起走，那便听你的，一起走。”
<br/>　　“结果到底如何，便听天由命。”
<br/>　　大长老说罢，众人一同离去。
<br/>　　嗖、嗖、嗖、、、一连窜身影，快速地离开紫云城。
