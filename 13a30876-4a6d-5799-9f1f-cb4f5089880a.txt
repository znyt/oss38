　　林峰目光落在他身上立即冲了过去，一刀就是劈砍在他身上，但是这一刀，却居然没有对对方造成多大的伤害，对方的身上有一层极其滑的粘液，就好像邪恶的润滑油一样，林峰一刀下去居然都会滑开。
<br/>　　“既然如此，汽油弹！”
<br/>　　林峰拿出五十个汽油弹就是朝变异娃娃鱼丢去，对方动作很灵活，想要钻回水里然后再伺机突袭，可惜汽油弹是范围攻击，而且足足五十个汽油弹，足够形成一片的火海，把这个变异娃娃鱼包裹其中。
<br/>　　哪怕这个变异娃娃鱼皮糙肉厚，但也被烫的不轻，在那里怪叫。几头八阶的变异老鼠忽然从林峰站着的泥土里钻出来就是朝林峰裤裆咬来。
<br/>　　砰。
<br/>　　瞬间这个林峰直接消散了，八阶老鼠一脸的茫然，米粒大的眼珠子里都是一头雾水，似乎在说，奇了怪了，怎么没有吃到人肉呢，这熟悉的人肉怎么没了。
<br/>　　刷。
<br/>　　下一刻。
<br/>　　一个人从天而降直接把他脑袋踩爆了。
<br/>　　其余几个变异老鼠反应过来还没叫出声来。
<br/>　　青莲剑歌。
<br/>　　数刀下去，凄厉的刀芒横贯而出直接将他们斩杀，倒在了血泊之中。
<br/>　　至此。
<br/>　　原本看上去极其豪华的海陆空，全军覆没，只剩下一个没了眼珠子的变异水鸟在那里瞎扑腾，原本还很圣洁的白色羽毛占满了河床边的烂泥。
<br/>　　林峰随手一丢，一枚飞镖**准的入了对方早就没有了眼珠子的眼眶子里，刺破了对方的脑组织神经，彻底的带走了他的生命。
<br/>　　从开始到结束，不过两分钟罢了。
<br/>　　陆勇周华等人都看呆了。
<br/>　　这是何等的手段。
<br/>　　他们还没反应过来，一切就已经结束了。
<br/>　　“战神！战神！战神！”
<br/>　　蓦地，有一个河南口音的年轻士兵第一个给林峰振臂狂呼起来，然后顿时，这就好像是病毒一样，一个个士兵都再次狂呼起来。
<br/>　　战神二字，林峰当之无愧。
<br/>　　每个人都无比狂热，每个人都无比崇拜。看着林峰的目光之中，就好像在目睹一个神将一般。
<br/>　　林峰，再一次挽救了他们。
<br/>　　“林峰小兄弟当真乃是天人也！！”
<br/>　　周华感慨道。
<br/>　　“这可是A级怪物啊，结果在林峰小兄弟的手中，却是如此不堪一击，首长，我强烈建议，这一次，若是我曙光基地可以从这次灾难之中挺过来，这个林峰小兄弟必须要授予他战神勋章，他当之无愧！！”
<br/>　　战神勋章，是基地的最高荣誉勋章，代表的是无尽的荣耀，战神勋章，他虽然只是个勋章，但是其中蕴含的价值却是陆勇这个将军都无法媲美的。能够得到这个勋章的人，那必须是守护基地于水火之中的人，属于特等功之中的特等功。
<br/>　　“一定！只要曙光基地还在！别说战神勋章了，就是上将军衔，都不含糊！！”
<br/>　　陆勇认真的说道。
<br/>　　“兄弟们，战神来了，一起出手！保护基地！！”
<br/>　　“保护基地！！”
<br/>　　顿时每个士兵都来了战意，就好像一个在沙漠之中独自一人穿梭的旅者一样，渴死在沙漠之中是必然的，一切都是时间问题罢了。
<br/>　　但是现在，因为林峰的出现，让他们看到了一片绿洲，虽然他们距离绿洲还有一段路要走，在这段路程之中他们可能还是要渴死，但他们每个人都有了希望和动力。
<br/>　　基地内。
<br/>　　有将近万民幸存者，他们其中一部分满脸害怕，听着外面的枪响和丧尸怒吼，一个个都瑟瑟发抖，有吓傻的，有抱着自己老婆哭的，有趴在自己妈妈怀里痛哭的，也有安危自己孩子不要害怕的军部会保护他们的，也有人直接拿出十字架祷告的。
<br/>　　除了他们之外，原本隶属于特等区的宪兵们都紧张的握着手中的钢枪，他们虽然没有去前线，但是他们一个个都无比紧张，因为他们已经接到了命令，知道有些怪物会越过城墙从地底下出现，所以没有人敢懈怠。
<br/>　　“队长，我听我一个在平民区的表哥说，平民区已经沦陷了，死了好几千人。”
<br/>　　一个宪兵紧张兮兮道。
<br/>　　“别胡说。”
<br/>　　队长皱着眉低喝道。
<br/>　　“这个时候，别散播这些谣言，你亲眼看到平民区现状了还是你刚从外面进来？”
<br/>　　“我，我没有亲眼看到，但是我表哥他……”
<br/>　　“没看到一切就不能为真，做好你该做的事，端好你该端的枪，看到有怪物丧尸，扣动扳机，就这么简单的事情，不该问的别问，不该说的也别说。”
<br/>　　队长训诫道。
<br/>　　“知道了队长。”
<br/>　　几个宪兵都咽了口唾沫，他们都是预备役的，和队长不一样，队长是老兵了，特等区内有很多重要的实验室，甚至一些实验农田也都在特等区内，所以宪兵的素质要相对高一些，不像平民区内，很多宪兵都是走关系走后门混上去的一官半职，可能之前就是个杀猪的，也可能就是个天桥上给手机贴膜的。之前甚至都没有受过专业的军事化训练，枪都不会用。
<br/>　　也就在这个时候。

<br/>　　很快大门开了，一排排士兵顿时射击把门口的一些怪物全部击退，然后所有人就看到了从外面走进来了一个人。

<br/>　　“没错，这不就是压根不在乎我们在做的每个人生命吗？”
<br/>　　“就是这样，万一他们开门了，怪物冲进来了怎么办。太冒险了。”
<br/>　　许多人都议论纷纷，甚至还有人去抗议的，表示刚从那样的行为太过于疯狂，搞不好就是被怪物冲进来然后把他们都给吃了。
<br/>　　其中一个实力足足有七阶的，他似乎是个地下帮派的老大，他一说话，顿时一群人都跟着应付。
