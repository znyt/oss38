<br/>    时间缓缓地过去，凌仙一直在战斗，他不知道自己杀了多少仙人，但是这仙人，如同是杀不尽一般，杀了一个，又来一个，无尽的恐惧席卷了他。
<br/>    
<br/>        他明白了过来，是的，所有人都要他死。
<br/>    
<br/>        想要在此刻活下来，唯有真正参悟戮仙的境界。
<br/>    
<br/>        凌仙浑身是血，依旧是向前冲锋，挥拳打向一个个地仙，在他的身体周围，全是地仙的尸体。
<br/>    
<br/>        一具具尸体或是横沉，或是跪倒，或是死不瞑目。
<br/>    
<br/>        可是终究，没有人冲到他的身体周围。
<br/>    
<br/>        亡命和众多仙人对峙之间，从地面上，有一个带着斗笠，怀中抱刀的修行者飞掠而来，他一步落在地面上，而后便是跨过一里的距离。
<br/>    
<br/>        他是冷血，无冕者议会刀道首席。
<br/>    
<br/>        冷血路过众多的仙人，走到亡命之前，淡淡道：“让开。”
<br/>    
<br/>        亡命呵呵一笑，道：“地仙巅峰，去欺负一个超脱，脸都不要了。”
<br/>    
<br/>        冷血笑道：“看来你并不知道禁忌到底意味着什么，杀了他，对我来说很重要。”
<br/>    
<br/>        是的，很重要，四大仙国已经无冕者议会都很厌恶修行禁忌功法的家伙，原因很简单，因为至高无冕，修行的便是禁忌功法。
<br/>    
<br/>        四大仙国不想看到第二个，无冕者议会也不想看到第二个。
<br/>    
<br/>        亡命道：“拿人钱财，与人消灾。若是没有什么危险，也不配拿这个钱了。”
<br/>    
<br/>        冷血道：“那么你们，是想要死了？”
<br/>    
<br/>        这个时候，从人群之中传出一声放诞不羁地笑，“哎呦呦，很久没有试试看真正的战斗了，看来今天，要试试看了。”
<br/>    
<br/>        众人朝着那个声音看去，想要看看到底是谁，敢在冷血的面前说出这样的话来，那是一个背着血棺的年轻人，此刻正靠着一棵大树坐着，手中还提着一个酒壶。
<br/>    
<br/>        冷血看向李素衣，笑道：“看来你很聪明，还给自己背了棺材。”
<br/>    
<br/>        李素衣呵呵一笑，道：“试试看。”
<br/>    
<br/>        李素衣起身，走到亡命的身前，淡淡道：“后撤吧，阻挡住其他的地仙，这个家伙，交给我了。”
<br/>    
<br/>        亡命虽然不知道对方的来历，但是也是知道，李素衣是来帮他们的。
<br/>    
<br/>        冷血道：“杀你，用不着拔刀。”
<br/>    
<br/>        李素衣却是没管这些，这么多年的修身养性，看来，没什么气势了。
<br/>    
<br/>        冷血狂奔向李素衣，距离李素衣数十丈便是一拳直接轰在地面上，地面之上，一层泥土翻滚，拳威朝着李素衣飞去。
<br/>    
<br/>        李素衣看了一眼，这个肉身强度，和大师姐一比，还真是差远了。
<br/>    
<br/>        李素衣抬腿一脚落在地面上的翻滚泥土之前，翻滚的泥土瞬间停住，并没有冲到李素衣的身前。
<br/>    
<br/>        冷血仔细看了一眼，“法苦双修？而且都是修行到了地仙大圆满，还真是一个有趣的家伙，你，拥有了让我出刀的资格。”
<br/>    
<br/>        李素衣看了对方一眼，真正战斗开始之后，师父说过，不要说话，而且，能够最快斩杀对手，便是好的。
<br/>    
<br/>        李素衣伸手解下背后的血棺，落在身前，惊起一地的泥沙。
<br/>    
<br/>        他伸手拍开血棺，其中沉睡着两柄刀，他伸手握住两柄刀，左手一柄，右手一柄，左手仙道，右手霸道！
<br/>    
<br/>        左手施展仙道的术法，右手施展苦修的刀法。
<br/>    
<br/>        滚滚的魔气从李素衣的身体之中流窜而出，魔气滔天，如同魔君降世！
<br/>    
<br/>        两柄刀都很巨大，乃是重刀，李素衣提刀，脚下踏步，一跃冲向冷血，挥手之间便是两刀落下！
<br/>    
<br/>        冷血也是惊了惊，“魔道余孽？”
<br/>    
<br/>        这一片星空之中，已经少有魔道了，只剩下仙道在争锋。
<br/>    
<br/>        周围的修行者也是吃了一惊，“刚刚出现一个修行了禁忌功法的人，现在又有了一个魔道余孽！”
<br/>    
<br/>        “这片星空，到底是怎么了。”
<br/>    
<br/>        “如今，竟然有这么多余孽！”
<br/>    
<br/>        “而且竟然还敢大摇大摆地出现在世间！”
<br/>    
<br/>        冷血拔刀，那是一柄满是寒霜的幽蓝色长刀，世人都说他的前世是烈刀仙王，他却不在这么觉得，他修行的功法，刀法，都偏向于寒冰一道，他是在万载寒冰之中修成的强大肉身。
<br/>    
<br/>        同样是在无尽冰
<br/>    
<br/>        雪之中修出的刀法，“冷血，苦修地仙大圆满，斩雪刀法，刀名霜降，参上！”
<br/>    
<br/>        三柄刀在空中剧烈碰撞，魔气和雪花齐飞。
<br/>    
<br/>        两人各自退后半步，而后又朝着对方冲去。
<br/>    
<br/>        李素衣并没有说话，如果说他现在的刀法叫什么名字的话，那只能叫做斩魔刀了。
<br/>    
<br/>        他是无数次自斩心魔修出的刀法！
<br/>    
<br/>        冷血挥动手中的霜降，蓝色的刀气弥漫成一片，李素衣双刀横扫，斩断一切！
<br/>    
<br/>        地面，在两人的战斗余波下，下降了三尺，两人站在群仙之中开战，根本没有任何顾忌。
<br/>    
<br/>        周围的仙人都是避让开来，这两人的战斗太过凶猛，碰着就是死，擦着就是伤。
<br/>    
<br/>        “这两人，到底是什么来历，为啥我觉得，他们的实力，和九太子那般天骄都差不多。”
<br/>    
<br/>        “这些天骄，到底从何而来？”
<br/>    
<br/>        “无字劫落下，各种天骄出世，还真是一个苍茫大世啊！”
<br/>    
<br/>        冷血的手中，霜降不断地挥舞，形成强大的刀势，弥漫周天，就在这时，所有的刀势凝聚在一刀之中，“万千刀气，化作一刀。我这一生，无数次与刀修战斗，这一招之下，葬送了无数天才，而你，也躲不过！”
<br/>    
<br/>        咻！
<br/>    
<br/>        一道璀璨的刀光飞出，蓝色的刀光如同可以割开这天地，周围观战的仙人，都是在这一刀之下被震退数步！
<br/>    
<br/>        “这一刀，竟然如此强大！”
<br/>    
<br/>        “我知道了，他是冷血，无冕者议会的刀道首席！”
<br/>    
<br/>        “据说，冷血只要突破到金仙，便能拥有无免者的面具！”
<br/>    
<br/>        李素衣看向身前的这一刀，双手将自己的双刀插在身前，活动了一番嘴唇。
<br/>    
<br/>        周围的修行者都看呆了，“他这是准备做什么？”
<br/>    
<br/>        “空手接这一道刀光不成？”
<br/>    
<br/>        “他这是想要找死不成，就是地仙大圆满，也得瞬间死在这一刀之下！”
<br/>    
<br/>        李素衣的身躯之上，浑身一震，一只青色的饕餮从他的身躯之中爬了出来，饕餮的残暴之意瞬间弥漫开来。
