　　我其实是不老实的，不过我也不知道为什么我爷爷会觉得我老实，说我能守得住寂寞，而且一旦守得住，将来有大的机缘……所以把这把剑传给了我。”
<br/>　　他说的倒还没什么，叶少阳听的却是心惊不已，联想到道渊真人后来的精力……可不是在玲珑塔中守了一辈子，而且因为枯坐塔中，与世隔绝，悟得真道，法力突飞猛进，后来成为了一代宗师，人间最强的几位之一……
<br/>　　道渊真人说完之后，想到了一件事情，问叶少阳：“叶天师……”
<br/>　　叶少阳摆手道：“别老这么叫了，你就叫我少阳吧。”
<br/>　　“行，你叫我名字也行，或者也可以叫一声大哥。我名字就叫张道渊。”
<br/>　　叶少阳嘴角抽了抽，想到自己那个世界一百多岁的道渊真人满头银发的样子，这声“大哥”还真的有点叫不出口，总有一种乱了辈分的感觉。
<br/>　　毛小方也说道：“说的是，你我都这么熟了，称呼名字就行。你们叫我小方就可以。”
<br/>　　小方……虽然对前辈有点不敬，不过叶少阳真的觉得这个名字有点娘，一下子想到了草字头的“芳”……
<br/>　　“对了，道渊……大哥，你刚才要说什么？”
<br/>　　道渊整理了一下思绪，说道：“我想说的是，既然百年后的我，把松纹古定剑给了你，如果这把剑现在你的身上，加上我的，那不是有了两把一模一样的剑，那样的话，哪一把才是真的？”
<br/>　　叶少阳当场愕然。他还从来没有考虑过这个问题……
<br/>　　“我想，应该没什么真不真的问题，两把都是真的，应该都有同样的威力……”叶少阳思索了一下之后说道。
<br/>　　毛小方抓住叶少阳的肩膀，道：“我们不说松纹古定剑了，少阳子，你身上那些茅山的法器，是从这个时代传承到你手中的，也就是说……”
<br/>　　叶少阳猛然想到了什么，嘴巴张大，说不出话来。
<br/>　　毛小方继续说道：“也就是说，这个世界上，已经有了一把七星龙泉剑，你又带了一把过来……”
<br/>　　叶少阳深吸一口气，点了点头，现在的茅山上，自然还有另外一把七星龙泉剑，还有阴阳镜等等。
<br/>　　叶少阳舔了舔嘴唇，说道：“那又怎么样呢，一把是未来世界的，一把是属于现在这个世界的，好像……并不冲突？”
<br/>　　道渊真人道：“七星龙泉剑是你茅山镇山之宝，只有独一，换句话说，如果你把你手中的放在茅山上，然后流传下去，那就岂不成了两把？”
<br/>　　叶少阳怔住。
<br/>　　道渊真人接着说道：“我的意思是，有些东西——还是说这个龙泉剑，明明历史上只有一把的，如果传下去两把，那这多余的一把是从哪来的呢？无中生有？”
<br/>　　叶少阳没法回答这个问题，的确，按说起来，茅山上的那把七星龙泉剑，才是真正传承下去的那一把，但是自己手中的……明明也是从师父手中接过来的真正的七星龙泉剑，自然也不是假的……
<br/>　　叶少阳想来想去，最后也释然了，说道：“这个问题，好像其实并没有那么复杂。因为本来就是没有逻辑的，要是按照正常的逻辑，别说这把剑了，连我都不应该出现在这里，所以咱们还是不要纠结这个了，只要我能回去，一切都能回归正常。”
<br/>　　两人一想也是，叶少阳是一百年后的人，在科学观念上领先于他们，尚且想不明白这件事，以他们的思维，就算想一辈子也想不明白，干脆还是不想了。
<br/>　　“少阳子，我觉得，你要不要回茅山一趟，跟云秋生或者伏明子说说你的情况？”毛小方提议道。
<br/>　　叶少阳摇摇头，这个念头，其实他早就想过，不过他觉得暂时没必要去，就算自己出示七星龙泉剑等法器，证明了自己的身份，又能怎么样？没有山海印，就算是茅山掌教也没法送自己回去。
<br/>　　而且……叶少阳真的有点惧怕见到茅山的祖师爷们，不然让他们知道有关自己和青云子的一切，会有一种泄露天机的感觉，叶少阳也是打定了主意，除非必要，绝对不回茅山去。
<br/>　　“对了，我还有几个朋友，很有可能也跟我一样穿越过来了，有没有办法找到他们？”叶少阳想起了这茬，把情况跟他们说了。
<br/>　　两人都觉得很为难，华夏这么大，想要毫无线索地找几个人，简直就是大海捞针。
<br/>　　毛小方突然想起什么，说道：“你那几个朋友，既然也是法师，如果他们找你的话，你觉得他们会通过什么办法找你？”
<br/>　　叶少阳想了想，摇了摇头，一点头绪也没有。
<br/>　　毛小方说道：“我觉得，他们既然是法术界的，没准去了自己所在的宗门，可以发法帖去他们的宗门问问，一个是五台山弟子，一个是崂山弟子是吧，都是大门派，也联系得上。”
<br/>　　道渊真人道：“这件事交给我吧。顺便我再把你的名字放出去……嗯，这个是有点麻烦，好像找不到什么合适的理由。”
<br/>　　三人各自思索起来，最后毛小方突然想起一个法子，拍着叶少阳的肩膀说道：“我倒是有个办法，就是不知道少阳子你肯不肯。”
<br/>　　叶少阳道：“只要有机会，哪有什么不肯的，只要你别让我杀人放火就行。”
<br/>　　“说起来，还真就是杀人放火！”
<br/>　　叶少阳当场愣住。
<br/>　　“嘿嘿，我的意思，是发出法术界的通缉令，就说你是邪修，做尽坏事，让法术界各大门派留意你的行踪，这就是最大的理由，各大门派虽然不会用心去找，不过你的名字也会传遍法术界，如果你的朋友也在利用法术界的力量寻找你的话，很可能会得到你的消息，前来找你。”
<br/>　　叶少阳听他这个提议，不由有点心动。
<br/>　　道渊真人却道：“这个办法有点太危险了，容易引火上身，到时候万一有某个宗门的高人不分青红皂白，要对少阳子下去，岂不麻烦？”
