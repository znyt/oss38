　　剧情中，天帝宝库会在十年后开启，但是让他继续等下去，慕清霄绝对没有这种耐心。
<br/>　　天地间最遥远的距离，便是你明明知道，最后一卷天书就在天帝宝库中，而天帝宝库就在死亡沼泽，可你却触摸不到，非要等待十年。
<br/>　　来到死亡沼泽，慕清霄也是抱着试一试的心态，若是能够开启宝库，那自然最好，若是不能开启，那就只能继续等待，待十年后在来此地。
<br/>　　“哗啦啦...!”
<br/>　　倾盆大雨落下的没有一点预兆，在死亡沼泽中，谁也不知道下一秒会是什么天气，或许是阴天，或许会有雨水落下，唯独没有阳光投射进来。
<br/>　　有灵气护体，两人自然不担心被雨水打湿，但处于如此恶劣的环境下，心情也好不到哪里去。
<br/>　　而且这一路下来，各种毒虫让陆雪琪脸色都有些苍白，她虽有修为在身，但却是女儿家，碰到克制她的东西，依旧会害怕，比如毒蟾蜍等等类型的毒虫。
<br/>　　慕清霄无奈的摇摇头，将她轻轻的搂在怀中，不断向死亡沼泽内部进发，随着深入，周围树木也明显变得比外围粗壮许多，甚至偶尔能看到几百年的老树。
<br/>　　就在这时，慕清霄止住脚步，仔细的看了看周围的情况，陷入沉思之中。
<br/>　　“有什么发现吗？”
<br/>　　陆雪琪抬起臻首问道，她心中有些期待，若是慕清霄有所发现，他们便能早些离开此处。
<br/>　　两人在这片无边无际的死亡沼泽中找了好几日，却什么都没有发现，一路下来毒虫倒是遇到无数。
<br/>　　这里除了树还是树，然后就是变幻莫测的天气，地上无数毒虫，委实令她头疼无比，有时想想，也难怪此处从无人烟，哪里是人住的地方。
<br/>　　虽然害怕毒虫，但陆雪琪却未曾提出离开，她深知天书对慕清霄的重要性。
<br/>　　她此刻见慕清霄似乎看到了些什么，一直盯着森林深处看，刚想开口问，就听到慕清霄道：“雪琪，你看哪里，似乎有些古怪。”
<br/>　　陆雪琪顺着慕清霄的视线望去，只见在层层雨丝背后，浓密枝叶之间，突然有一道微弱金光一闪，随即消失。
<br/>　　陆雪琪美眸凝视着前方，却见金光消失许久，片刻后又再次亮起，只是距离两人似乎又远了一些，仿佛是在移动。
<br/>　　慕清霄嘴角勾起一丝笑意，喃喃自语道：“或许，天帝宝库能够开启也说不定。”
<br/>　　说罢，慕清霄就搂着陆雪琪，化作剑光腾空而起，向金光闪烁之地急速冲了过去。
<br/>　　追了许久，慕清霄忽然发现，金色光芒失去了踪影，剑眉顿时皱在一起，金色光芒的气息有些熟悉，古朴沧桑，与天音寺无字玉璧上的气息相同。
<br/>　　所以，金色光芒出现在此处绝非偶然，肯定与天帝宝库有关，但现在金色光芒失去踪影，难道还要继续地毯式搜索不成？
<br/>　　正当他准备再次释放神识的时候，死泽深处，潇潇不歇的雨势远方，忽地传来一声如虎啸龙吟的巨响声！
<br/>　　声音震耳欲聋，死亡沼泽深处的动物作鸟兽散，连站在距离声音发源地极远的慕清霄与陆雪琪，都觉得耳膜嗡嗡作响。
<br/>　　慕清霄眼中闪过一丝精芒，深深的望向死亡沼泽深处，如此威势绝对与天帝宝库有关，难道产生了什么变故，天帝宝库提前开启？
<br/>　　瞬间，天空中风云几乎以肉眼可见的速度变幻翻涌，云气蒸腾，一层层一幕幕如咆哮奔流的海浪，当真能以天地为之变色来形容！
<br/>　　慕清霄脸色平静，看不出有任何波澜，陆雪琪可是第一次见到如此场景，美眸中尽是骇然，若非靠在丈夫的怀中，刚才的威势就让能让她立脚不稳。
<br/>　　片刻后，在她惊愕的注视下，在死亡沼泽的最深处，骤然腾起一道璀璨耀眼的金色光芒，逐渐明亮，逐渐粗大......
<br/>　　到最后，直接化作无比巨大的金色光柱，携带者震耳欲聋的轰鸣声，直冲云霄，刺入云层之中，霎时间将天上地下照的明亮无比，映入眼帘的都是金色光芒，云是金云，树是金树。
<br/>　　饶是见多识广，慕清霄也是被这天地奇观所震慑，陆雪琪更是如此，两人昂首观望。
<br/>　　只见金色光柱越来越耀眼明亮，到最后几乎不可目视，天际风云翻涌更是激烈汹涌，围绕着金色光柱急速旋转，形成一个被金色光芒照的透明的巨大漩涡。
<br/>　　在这等壮观景象下，天地肃然，仿佛其中有什么尊至尊至贵的神物一般，让人不由得心生敬意。
<br/>　　许久之后，突然出现的金色光柱才缓缓停歇了下来，随即迅速减弱，来的快，去的也快，只不过一会的工夫，刚才惊天动地，叱咤风云的景象，便已经消逝于天地之间。
<br/>　　金色光柱消逝，但异象所带动吹动的强风，将茂密的树林枝叶吹得整齐向一面倒去，慕清霄搂着陆雪琪站在枝干上，随风沉浮，不为所动。
<br/>　　陆雪琪心思何等灵巧，很快就猜到，慕清霄需要的天书恐怕就在其中，银牙紧咬，道：“刚才的天地异象极其恐怖，其中凶险难测，清霄，我们要继续深入吗？”
<br/>　　虽然她不说，但慕清霄却能够从她美眸中读出担忧，轻轻在她粉唇上吻了一口，道：“我有把握，天帝宝库肯定是因为某种契机，提前开启了，我们进去吧。”
<br/>　　闻言，陆雪琪坚定的点了点脑袋，清楚此去凶险万分，但她却没有丝毫的犹豫，既然已经是慕清霄的女人，那不管对错，她都会紧随他的身后，她就是这样一位坚强的女子。
<br/>　　耀眼的光彩过后，更加深邃的黑暗降临大地，慕清霄清楚，刚才的异象，恐怕早就传到了死亡沼泽外，一旦消息传开，各门各派的修士都会前来查探。
<br/>　　刚才那片金色光柱规模实在太大，在修真门派眼中，便是异宝出世，在这些修真道上辛勤修炼的人们，哪有不怦然心动的道理？
<br/>　　到时候一旦各门各派齐聚，又是徒增麻烦，所以慕清霄直接搂着陆雪琪，化作一道剑光，以最快的速度向刚才金色光柱消失的地方冲去。
