        &lt;h3&gt;第三卷第一百五十八章  寒伯安的一天&lt;/h3&gt;
<br/>
<br/>        寒伯安快刀斩乱麻的处理了那个违规的事件，引来了安男的担心，但是他又不能回去陪安男，心里终究还是有些内疚。
<br/>
<br/>        他们家虽然是很富有，但是因为父亲性格的原因，家里一直都是那种冷冷淡淡的感觉，安男给了他一种温暖，安家也是充满了温馨的气息，安男的父亲和母亲相亲相爱，俩个老人似乎童心未泯，每次到他们家总是给人一种快乐的、如沐春风、宁静、和谐而美好的感觉。
<br/>
<br/>        在车上，他给安男打了电话：“男男，我今晚回不去了，抱歉，晚上要去北京，明天参加陈子昂他们公司的会议。”
<br/>
<br/>        自从元旦和陈子昂见过面之后，安男对陈子昂的好感度成直线上升，经常会问陈子昂的情况，还说要带着她妈妈去陈子昂的老家看看。
<br/>
<br/>        “不要紧的，你好好工作，带我向陈子昂问好，就说我很想念她。”安男柔声说。
<br/>
<br/>        提到陈子昂，寒伯安又想到了手镯的事情，一直让江俞轩问手镯，但是又不能确定，中间牵扯了那么多的事情，好像错综复杂的，那个手镯价值不菲，提起来怕也会伤到张函，真是让人头疼。
<br/>
<br/>        看来这件事情还得等一等了。
<br/>
<br/>        “好的，我一定会将你的问候转达给陈子昂的。”安男是从心里认可了陈子昂是她的亲人，但是陈子昂还一无所知。
<br/>
<br/>        陈家的手镯阴差阳错的跑到了张家的手中，真的是造化弄人，不可不说缘分的奇妙。
<br/>
<br/>        看来陈家和张家就是剪不断理还乱的那种关系。
<br/>
<br/>        如果安男想去陈子昂的老家的话，他一定陪着她们过去看看。
<br/>
<br/>        “晚上要早点休息哦，不要等我！”寒伯安轻声叮咛，每次他出差，无论多晚，安男都会等着他报平安的。
<br/>
<br/>        “我知道啦，你要注意点，吴家可能也不会善罢甘休的。”安男的忧虑不无道理。
<br/>
<br/>        “放心，我已经都想好了。”寒伯安稳稳的回答。
<br/>
<br/>        寒伯安的回答给了安男一颗定心丸。
<br/>
<br/>        挂了安男的电话，寒伯安又拨通了另一个电话：“是刘秘吗？我是寒伯安，我找一下吴伯伯。”
<br/>
<br/>        对方似乎是在征求意见，过了一会电话又重新拿了起来：“伯安？”
<br/>
<br/>        “吴伯伯，我有事情想向您汇报。”
<br/>
<br/>        “我都知道了！”
<br/>
<br/>        “那伯伯不会责怪我自作主张吧？因为对方太可恶了，竟然打着伯伯你的名号来威胁我，伯伯怎么会干这种事情呢？我接到电话时我就想到要立马洗白伯伯，召开记者招待会，希望没有给伯伯您带来麻烦！”
<br/>
<br/>        “你做的很对，以后切记不要听信别人的谣言！我还有事情，先挂了。”
<br/>
<br/>        寒伯安微微一笑，当我是三岁小孩？你能玩的我也会玩！
<br/>
<br/>        李长卿看着那自信满满的寒大总裁，心里佩服得五体投地，虽然别人也有可能这样做，但是寒伯安一声“伯伯”叫得对方想发火也发不出来。
<br/>
<br/>        人家就是一个小辈，哪里是寒家总裁，当时来不及和你沟通，但是事情已经做得很好了，不会给你抹黑的！
<br/>
<br/>        寒伯安挂了电话，却发现有了弟弟的信息。
<br/>
<br/>        臭弟弟：“我在欧洲，一切都很好，看到了大哥处理事情的果断和勇敢，为大哥点赞！”
<br/>
<br/>        黑虎：“难道你不给一个解释吗？”
<br/>
<br/>        臭弟弟：“无需解释，大哥已经做得很好了，希望大哥继续保持初心！”
<br/>
<br/>        黑虎：“你这是关心寒家呢？”
<br/>
<br/>        臭弟弟：“我是关心大哥！”
<br/>
<br/>        黑虎：“似乎寒家不是你的？”
<br/>
<br/>        臭弟弟：“我只知道大哥是我的，钱对我来说没有什么用。”
<br/>
<br/>        黑虎：“呵呵，那是你生活在富足的年代，感谢你的国家吧！感谢那些为我们创造盛世的人，感谢让我们安逸成长的家人吧！”
<br/>
<br/>        臭弟弟：“谨记大哥的教诲，一定不负大哥所望！”
<br/>
<br/>        黑虎：“春节回家吗？”
<br/>
<br/>        臭弟弟：“春节和落雪在欧洲过了，过一个二人世界。”然后是胜利的表情。
<br/>
<br/>        黑虎：“你怎么回事？难道落家真的要和吴家联姻？”
<br/>
<br/>        臭弟弟：“我到现在还不知道雪儿的心思，我觉得她是爱我的，但是我也觉得落家没有必要和吴家联姻，我还有事情要做，哥，希望你明白！也希望你能带领寒家走得更高更远。”
<br/>
<br/>        黑虎：“好，那你自己把握，有时间就回家看看，父母亲年龄也大了。”
<br/>
<br/>        臭弟弟：“好，带我问候安男嫂子，祝她貌美如花，永远十八！”
<br/>
<br/>        黑虎：“滚，你嫂子一直貌美如花、一直十八！”
<br/>
<br/>        臭弟弟：“有了老婆就让弟弟滚，我滚了滚了！”
<br/>
<br/>        寒伯安是长子，父亲退下来之后，所有的事情都是他在打理，弟弟寒冰玉占了寒家的一半股份，他们兄弟俩个人算是最大的持股人，这个是爷爷规定的，如果是父亲的话，可能他们也不会有那么多的股份，因为父亲还有其他的孩子。
<br/>
<br/>        看着弟弟发灰了的头像，他深深的叹了口气，他的这个弟弟从来都不管寒家的事情，倒是对落家的事情很上心，据他所知，他的弟弟已经为落雪将路全部都铺开了，就算是落雪什么都不干，这一生也是吃喝无忧的。
<br/>
<br/>        落雪！陈子昂公司最大的股东之一就是落雪！有些人天生就是含着金汤匙出生的人，还有别人为她打拼。
<br/>
<br/>        嗯，我也要为我家安男奋斗，为我家小安男奋斗！不对，为更多的小安男奋斗！
<br/>
<br/>        “长卿，我让你以安男的名义成立的爱心基金会最近准备得怎么样了？”寒伯安想到更多的小安男时问李长卿。
<br/>
<br/>        “基本好了，看你们什么时候举行婚礼。”
<br/>
<br/>        “好，最近很忙，还没有和安男商量，不要透露出去，那个是我送给安男的结婚礼物。”这个结婚礼物安男一定很喜欢。
<br/>
<br/>        安家的人是那种书香世家，对物质看得比较淡，但是他们不拒绝帮助别人！
<br/>
<br/>        寒家也有很多爱心基金，能够和自己心爱的人一起参与到更大的善举中来，那意义就更加非凡了。
<br/>
<br/>        想到此生一直会和安男相濡以沫下去，寒伯安心里泛起了甜甜的涟漪，风划过涟漪的波，荡起柔风细雨，柔软着那颗在商场上杀伐果断的心。
<br/>
<br/>        “男男：有你的每一天，都是甜蜜的每一天！”
<br/>
<br/>        世态安好，有汝为伴，千媚淡妆，素手相恋，至此一生，便是足愿！
