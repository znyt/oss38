        “我待在这里的半个月，你每天都要睡我好几次。性欲太强了，对身体不好。大哥，乖，今晚老老实实睡觉。你看你，最近黑眼圈都出来了。”童筱颖伸手摸了摸傅夜沉的眼睑，满是心疼地说道。
<br/>
<br/>        傅夜沉闭了闭眼睛，嘴角微扬，倏然搂住童筱颖的腰肢翻了个身，将她反压在自己身下。
<br/>
<br/>        但是，他只是蜻蜓点水般地吻了吻她的唇，然后侧过身去，将她搂入怀中，并没有进一步的动作。
<br/>
<br/>        童筱颖欣慰地笑了笑，伸手抱住他，在这男人怀里，缓缓地闭上了眼睛，甜甜地进入了梦乡。
<br/>
<br/>        第二天，她“飞”回了滨江城。第三天，她一大清早就起床了，洗漱完毕，在家吃过早餐，穿戴整齐地出了门。
<br/>
<br/>        她还是住在傅氏集团大厦对面的单身公寓里，要去傅氏集团上班，得过一条大马路。
<br/>
<br/>        此时，她刚好接到了部长孙莉莉的电话。
<br/>
<br/>        “嗯，是的。”
<br/>
<br/>        “我已经回来了，在去公司的路上。”
<br/>
<br/>        “那好，待会我到办公室后，和你们一起去甲方单位参加图纸答疑。你们等……”
<br/>
<br/>        “嘀嘀——”突然一声鸣笛将童筱颖吓了一跳。
<br/>
<br/>        她手里还举着手机，双眼却循声望去，只见一辆失控的面包车正向她快速撞过来。
<br/>
<br/>        童筱颖瞪大了眼睛，瞬间懵了。
<br/>
<br/>        当她以为自己死定了的时候，一抹黑影迅速闪过她眼前，她只觉整个身子被腾空飞起，随即重重地跌落在一堵肉垫上。
<br/>
<br/>        耳畔传来“乓嘭乓”地连环巨响，那辆失控的面包车撞到了马路对面的路灯杆上，同时也连撞了数辆小轿车。
<br/>
<br/>        余光里，有只白皙的手肘，在地面上擦出了一片血肉模糊的伤口。
<br/>
<br/>        童筱颖回过神来时，不经意间地抬眸，与一对炯炯有神如鹰隼般犀利的目光不期而遇。
<br/>
<br/>        他有一张很有西方男人味的俊脸，额头饱满，卧眉浓密，双眼皮，眼窝深邃，鼻梁高挺，但面无表情。
<br/>
<br/>        “谢谢你！”童筱颖一边忙着道谢，一边匆匆地从这男人怀里爬起来，然后伸手想看看他手上的伤势时，忽然间瞪大了眼睛。
<br/>
<br/>        这男人手肘上那片血肉模糊的伤口，竟然在迅速地自动愈合，仅仅只是一瞬间，若不仔细去看的话，真像是看花了眼的那种，稍纵即逝。
<br/>
<br/>        “你的手……”童筱颖刚想关问他的时候。
<br/>
<br/>        但男人并未理会她的道谢与关心，而是自顾自地站起来后，只身离去。
<br/>
<br/>        或许是因为同路，童筱颖默默地跟在了这男人的身后，他身着白色短袖衬衫，银灰色的西裤，黑皮鞋，但气度上不像是普通的打工族，只是令她没想到他也进了傅氏集团大厦的大门。
<br/>
<br/>        “裴总，早上好。”
<br/>
<br/>        他所经之处，不断地有问候声响起。
<br/>
<br/>        童筱颖记得，傅氏集团里新招来了一个执行总裁顶替了傅夜沉的位置，掌管傅氏集团旗下的大小事宜，难道他就是那个新来的CEO？！

<br/>        “金澈！”忽然，一个突兀的女音，响彻整个大厅。
<br/>
<br/>        大厅里，几乎所有人都循着这个女音的来源处望了过去。
<br/>
<br/>        童筱颖也下意识地回头，看向了那个声音的女主人。

<br/>        只见那女孩身形娇小，巴掌大的瓜子脸上，两眼大而水灵，鼻尖翘挺，小嘴粉嫩，及肩的长发束成马尾，露出白皙的颈项。

<br/>        她身着浅黄色连衣裙，恍若蕴涵着这夏末最后一股执着而旺盛的活力。

<br/>        可结果，她的话还未说完，人也刚走到他的跟前，便被他毫不留情地打断：“这位小姐，我们很熟吗？”

<br/>        夏玄灵顿时哑口无言，心痛地张着嘴，却不知道自己接下来该说什么。
