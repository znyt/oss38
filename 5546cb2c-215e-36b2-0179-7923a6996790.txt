<br/>        陆寒霆英俊的眉心里覆上了一层阴霾，他一点都没有减速，而是将油门踩到了底。

<br/>        夏夕绾坐直了身，生气的将小脸扭向了窗外，也不理他了。

<br/>        寂静的豪华车厢顿时陷入了一阵压抑的沉默，这算是两个人第一次真正意义上的小吵架。

<br/>        他不喜欢她被别人惦记的感觉。

<br/>        一点都不喜欢。

<br/>
<br/>        陆寒霆满腔的怒火像是被一盆冷水给淋灭了，作为一个成熟理智的男人他当然知道自己将油门踩到底在路上飙车是有多么的危险，但是他没能控制好自己，他甚至没能顾及她的感受，带她做这么极端的事情。
<br/>
<br/>        陆寒霆出了一手的冷汗，从来没有任何的时刻让他这么清晰的意识到，失控的自己是多么可怕。
