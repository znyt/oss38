　　林峰目光落在一个湖泊旁边的变异蜥蜴，那变异蜥蜴很大，身上的条纹看上去也很狰狞可怖，正在湖泊边喝水。
<br/>　　“开始猎杀吧！”
<br/>　　三天后。
<br/>　　林峰完成了任务，这些九阶怪物，倒也好对付，不像变异穿山甲变异鳄鱼那样有厚皮的。
<br/>　　三天时间。
<br/>　　林峰任务全部搞定。
<br/>　　回到曙光基地后。
<br/>　　林峰第一时间就去军部交任务了。
<br/>　　“快看，这不是那个疯子吗？”
<br/>　　“真的是他？好几天没看到了我还以为他死在外面了。”
<br/>　　“也没很多天吧，不过才三天罢了，别告诉我，三天时间，他就把任务全部搞定了？”
<br/>　　“你是疯子吧？三天可以杀的了这么多A级怪物？你是猴子请来的逗逼吗？”
<br/>　　“也是哦。”
<br/>　　许多人看到林峰顿时都来了兴趣，他们想要看看，林峰到底有没有完成任务，在许多人看来，林峰也许可能完成了其中的一个任务，这已经是他们说服自己所能相信的最大限度了。
<br/>　　“这小子就是你们说的那个什么疯子？疯狂接任务的疯子？”
<br/>　　一个汉子走了过来，在他背后还跟着七八个小弟，看上去气势汹汹的，这个汉子足足有七阶的实力，非常的强大。
<br/>　　“是鬼王来了。”
<br/>　　“真的是他，这家伙上次一拳捏碎了一个七阶怪物的脑袋，非常残暴。”
<br/>　　“这家伙也是个狠人，接取了不少任务，至今没有一个失败过的。”
<br/>　　“这个鬼王莫非是找疯子来的。”
<br/>　　“我听说鬼王这个人喜欢装逼，而且心胸狭隘，最看不得什么人在什么方面比他强的，典型的小心眼，不过实力很强，这下有好戏看了。”
<br/>　　林峰排队排的好好的，就看到一个家伙带着一群人来到他面前。
<br/>　　“大哥，就是他，别人口中的疯子。”
<br/>　　一个小弟指着林峰到。
<br/>　　“你就是疯子？”
<br/>　　鬼王上下看了眼林峰，嘴巴上毛都没有，非常的年轻。
<br/>　　“原来是个00后。”
<br/>　　鬼王嗤笑道。
<br/>　　“你谁啊。”
<br/>　　林峰看了对方一眼，对方故意把自己的气势放出来，七阶。在基地内那也是前五十的人物了。
<br/>　　“人送外号！鬼王！！”
<br/>　　“鬼王？”
<br/>　　林峰呵呵一笑，这么中二的名字。
<br/>　　“不知道不认识没兴趣。”
<br/>　　林峰瞥了眼就懒得搭理。
<br/>　　“呦呵，还挺拽，疯子是吧，他们都说你最起码是个七阶甚至是八阶的高手，我倒不怎么相信。”
<br/>　　“对了，这一次，你还接了一堆猎杀A级怪物的任务吧，疯子，我看你不应该叫这个名字，干脆叫傻子不错。”
<br/>　　“鬼王老大言之有理！”
<br/>　　“这个毛头小子还真是不知天高地厚，接取那么多任务，真当自己是周华贺寒鸦啊，不对，就算是周华和寒鸦两个人联手，也根本对付不了这么多A级怪物。”
<br/>　　一群人都嗤笑。
<br/>　　“原来是因为这个啊。不过不好意思，那些任务我已经全部完成了。不然我也不会来这个地方。”
<br/>　　林峰恍然大悟笑呵呵道。
<br/>　　“全部完成了？”
<br/>　　鬼王等人楞了一下，然后一个个都噗嗤的笑了。
<br/>　　就是其余一些吃瓜群众他们也都摇了摇头，显然没有人相信林峰说的。
<br/>　　“现在的年轻人啊，就是喜欢吹牛逼。行，你继续吹牛逼吧，最好吹破这天吹破这地吹得这斗转星移，我倒要看看你等下怎么收场。”
<br/>　　鬼王嗤笑道。
<br/>　　很快轮到了林峰。
<br/>　　林峰拿出了自己的身份证，这个东西算是个身份信息的录用凭证。每次接任务交任务时候都要这东西。
<br/>　　“来来来，大家都来看看大高手是怎么杀死一群A级怪物的。”
<br/>　　鬼王笑道。
<br/>　　一群人都凑过来看热闹。
<br/>　　“我看你装，继续装。”
<br/>　　鬼王看到林峰还在那里有模有样的，也不急，就干脆抱着胳膊准备看好戏了。
<br/>　　工作人员用个仪器一扫林峰的身份证，顿时电脑屏幕上就出现了林峰现在正在进行之中的任务和已经完成的任务，这一看吓了一大跳，已经完成的任务他没看，看了些正在进行之中的任务。
<br/>　　“全都是猎杀A级怪物的？”
<br/>　　工作人员不信邪的擦了擦自己的眼睛，确定自己没有看错，然后他古怪的看了眼林峰。
<br/>　　“林峰先生，你应该知道，在基地内恶意接取任务，是个性质非常恶劣的事情吧。”
<br/>　　工作人员严肃道。
<br/>　　“我知道啊。”
<br/>　　林峰点了点头。
<br/>　　“知道你还接这么多任务？”
<br/>　　工作人员有些恼怒，在他看来，林峰这种人就是纯粹来戏弄他们的。
<br/>　　“因为我可以完成任务啊。”
<br/>　　林峰认真道。
<br/>　　“可以完成任务？”
<br/>　　工作人员都不知道该说什么了。
<br/>　　“小伙子，别吹牛逼了，你再这样逗人家玩，小心被带走关小黑屋。”
<br/>　　鬼王笑道。
<br/>　　“就是啊，你这已经算是占用公共资源，扰乱公共场所秩序了。这是违法的你知道吗？而且还是重罪。”
<br/>　　工作人员摇了摇头，“看你这么年轻，我今天也不追究你的责任了，这些任务我会清理掉。”
<br/>　　他是有权力立即叫宪兵队过来的，在他们桌子底下就有个按钮，就像是银行柜台一样下面有报警的按钮，一旦有人抢银行他们一按就直接拨通警局内线。
<br/>　　他们这个也是差不多的道理，若有人在这里闹事，他们一按，立即全副武装的宪兵队就会出现。
<br/>　　林峰见他们这样，也懒得再废话。
<br/>　　“你们稍等。”
<br/>　　说着他就离开了这里。
<br/>　　“啧啧，吹牛逼大王要跑咯。”
<br/>　　“知道这牛皮吹不下去了，哈哈，怎么，现在才觉得丢人了？我还以为你是不见黄河心不死呢。”
<br/>　　“还疯子，我看叫傻子吧。”
<br/>　　鬼王等人立即哈哈大笑嘲讽起来。
<br/>　　工作人员也摇了摇头，很显然，在他看来，林峰这是自己给自己台阶下，一去肯定是不复返了。
<br/>　　不过下一刻。
<br/>　　原本还哈哈大笑的鬼王等人一个个都笑不出来了，甚至表情都僵硬了。
<br/>　　因为他们看到林峰拖了个东西进来。
<br/>　　这个东西出现的一瞬间把一群人吓了一跳啊，因为这是一个硕大的尸体，血淋淋的，两个眼珠子还瞪着。透露着一股凶悍的气息。
<br/>　　A级怪物的尸体？？？每个人的表情都彻底的呆住了。
