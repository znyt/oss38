        &lt;h3&gt;第三卷第四百零三章  转机&lt;/h3&gt;
<br/>
<br/>        凌辰在杭州呆了很多天，家里也有人在调查，似乎形势越来越紧张，凌辰和助理每天都要被叫着进行财务核算。
<br/>
<br/>        突然有一天宁少天给凌辰打了个电话：“凌少，听说你在杭州，有什么需要我帮忙的吗？”
<br/>
<br/>        “伯父您好，我最近一直在忙着配合有关部门调查，所以没有时间去拜访您，等过这一段时间忙完了，我一定登门谢罪。”
<br/>
<br/>        “我听说你是被人举报的？”
<br/>
<br/>        “嗨，我也不知道是那个丧尽天良的人举报的，不过也得感谢人家，要不然我还不知道这些事情，等以后知道了我一定要好好感谢一下他。”
<br/>
<br/>        宁少天听着凌辰的话，气得七窍冒烟。
<br/>
<br/>        “做生意啊，最主要的是要讲究诚信，切不可贪图眼前的利益，如果你需要我帮忙吭一声，我和正雄的关系不错，能将大事化小，小事化了最好了。”
<br/>
<br/>        “伯父，别，您可千万不要去欠人家的这个情，说实话，我们有些事情不好出面，正需要这样正义的人来做这个事情，人正不怕影子歪，您别担心我。谢谢您啊，伯父，改天我和雅儿一起去看您。”
<br/>
<br/>        “你和雅儿在一起？”
<br/>
<br/>        “嗯，我和雅儿在上学的时候就在一起了，只是雅儿一直没有给您说，怕您不同意我们的事情，唉，都怪我！”凌辰心一横开始编起了故事，反正总有一天会面对的。
<br/>
<br/>        “你说什么？你和雅儿在上学的时候就在一起了？”宁少天一拳头砸在办公桌上。
<br/>
<br/>        “是啊，我们真心相爱，之前一直怕您不同意，所以迟迟不敢给您说。年前原本想着刘伯父会给您提了，谁知道您们误会了。”
<br/>
<br/>        宁少天突然就想起来了在刘诚之家下棋的时，刘诚之的父亲对他说的话。
<br/>
<br/>        “那你和玥儿？”
<br/>
<br/>        “我和玥儿清清白白，您想想玥儿是雅儿的妹妹我怎么会对她怎么样呢？我再渣我也不敢再您身边这样做吧？您借我一百个胆子我也不敢啊，况且这么多年，除了雅儿我心里再也没有别人了。”
<br/>
<br/>        “凌辰，你说的可是真话？”
<br/>
<br/>        “如果有一句假话，让我事业无成，娶不到老婆！”凌辰赌咒发誓。
<br/>
<br/>        “呸呸呸，凌辰的嘴骗人的鬼，我就是骗人的，不过我爱雅儿是真的。”凌辰在心里默默的给自己解释了一下。
<br/>
<br/>        “那你就打算和雅儿就这样过下去？”
<br/>
<br/>        “不，我希望能得到您的祝福，但是首先我需要有足够的能力，所以伯父，凌辰一直在努力，这么多年一直在为雅儿努力着。”
<br/>
<br/>        宁少天默默的听着凌辰的话，凌辰这些年的努力其实他也有过调查，要不然宁玥要嫁给凌辰他会同意？
<br/>
<br/>        “那你改天和雅儿一起回来一趟吧。”
<br/>
<br/>        “好的，谢谢伯父。”
<br/>
<br/>        宁少天虽然当时很生宁雅的气，可是毕竟他是从商场摸爬滚打过来的人，仔细的安静下来分析了一下情况，宁雅做的事情无可厚非。
<br/>
<br/>        陈子昂是个人精，早就算好了结果，之所以让宁雅回来可能也就是试探他们的诚意和公正度。
<br/>
<br/>        成了他们名利双收，不成，陈子昂并不损失什么，相反，他宁少天可得被人笑话了，原本女儿代表别的公司回来和他谈判就应该是一个笑话，如果再谈崩了就更是大笑话。
<br/>
<br/>        权衡利弊之下，宁少天感觉自己冤枉了宁雅。
<br/>
<br/>        今天听凌辰的一番话，想起了刘诚之父亲的话，突然就对宁雅心生愧疚，所以让凌辰和宁雅一起回家。
<br/>
<br/>        宁雅也是他宁少天的女儿，谁嫁给凌辰都可以，他们两家还是会合作的，谷正雄告诉他，山西凌家查不出来什么，一切都是清清楚楚明明白白。
<br/>
<br/>        凌辰这小子还是不错的，多一个朋友多条路，况且刘家和他们家的关系都不错。
<br/>
<br/>        凌辰挂了宁少天的电话，从座椅上一跃而起：“耶！”
<br/>
<br/>        “胜利在望，胜利在望啊！”
<br/>
<br/>        他得速度的赶到北京去，和宁雅沟通一下，也看看宁雅的态度，宁雅被伤了心，不知道还愿不愿意回宁家。
<br/>
<br/>        凌辰是个行动派，说走就走，撇下助理直接飞了北京。
<br/>
<br/>        当宁雅和胡婷在办公室里商量工作计划的时候，凌辰敲开了宁雅办公室的门。
<br/>
<br/>        “凌辰！”宁雅抬头看见凌辰，有一瞬间失神，便绕过胡婷直接扑到了凌辰的怀里。
<br/>
<br/>        “我回来了！”凌辰紧紧的拥抱着宁雅，这种感觉真好。
<br/>
<br/>        “哎！哎！哎！辣眼睛了啊。”胡婷在旁边叫着。
<br/>
<br/>        “辣眼睛你去用水洗洗，别在这里碍眼。”凌辰连一个眼神都不给胡婷。
<br/>
<br/>        “噗嗤。”宁雅被逗笑了。
<br/>
<br/>        “你怎么来了？”
<br/>
<br/>        “我想你我就回来了。”
<br/>
<br/>        “咦，还回来了呢？这是你的家？”胡婷开始打击凌辰。
<br/>
<br/>        “小雅在哪里哪里就是我的家！”
<br/>
<br/>        胡婷被凌辰的狗粮暴击，流量直线下降，她想撤了。
<br/>
<br/>        她得回去将她的狸猫提溜出来，要不然她这几天估计是要吃狗粮吃到撑了，而这几天工作在紧要关头，她还必须的帮忙支撑着。
<br/>
<br/>        “我走了，我给你们腾地方。”胡婷抱着资料就准备出门。
<br/>
<br/>        “胡婷，你去哪里啊？”
<br/>
<br/>        “我重新找个地方办公啊，嘶，迟不来早不来，偏偏最忙的时候来，真是没眼色。”胡婷瞪了一眼凌辰。
<br/>
<br/>        凌辰看着胡婷笑了，他感觉胡婷其实挺可爱的，也许这就是爱屋及乌的原因？
<br/>
<br/>        “你们继续忙你们的，我就坐这里看你们工作。”
<br/>
<br/>        “要不，你先回家去，我把钥匙给你？”宁雅看了一眼凌辰，手头上的事情确实是比较紧急。
<br/>
<br/>        “不，我等你一起，你们当我是透明的。”凌辰温柔的说，他也是经常忙，当然知道工作的重要性，宁雅才开始创业，一切都是从头做起，什么都要摸索着前进。
<br/>
<br/>        宁雅和胡婷也就不再理会凌辰，不过凌辰很快就加入到了他们的讨论中，给了他们很多中肯的意见。
<br/>
<br/>        “凌辰，才发现你撩妹一流，智商也一流，不错啊，这些建议都非常好啊。”胡婷大大咧咧的捶了一下凌辰。
<br/>
<br/>        “三个臭皮匠顶个诸葛亮，人多了集思广益，眼界就宽了。”凌辰笑着说。
<br/>
<br/>        “哈，说你胖你就喘上了？能不能谦虚点？”胡婷笑骂着。
<br/>
<br/>        “像我这种自带优秀气质的人靠谦虚是隐藏不住闪光点的，也是你羡慕不来的。”凌辰搂着宁雅的肩膀嘚瑟的看着胡婷。
<br/>
<br/>        “你干嘛让陈子昂将款都打到我的卡上？”宁雅看着凌辰那嘚瑟的样子，心里甜蜜蜜的。
<br/>
<br/>        “我要是直接给你，你就不会拿了，反正陈子昂欠咱的钱，给你给我都一样，连我都是你的。”凌辰不要脸起来是不管有没有别人在旁边的。
<br/>
<br/>        “凌辰，得了吧你，这八字还没有一撇呢，我告诉你，小雅以后结婚了，也要和你做婚前公证，你们的财产都各是各的。”胡婷炸呼呼的喊，敢在我面前撒狗粮，你也得看看我是谁。
