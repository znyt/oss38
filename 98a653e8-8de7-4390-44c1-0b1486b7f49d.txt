　　第585章
<br/>　　程大雷是哑巴吃黄连，有苦说不出。他现在被绑在大树上，身上一丝力气也提不起来。扯着嗓子喊了半天，结果没喊来救星，却又招来一伙要杀自己的人。韩通一伙明显比李如兄妹猛得多。
<br/>　　话说，如今在帝国找个会救程大雷的人还真不容易。
<br/>　　韩通上三眼下三眼打量程大雷，好事从天而降，好到令他有些不敢相信。
<br/>　　“你们怎么抓到这贼人的？”韩通问。
<br/>　　李如款款向前，道：“我们是沙家集的酒家，这贼人来我们酒馆用饭，被我兄妹认出，所以在他的酒中下了麻药，正准备报告官兵。”
<br/>　　韩通看了李如一眼，虽然饱经风霜，却仍然有几分姿色。并不像抛头露面酒馆掌柜，像个大户人家出来的夫人太太。
<br/>　　“大人，会不会是个冒牌货，想那程大雷何等人物，怎么会被他们两个抓住。”手下谨慎道。
<br/>　　韩通点点头：“无论如何，还是小心为上。来人，先挑了他手筋脚筋，把他带走。”
<br/>　　立刻有人从马上下来，手中提着佩刀，程大雷提不起力气，否则一根麻绳也困不住他。
<br/>　　正这个时候，又有一队人闯进林子，看到这一幕时忽然停住。韩通一帮人也是如临大敌，各自握紧兵器，警惕的盯着对方。
<br/>　　青州大盗刘黑虎。
<br/>　　刘黑虎占山为王，手上有千余弟兄，他为祸一地，自然少不了和韩通打交道。不过他们打交道的工具是长枪和大棒，今天你杀我两个兄弟，明天我再杀过来。
<br/>　　今天刘黑虎是准备去踩盘子，快入冬了，山寨总要想办法度过这个冬天，却也是没想到会在这里碰见韩通。
<br/>　　双方剑拔弩张，刘黑虎困惑的盯着周围，也搞不太懂眼前的状况。
<br/>　　韩通心里一激灵，他今天出来带的人本就不多，这刘黑虎又是一员悍匪，真打起来，己方可讨不到好处。
<br/>　　刘黑虎目光落在程大雷身上，皱眉道：“这位是？”
<br/>　　“你好。”程大雷有气无力道：“凉州，程大雷。”
<br/>　　“那个程大雷？”刘黑虎稀里糊涂道了一声，顿时一惊：“该不会那位吧？”
<br/>　　韩通此刻已经想撤，抓到程大雷自然是大功一件，可想来想去，还是性命要紧。
<br/>　　程大雷觉得他想多了，他虽然和刘黑虎都是绿林一路，可二人委实没什么交情。韩通该不会天真的以为，现在要杀他的，只有官家吧。
<br/>　　官兵要抓了程大雷立功，匪路也要踩着程大雷的脑袋往上爬，所以程大雷的情况并没有变得更好，而是更糟糕。
<br/>　　“该不会是真的吧？”
<br/>　　刘黑虎也是将信将疑，确实是听说程大雷到了青州，但他那样响当当的人物，怎么会被人像病猫一样绑在树上。可看到韩通如临大敌的眼神时，他心中一惊：这还的确是真的。
<br/>　　“姓韩的，今天也是冤家路窄，但程当家的命我要了，你哪里来的回哪里去，我不找你麻烦。”
<br/>　　韩通本来就想撤，可刘黑虎这句话却让他留了下来。毕竟是军中大将，他也是要面子的。
<br/>　　“刘黑虎，你忒狂妄了些吧。我早就想带兵平了你的黑虎寨，一时没抽出时间，你当真就以为我怕了你不成。”
<br/>　　“呵，我们也不是没有交过手，今天就在此地分个雌雄。”刘黑虎。
<br/>　　二人剑拔弩张，立时就要动起手来。正这时候，几匹快马闯进了林子，为首之人看了一眼程大雷，立刻惊喜道：“找到了，好贼，看你还往哪里逃。”
<br/>　　程大雷无奈的闭上了眼睛。
<br/>　　书院来的，为首两个人是钟牧山和李舜时，都与程大雷照过面。从徐州一路追到青州，现在都快到幽州了，他们还不肯放过程大雷，可见心中对程大雷的仇恨。
<br/>　　今天这里委实热闹了，兵、匪、读书人三家凑到一起，程大雷也不知道，还会不会有更多人出现。反正此刻的状况，已令他束手无策，他空有一身本事，但如今连剑都握不紧，干脆选择闭上眼睛，专心恢复体力。
<br/>　　钟牧山目光转了一圈，道：“书院做事，还请诸位行个方便？”
<br/>　　“书院，哪个书院？”刘黑虎一头雾水。
<br/>　　“这里青州，我可从未听说过什么书院。”
<br/>　　韩通说罢，与刘黑虎相视一笑，二人颇有一种心有灵犀的感觉。
<br/>　　就算在青州，韩通和刘黑虎也不可能没听说过书院。但书院的影响力主要在徐州，到了青州，大家也就不再将书院当会事，尤其是对韩通和刘黑虎这种人来说。
<br/>　　钟牧山已经面露怒色，再次重复道：“请诸位行个方便，改日书院必有重谢。”
<br/>　　“呵，你当自己是什么人，这里是青州，不知道什么书院，滚回你的徐州去。”刘黑虎冷喝道。
<br/>　　钟牧山和李舜时都没有行走过江湖，也就是目前书院缺人手，才将二人派了出来。所以他们都是眼高于顶，不知道江湖的人心险恶。
<br/>　　现在三方都要程大雷的性命，却是谁也不肯想让。刘黑虎出门带了二三十人，人数上占绝对优势，韩通的人少一些，钟牧山带的人更少，只有七人。
<br/>　　但这七人恰恰能布成书院剑阵，七人成阵，不说必胜，但至少可以立于不败之地。
<br/>　　三方都觉得会赢，于是谁也不服气谁，口里说不清道理，也就只有手底下见真章。
<br/>　　所有人都忽略了李如和李大头，甚至程大雷也忘了他们。毕竟，无论哪一方都能威胁到程大雷的性命，相比之下，李如兄妹就显得有些无足轻重了。
<br/>　　二人也感受到了这一点，身体哆哆嗦嗦，脸色苍白。他们没有想到，程大雷招惹到的，都是这样的人物，在他们印象中，程大雷该还是青牛山那个小小不言的山贼。
<br/>　　二人想走，趁着三方都没有注意到他们时，悄悄的溜走。
<br/>　　“你们两个往哪里去？”韩通拦在二人身前。
<br/>　　李如干笑两声，本想施展妩媚之术，从韩通这里逃一条生路。
<br/>　　咔嚓。
<br/>　　一柄大刀忽然抡了下来，将二人脑袋一下一下斩成两半。
<br/>　　“大人，和他们废话什么，他们看着也不像好人。”
<br/>(本章完)
