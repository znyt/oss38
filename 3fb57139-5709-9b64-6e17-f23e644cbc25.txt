　　唤醒剑鼎，不是一息之间的事。
<br/>　　这剑鼎已经是沉睡了万年之间，其中的阵法禁制，无数纹路，都全部沉睡这。
<br/>　　楚风眠只能够是以极道剑印，一点点的唤醒剑鼎。
<br/>　　不过在这段时间内，楚风眠倒是可以做些其他事情。
<br/>　　楚风眠大手一挥，那荒鎏金的身形，突然间显现了出来，跪在了楚风眠的面前。
<br/>　　这荒鎏金跪在地上，脸色无比愤怒，一双眼睛恨不得是要杀了楚风眠。
<br/>　　可是在他身上那一股龙威的压迫，让他根本没有反抗的能力了，一身灵力是被死死的镇压这。
<br/>　　在这镇压之下，现在荒鎏金的力量，跟一个普通人，都强不了多少，一名普通人，手持灵器，现在都足以轻易杀了这荒鎏金。
<br/>　　楚风眠踏入了九劫古帝之后，一切的帝尊境武者，楚风眠挥手之间，就足以镇压轰杀。
<br/>　　像是无道，纯剑长老，甚至是屠人间，都是号称帝尊无敌的人物。
<br/>　　因为以他们的实力，轰杀其他九劫古帝，几乎是轻而易举的事情。
<br/>　　可现在，这帝尊无敌的称号，只有楚风眠才配拥有，因为哪怕是无道重生过来，现在楚风眠一招之下，都足以将他轰杀，这才是真正的无敌。
<br/>　　那荒鎏金挣扎了一下，发现无法反抗之后，便是看向了楚风眠，正好是看见了楚风眠炼化剑鼎的一幕，眼神之中露出无比不可思议的神情道。
<br/>　　“极道剑印，小子，你到底是什么人？这极道剑印，应该只有剑道之主的弟子才懂得才对，你怎么可以施展的出？”
<br/>　　这剑鼎落入到荒鎏金手中可是足足万年的时间，在这万年的时间内，荒鎏金也无数次的想要炼化剑鼎，让剑鼎认主。
<br/>　　因为荒鎏金也知道，只有让剑鼎认主，才可以发挥出剑鼎之中种种神奇的力量，那时才是这一件至宝的真正威能。
<br/>　　让剑鼎沉睡，只是后来荒鎏金尝试了无数次失败之后，最终想到的办法。
<br/>　　想要让剑鼎认主，只有拥有极道剑印的人，才有资格。
<br/>　　寻常剑道门弟子，都没有这个资格。
<br/>　　曾经万年之前，剑道门覆灭一战之中，也有一些的剑道门弟子，选择背叛剑道门，投敌以求生路。
<br/>　　像是那枯叶，便是如此。
<br/>　　所以荒鎏金也寻找到过一些当年剑道门弟子，想要借助他们的力量，去炼化剑鼎。
<br/>　　可最终却是一无所获。
<br/>　　这剑鼎的认主条件，最为关键的一点，就是极道剑印，。
<br/>　　没有极道剑印，除非的一身实力要超越剑道之主，不然根本没有可能让剑鼎人主。
<br/>　　剑道之主的实力，曾经楚风眠不知道，可是之前从剑凌哪里，楚风眠可是了解到。
<br/>　　剑道之主的真正实力，可是一位仙帝，仙人九阶之中的巅峰。
<br/>　　想要拥有超越他的实力，几乎没有可能。
<br/>　　所以拥有极道剑印，是唯一一个让剑鼎认主的办法。
<br/>　　可剑道门当年剑道之主的弟子，却无一例外，没有任何一人投敌，这也让荒鎏金没有了办法，才施展出手段来，让剑鼎沉睡。
<br/>　　如今看到楚风眠施展出极道剑印来，唤醒剑鼎，荒鎏金几乎是震惊的说不出话来。
<br/>　　想破脑袋都想不明白，这一切到底是为什么。
<br/>　　“想不明白？”
<br/>　　楚风眠看到荒鎏金的表情，嘴角也是浮现出几分笑意。
<br/>　　也怪不得这荒鎏金想不明白。
<br/>　　二世为人，本就是虚无缥缈的事情。
<br/>　　哪怕是剑道之主那个层次的存在，都无法做到的事情，楚风眠居然是做到了。
<br/>　　楚风眠从天绝峰上，一跃而下摔死，可万年之前，居然重生归来，这一切玄妙，让荒鎏金想一万年，他都想不明白。
<br/>　　不过楚风眠也懒得解释，他表现的越是神秘，这荒鎏金也就越是惧怕楚风眠。
<br/>　　果然那荒鎏金看向楚风眠的表情，也是无比复杂，似乎是无法看透楚风眠一样。
<br/>　　这正是楚风眠的目的。
<br/>　　“荒鎏金，说吧，你们荒古门的老古董，谁还活着？”
<br/>　　楚风眠看向荒鎏金，冷声开口道。
<br/>　　万年之前围攻剑道门的一战之中，七大宗门皆是损失惨重，至少是有着九成以上的强者，都陨落了。
<br/>　　但是依然会有一些老古董，活了下来。
<br/>　　这些老古董，纵然没有踏入天人境，只怕经历万年的时间，实力积累到了一种不可思议的地步。
<br/>　　楚风眠已经是跟七大宗门彻底撕破了脸皮，如今也是要知己知彼，才可以早早做好准备来应对。
<br/>　　“小子，你不用想要从我的口中知晓任何的事，反正你也杀不了我，用不了多久，这里是消息传递出去，我的师尊就会出手来救走我的。”
<br/>　　那荒鎏金听到楚风眠的话，冷笑连连，纵然是被楚风眠所镇压，但是他的脸上也没有任何的惧怕之色。
<br/>　　似乎是有着绝对的把握，楚风眠杀不了他。
<br/>　　“小子，你哪怕是继承了剑道门的传承，在七大宗门的面前，依然是不值一提，曾经万年前剑道之主都被七大宗门所杀，你算什么？以你的实力，跟七大宗门为敌，是自寻死路。”
<br/>　　“我劝你现在早早臣服七大宗门，还可以保全性命，不然等到剑道门灭门之日，你也得跟这剑道门一起陪葬。”
<br/>　　这荒鎏金有恃无恐，更是对着楚风眠直接威胁了起来。
<br/>　　这威胁的话，楚风眠懒得理会，不过从这荒鎏金的口中，楚风眠倒是发现了一个秘密。
<br/>　　师尊？
<br/>　　这荒鎏金的师尊，楚风眠可知道是谁，那可是曾经万年前，荒古门的门主，荒天王。
<br/>　　在如今九域的记载之中，都有记载这荒天王，也是死在了万年之前，剑道门的大战之中。
<br/>　　可听着荒鎏金的话，荒天王，居然没死？
<br/>　　荒天王在万年之前，就身为荒古门的宗主，一身实力在万年之前，就已经是踏入了半步天人境的境界。
<br/>　　如果他还没有死，在经历这万年的积蓄，只怕是这荒天王的实力，已经是达到了一种不可思议的地步。
<br/>　　楚风眠心中是猜到了，荒古门之中肯定还有着一些老古董的存在，可没想到第一个知晓的，就是这样强横的人物。
