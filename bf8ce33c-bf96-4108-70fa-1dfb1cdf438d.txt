　　随着时间过去，一个个武者的分数和排名被公布，前五百名者，通过了这次的考核。
<br/>　　值得一提的是，顾长风和赵不群竟然也通过了考核，而且排名第九和第十。
<br/>　　这两人终归是先天九重，而且手上还有灵器。
<br/>　　虽然被萧逸打伤过，但还是成功抢夺到了分数。
<br/>　　排名第十之后的武者，分数开始锐减。
<br/>　　特别是到了二十后，分数跌倒七八十。
<br/>　　到了一百之后，分数几乎普遍在十分以下。
<br/>　　很多武者都是以并列的分数通过考核的。
<br/>　　排名四百到五百，分数都是5分。
<br/>　　排名前五百，通过考核。剩下的武者，将在第二天由剑派内的执事送离裂天剑派。
<br/>　　他们，将无缘后续的考核。
<br/>　　裂天剑派的考核，共有三个。
<br/>　　如今才是第一个，淘汰率便已经如此高；不少人都对后面的考核充满了担忧。
<br/>　　.....
<br/>　　翌日，排名前十的人，在易老的带领下，被带到裂天剑派的外门宝库中，领取属于他们的奖励。
<br/>　　外门宝库内，一件件奇珍异宝，琳琅满目。
<br/>　　功法、武技、天材地宝、珍稀物品，应有尽有，看得众人眼花缭乱。
<br/>　　“记住了。”易老严肃地说道，“宝库内，宝物不下十万之数。但你们只有半天的时间去选，能否选到适合自己的东西，全看你们自己的机缘。”
<br/>　　“另外，每件宝物上都有禁制，不得擅自触碰，选好后便叫我，我会替你们取出来。”
<br/>　　易老吩咐完毕，在一桌案前坐了下来，随后拿起一本书籍翻看起来。
<br/>　　考核的前十名，第一名可随意挑取三件宝物，第二第三名两件，四至十名一件。
<br/>　　宝库内，放置着不下十万之数的宝物，可想而知面积有多大。
<br/>　　萧逸随意走着，寻找着自己需要的东西。
<br/>　　“玄阶高级武技，叠浪掌。”
<br/>　　“玄阶巅峰武技，霸天三刀。”
<br/>　　“四品天材地宝，雾中花，海心莲，妖身树...”
<br/>　　“四品丹药，御空丹，巨神丹，聚灵丹，残血丹....”
<br/>　　每样宝物，都有一木台放置着，木台前，有该宝物的文字介绍，品阶、功效等等。
<br/>　　萧逸正看得起劲。
<br/>　　忽然，顾长风出现在了他十数米之外，正对着一个执事耳语着什么。
<br/>　　半晌，该执事便陪同顾长风来到萧逸身边。
<br/>　　“喂。”顾长风冷冷地叫了一句。
<br/>　　萧逸没有搭理，自顾地看着各样宝物上的介绍。
<br/>　　“我在叫你，没听到吗？”顾长风脸色变得愤怒。
<br/>　　萧逸仍旧没有搭理。
<br/>　　这时，顾长风身旁的执事，脸色也变得难看起来。
<br/>　　“萧逸，没看到本执事吗？”该执事沉声问道。
<br/>　　“看到了，那又如何？”萧逸不冷不热地回了一句。
<br/>　　他自然是看到这个执事的，不过这个执事既然是跟顾长风一起来的，而且似乎和顾长风关系很不错的样子，想来也不是什么好鸟，自然便不去搭理了。
<br/>　　“萧逸，不必装模作样，快把本公子的风吟剑还回来。”顾长风冷冷地说道。
<br/>　　“为何要还你？现在那是我的东西了。”萧逸头也不回地说着，双眼仍旧盯着木台上的宝物介绍。
<br/>　　“风吟剑乃是我顾家的东西，什么时候成你的了？”顾长风怒声道，“哦，我知道了，像你这种土包子，怕是一辈子都没见过灵器吧。”
<br/>　　“如今，抢了我的灵器，便不愿意归还了。”顾长风满脸不屑之意。
<br/>　　“随你怎么说。”萧逸耸耸肩，走向另一个木台，看向其他宝物。
<br/>　　“给我站住。”那执事冷喝道，“现在，本执事命令你立刻归还风吟剑。”
<br/>　　“凭什么？”萧逸语气开始变得冰冷，一而再再而三地来打扰自己选宝物，他自然不会给好脸色。
<br/>　　不过他也在暗暗腹诽，顾家真不愧是百武城的大家族之一，顾长风竟然能找到一个剑派内的执事为他出头。
<br/>　　只是，这执事不过是洞玄三重修为，想来也不是什么实权人物，萧逸自然也不用给他面子。
<br/>　　“就凭我是执事，而你连普通弟子都不是。”执事霸道地呵斥道。
<br/>　　“呵。”萧逸不屑道，“抱歉，首先，严格来说，我还不是裂天剑派的弟子，我只是来参加考核罢了。”
<br/>　　“所以，你还没有那个权利命令我。再者，别以为我不知道裂天剑派的规矩，就算我成为弟子了，若没犯门规，你也同样无权管我。”
<br/>　　“其次，你什么时候见过武者间战斗，得到的战利品需要归还的？”
<br/>　　那执事脸色一黑，呵斥道，“休要牙尖嘴利，我们裂天剑派乃是武道圣地，不容弟子做那等偷鸡摸狗，抢夺他人东西的卑劣之事。”
<br/>　　“哦，是吗？”萧逸反问道，“我倒想问问执事你，那顾长风在考核中杀了不少人，夺了别人的性命。”
<br/>　　“照你说来，顾长风需要还命吗？”
<br/>　　炎龙大陆，武道为尊。只要拥有足够的实力，便能无视一切，甚至凌驾于皇权之上。
<br/>　　杀戮、抢夺、争斗，所谓的律法、法规，在这个世界，不过是弱者的借口。
<br/>　　拳头、实力，才是这个世界的一切。
<br/>　　“说得好。”忽然，一道淡淡的声音从角落处传来，竟是在角落处惬意看书的易老。
<br/>　　那执事顿时一惊，连忙收敛了刚才的嚣张和霸道。
<br/>　　“你...”那执事咬了咬牙，说不过萧逸，又有易老发言，只能无奈地看了顾长风一眼，而后转身离去。
<br/>　　“三叔。”顾长风顿时一急，说了一声，但马上又闭上了嘴巴。
<br/>　　萧逸顿时恍然，原来是顾家的人，难怪会为顾长风出头。
<br/>　　顾家作为一个家族，有族人在裂天剑派担任执事并不奇怪。
<br/>　　这执事，年轻时也来裂天剑派参加过考核，并通过。但排名并不高，也算不得什么天才。
<br/>　　在裂天剑派内熬了十多年，才算达到洞玄三重修为，成了个执事，但没有什么实权。
<br/>　　“萧逸，你要怎样才肯归还我的风吟剑。”顾长风不得不放低姿态，哀求起来。
<br/>　　后面的考核中，还有战斗，若他没有风吟剑在手，必将实力大减。
<br/>　　而且，后面的考核，也分排名，奖励也更高。
<br/>　　萧逸淡淡道，“很简单，风吟剑现在是我的东西，你想要，便买回来吧。”
<br/>　　“这...”顾长风迟疑道，“你想要多少钱？”
<br/>　　“不多，就五十万两吧。”萧逸淡淡地说道。
<br/>　　“五十万两？你不如去抢好了。”顾长风气急败坏地骂了起来。
<br/>　　“你买便给钱，不买我也无所谓，不差那几十万两。”萧逸耸耸肩，就要离开。
<br/>　　“慢着。”顾长风连忙叫住萧逸，“我...我...”
<br/>　　顾长风思索着，五十万两，可不是个小数目。
<br/>　　但，后面的考核中，那些奖励，可是连顾家这种大家族都不曾拥有的东西，他绝不可能放弃。
<br/>　　若没有风吟剑在手，他实力大减，毫无把握可以得到那些奖励。
<br/>　　一想到那些让他都无比眼馋的奖励，他顿时点了点头，“好，我答应你。”
<br/>　　“但是，我现在没有那么多钱。”顾长风说道。
<br/>　　萧逸脸色一冷，“浪费我时间。”
<br/>　　说罢，转身便走。
<br/>　　“慢着。”顾长风连忙道，“我如今只有三十五万两，剩下的，用物品相抵可行？”
<br/>　　这三十五万两，是他父亲给他用于往后数年修炼时可以购买所需的修炼物品的。
<br/>　　这一下子就没了，他的心在滴血。
<br/>　　“你拿什么东西相抵？”萧逸挑了挑眉毛，问道。
<br/>　　顾长风拿出一个低级乾坤袋，道，“这里有一些三品天材地宝，三级的妖兽内丹和精血，加起来，够十五万两的了。”
<br/>　　这些东西，也是顾家为他准备的日后数年的修炼之物。
<br/>　　“这些东西，我要来无用。”萧逸表面上皱了皱眉头，心里却是笑开了花。
<br/>　　“这样吧。”萧逸满脸嫌弃地说道，“你排名第十，能有一次机会在宝库内选取一样东西。这个机会归我了，加上你那三十五万两，还有这一堆废铜烂铁，勉强抵得上你的风吟剑吧。”
<br/>　　“废铜烂铁？”顾长风咬了咬牙，暗道，那可全都是值钱的东西，但却不敢发作出来，嘴上只得说道，“好，成交。”
<br/>　　萧逸点点头，冷声道，“待我选好物品之后，会去找你交易。”
<br/>　　说罢，转身便走。
<br/>　　顾长风恨恨地盯着萧逸离开的背影，心中阴狠道，“萧逸，给我等着，要不了多久，我会让你十倍奉还。”
<br/>(本章完)
