        “大哥，你来得正好，我有新的证据了！”童筱颖完全不顾及傅夜沉的心情，自顾自地从包包里掏出手机，把自己刚刚录的那段录像放给傅夜沉看。
<br/>
<br/>        傅夜沉此刻压根就没心情去看她的录像，而是一把抓住她的手腕，将她拽到自己跟前。
<br/>
<br/>        童筱颖被迫一头撞在他那结实健硕的胸膛上，仰起头来的一瞬间，她撞入了他那双清冷的黑眸。
<br/>
<br/>        她昨晚突然离开，保镖说她已经回了地利公寓，第二天，他以为她回来上班，结果呢？
<br/>
<br/>        她又背着他，偷偷跑回了临海城。
<br/>
<br/>        童筱颖不懂大哥傅夜沉为什么会如此生气，她甚至认为，他没有任何理由要生她的气。
<br/>
<br/>        她完全不知道他在担心她！
<br/>
<br/>        傅夜沉因为生气，握着童筱颖的手的力度越来越紧。
<br/>
<br/>        “痛！大哥，你放手！”童筱颖凝眉瘪嘴，扭动着手腕想从傅夜沉的手中挣脱。
<br/>
<br/>        傅夜沉不再二话，强行拉着她上了自己的豪华越野车。
<br/>
<br/>        童筱颖一屁股跌坐在后座上，手上又痛，刚刚折腾了那么久，结果何明旭一点事儿也没有，心里顿时觉得委屈，忍不住“啪嗒啪嗒”地掉起了眼泪。
<br/>
<br/>        她昨晚几乎一宿没睡，白天又计划了整整一天，好不容易约到何明旭，本想着诬陷他找小姐，让警察抓，然后害他被学校开除，让他也尝尝马智杰所承受过的痛。
<br/>
<br/>        可结果呢？
<br/>
<br/>        童筱颖顿时觉得自己像个白痴一样，白忙活了这么久。
<br/>
<br/>        傅夜沉一看到她掉眼泪，心口也跟着隐隐作痛。
<br/>
<br/>        为什么她每次伤心，他的心也会跟着痛？
<br/>
<br/>        童筱颖不想再理会傅夜沉，只是自顾自地哭了起来。
<br/>
<br/>        傅夜沉皱了下眉头，俊脸神色紧绷，他从衣兜里掏出一条干净洁白的手帕，侧过身去，一边替这小女人擦拭着眼泪，一边温柔地哄道：“刚刚是我不好，对你太凶，又把你弄疼了。”
<br/>
<br/>        童筱颖瞥了傅夜沉一眼，偏过头去，躲开他的手，不想让他触碰自己。
<br/>
<br/>        但是，她眼眶里的眼泪，仍旧止不住地往外冒。
<br/>
<br/>        这些天来，自从得知马智杰是因她而死后，她就变得特别爱哭了。
<br/>
<br/>        其实，她是心里难受，堵得发慌。
<br/>
<br/>        她好希望昼景能陪在她的身边，但是昼景很忙，忙到无暇顾及她的心情。
<br/>
<br/>        童筱颖一直觉得自己是一个非常坚强的女子，从未被生活上的任何困事打倒过，然而这次……
<br/>
<br/>        她害得一个默默喜欢她，默默为她付出过的男生丢了性命。
<br/>
<br/>        尽管她已经在金钱上做出了补偿，但是，她的内心一直不安，因为她总感觉马智杰死不瞑目。
<br/>
<br/>        现在，只有将何明旭和林若琴绳之以法，她的良心才会好过点。
<br/>
<br/>        傅夜沉见童筱颖的性子倔强，决定跟她妥协。
<br/>
<br/>        “你说，你要收拾谁，我替你去收拾！”傅夜沉深沉地说道。
<br/>
<br/>        童筱颖惊怔地回过头来看着傅夜沉。
<br/>
<br/>        从车窗外透过来的霓虹灯光，洒在他精致而冷硬的俊脸上，柔和了他冷酷的表情。
<br/>
<br/>        “怎么不说话了？”傅夜沉淡淡地问。
<br/>
<br/>        童筱颖再次拿起手机，点开自己录的视频，一边放给傅夜沉看，一边振振有词地说道：“杀害马智杰的真凶，是林若琴，何明旭是帮凶，这是何明旭亲手写下来的！”
<br/>
<br/>        “林若琴？！”傅夜沉惊愕。
<br/>
<br/>        童筱颖抹去眼泪，如小鸡啄米般猛点头。
<br/>
<br/>        傅夜沉顿时剑眉微蹙，薄唇紧抿，脸色也有些不大好看。
<br/>
<br/>        他的反常，让童筱颖产生了一种，他认识林若琴的错觉。
<br/>
<br/>        “大哥，你替我收拾他两！他两下个学期要作为交换生出国留学！我不希望马智杰死得不明不白！”童筱颖哽咽着哀求道，“大哥，算我求求你，好不好？”
<br/>
<br/>        “好！我答应你！”傅夜沉若有所思地开口。
<br/>
<br/>        童筱颖这才破涕为笑。
<br/>
<br/>        “你、马智杰、何明旭、林若琴，四个人之间，到底是什么关系？”傅夜沉突然问道。
<br/>
<br/>        童筱颖也没料到傅夜沉会这么问。
<br/>
<br/>        一时间，她也答不上话来。
<br/>
<br/>        “不想说的话，就算了！”傅夜沉又接着说。
<br/>
<br/>        童筱颖低着头，默不吭声。
<br/>
<br/>        其实，傅夜沉有感觉到，筱颖对马智杰的感情比较特殊，这种特殊又不同于她对阿景的爱。
<br/>
<br/>        是怜悯？是惋惜？是内疚？是亏欠？
<br/>
<br/>        至于那个何明旭，还有林若琴，筱颖似乎在憎恨他两，甚至恨不得他两下地狱。
<br/>
<br/>        傅夜沉只希望这是自己的错觉。
<br/>
<br/>        最近，为了筱颖的事情，他其实，身心上都很疲惫。
<br/>
<br/>        筱颖是他的软肋，这事，他无法否认。
