　　荒天王如果真的没死，还存活了下来的话，现在的一身实力，只怕是会比起少帝，还要可怕。
<br/>　　少帝虽然是被称为大罗天星域之中，最有希望踏入天人境的人物。
<br/>　　可是少帝毕竟是年轻一代的人物。
<br/>　　而荒天王，早在万年之前，就已经是踏入了半步天人境的境界。
<br/>　　经历了当年围杀剑道之主的一战，毁灭剑道门的大战，这一场场血战之中，能够活下来，都对荒天王的实力提升有着极大的帮助。
<br/>　　在经历过万年的沉淀，荒天王的实力绝对会达到一种不可思议的强横地步。
<br/>　　半步天人境的极限，可远远不止是少帝的实力。
<br/>　　之前樊老也曾经说过，少帝的实力，还距离天人境，有些差距。
<br/>　　一个少帝，让现在楚风眠全力之下，都难以招架。
<br/>　　如果说是比起少帝还要更强的荒天王呢？
<br/>　　楚风眠心中刚刚的欣喜，一下子沉淀了下来，楚风眠自己也知道，九域卧虎藏龙。
<br/>　　现在楚风眠踏入了九劫古帝的境界，拥有比拟了半步天人境的实力，其实也只能够是自保而已。
<br/>　　他的实力，还很弱小。
<br/>　　不过，就算是荒天王再强，他也依然是半步天人境的武者，而不是天人境。
<br/>　　只要是半步天人境的武者，那就还不是无敌，如果是真的天人境强者，楚风眠只怕是没有任何一点抗衡的希望
<br/>　　但若是半步天人境的武者，那楚风眠也并非是没有办法抗衡，楚风眠也有着他的底牌。
<br/>　　不管是少帝，还是荒天王，谁敢跟楚风眠血战，楚风眠也能跟他们鱼死网破。
<br/>　　楚风眠心中波涛汹涌，脸色却是不为所动，无比平静的看向荒鎏金，冷笑一身道。
<br/>　　“荒天王吗？很好，今日我倒是要看看，荒天王能不能救得了你！”
<br/>　　话音一落的瞬间，楚风眠突然出手了。
<br/>　　这荒鎏金的话，楚风眠可不会那么愚蠢的全部相信，也许这也是他的狐假虎威之计。
<br/>　　这荒鎏金有恃无恐，楚风眠从他的口中，也问不出多少的东西来了，不如直接杀了此人再说，也可以探探真假。
<br/>　　这荒鎏金可是荒天王最为出色的弟子，如果荒天王真的还活着，绝对不可能坐视不理，让楚风眠轰杀荒鎏金的。
<br/>　　楚风眠突然之间出手了，以雷霆之势，掌心凝聚灵力，向着荒鎏金的头顶猛然拍了过去。
<br/>　　这荒鎏金，已经是被楚风眠镇压，无法催动力量了，跟一个没有任何灵力的废物一样。
<br/>　　楚风眠随意的一掌，就足以轻易轰杀他。
<br/>　　荒鎏金似乎都没有想到，楚风眠居然是说杀就杀，丝毫不顾他的威胁，只见荒鎏金的眼神之中露出几分决断之色，突然间从他的头顶上，一枚玉符飞了出来。
<br/>　　这一枚玉符飞出的一刻，浩瀚的力量，当即降临了。
<br/>　　玉符之中的力量凝聚，就在这荒鎏金的面前，化为了一道身影，这是一名中年男子的样子，但是他的头顶之上，却带着一顶王冠。
<br/>　　虽然看不清其中的模样，但是一股尊贵的气势，便是降临了，宛如一尊王者。
<br/>　　哪怕是一方大陆的主人，都没有这眼前这一道身影身上如此的气势。
<br/>　　这是久居上位，掌控生杀大权的人物，才有的气势。
<br/>　　“荒天王！”
<br/>　　在那一枚玉符凝聚出身影的一刻，楚风眠也是脸色一变。
<br/>　　这荒鎏金，所说的不假，荒天王还没死，存活于世。
<br/>　　楚风眠感觉到，这一尊凝聚出来的身影，跟遥远的时空之中连接这，从遥远的时空之中，输送着灵力，涌入到了这一道身影之中来。
<br/>　　这就证明，炼制这一道玉符的主人，也就是荒天王，还活着。
<br/>　　“看来我还是小看了这小小的剑道门，没想到居然玉符逼我现身出来了。”
<br/>　　这一道身影现身，化为了荒天王的样子，乃是他的一道化身，降临了。
<br/>　　这荒天王的化身，面对这楚风眠轰杀荒鎏金的一掌，大手一挥，一股荒古之力当即凝聚而出，将楚风眠的一掌的灵力所打散。
<br/>　　“师尊，这小子可以凝聚极道剑印！”
<br/>　　那荒鎏金看到这荒天王的一道化身降临出来，大声的喊道，急忙是将这个消息，告诉荒天王。
<br/>　　本来七大宗门之中，都已经认为，所有剑道之主的弟子，应该都陨落了。
<br/>　　所以天下间，应该是无人懂得这极道剑印了。
<br/>　　可现在，居然还有楚风眠这样一个漏网之鱼。
<br/>　　这个秘密，事关重大。
<br/>　　七大宗门毁灭了剑道门，得到了剑道门之中的无数宝藏，但是却没有多少的收获。
<br/>　　其中最为关键的一点，便是不管是剑道门的宝藏，还是曾经剑道门之中的灵器，都是唯有极道剑印，才可以解开其中的封锁。
<br/>　　如果说能够抓住楚风眠，那么那些曾经落到七大宗门之中的剑道门灵器，都可以爆发出全力来，那绝对是无比恐怖的一件事。
<br/>　　任何一方得到这个秘密，都足以实力暴涨。
<br/>　　“极道剑印？”
<br/>　　那荒天王的化身，听到了这荒鎏金大声的喊话，眼神之中都是露出了一抹震惊之色。
<br/>　　显然是他也被这消息所震惊到了，同时一股贪婪，一闪而过。
<br/>　　这极道剑印的重要性，他当然知晓。
<br/>　　“灭！”
<br/>　　荒天王的化身，猛然看向楚风眠，凝聚灵力，一掌对着楚风眠轰击而去。
<br/>　　荒天王这一道化身，所爆发出来的力量，几乎就不亚于真正的一位半步天人境强者。
<br/>　　这还只是荒天王凝聚的一道玉符之中，显现出来的化身，跟真正的荒天王相比，其中的力量不过只是冰山一角罢了。
<br/>　　这一招打出，荒古之力，便是铺天盖地一般的降临了。
<br/>　　这荒天王对荒古之道的参悟，比起荒鎏金要强得多。
<br/>　　哪怕同样都是半步天人境的境界，二人的实力，截然不同。
<br/>　　楚风眠看到这荒天王出手，眼神之中也不敢有着任何的小看之色，哪怕这只是荒天王的一道化身。
<br/>　　双剑齐出，两道耀眼的剑光，同时向着那荒天王斩杀而去。
<br/>　　跟那荒天王化身打出的一掌，在这云羽天宫之中猛然碰撞起来。
