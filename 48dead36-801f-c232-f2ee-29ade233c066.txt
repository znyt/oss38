        &lt;h3&gt;第四卷第七章  闫欢和谷正娟合作&lt;/h3&gt;
<br/>
<br/>        闫欢和谷正娟开始合作了。
<br/>
<br/>        谷正娟按照闫欢的要求开始进行楼宇装修，闫欢则进行招商，前期手中有客户资源，可以说是轻车熟路，她只需要给她的那些供货的厂家打电话，进行沟通，那些厂家已经和他们有过合作，对他们的营销策略是认同的，闫欢打电话说了要拓展新的地方，很多厂家都愿意和他们再次合作，闫欢和陈子悦是从底层做起，更能了解做企业的艰难，无论是什么品牌都一视同仁，所以闫欢打电话联系供货商的时候，大家愿意和闫欢一起共进退。
<br/>
<br/>        一个礼拜闫欢就将西城的所有供货商敲定全部虽她一起到甸城开发新的市场，闫欢都没有想到，这些人连考察都不考察，便答应随她一起前来，有些人还要给她打预付款，有些人直接告诉她，在装修的同时过来看看，看好地方一起进行装修。
<br/>
<br/>        最让闫欢为难的还是吃的问题，每个地方每一种口味，每个地方饮食文化都不一样，她跑了几个县区，将各地的特色美食收集了一些，但是那些还是远远不够。
<br/>
<br/>        衣服和奢饰品这些东西都好说，甸城的消费水平也不错，但是各地的饮食不同，不知道大家愿不愿意来。
<br/>
<br/>        谷正娟眼见着闫欢打了一段时间的电话，就搞定了招商，她对闫欢充满了好奇和崇拜，没有想到闫欢这么能干，想一想如果闫欢那个时候不被陆玉陷害，现在的闫欢又是何等的成就？
<br/>
<br/>        这个世界真是小，山不转路转，陆玉怎么也不会想到，现在的闫欢也会到甸城，而且还是以一个投资者的身份出现。
<br/>
<br/>        “闫总，我们进驻了十一个装修工程队，每个工程队负责一个楼层，按照秦总的吩咐，装修会在二十天完成。”谷正娟拿着装修方案来找闫欢。
<br/>
<br/>        闫欢还是住在闫芈的家里。
<br/>
<br/>        “都说甸城的谷家办事情雷厉风行，是云深之处的大神，果然不错，谁能同时调齐十一个装修队同时开工啊，你们的速度之快真是让人望尘莫及啊。”闫欢感叹道。
<br/>
<br/>        谷家真的是不容小觑呢。
<br/>
<br/>        “这些装修队基本都是我们自己的产业，我们手低下这样的装修队很多，在全国都有，这些人其实工作都很灵活，有活干的时候就抽调回来，没有活干他们就自己找工作。”
<br/>
<br/>        “那如何能保证他们随叫随到呢？”
<br/>
<br/>        “他们是由我四叔管理，我四叔有自己的一套管理方式，我也不知道他们是怎么管理的。”
<br/>
<br/>        “你这一个装修队有多少个人？”
<br/>
<br/>        “有设计、水、点、泥瓦、木工、杂工，一个队大概14个人，主要是他们一个团队除了设计外，他们都是全能，他们可以同时兼几个工种。”
<br/>
<br/>        “这么厉害？”
<br/>
<br/>        “是的，每个装修队除了设计之外，离开谁都可以干活。”
<br/>
<br/>        “那光你们装修队这个团队大概就几百好人了啊。”
<br/>
<br/>        “应该比几百号人还要多。”
<br/>
<br/>        “那你现在手中的装修队都是你们当地的吗？”
<br/>
<br/>        “有三个是从外地抽调回来的。”
<br/>
<br/>        “你们家的底蕴真的是好厚！”闫欢感叹着。
<br/>
<br/>        “还好吧，毕竟我爸爸他们兄弟几个穷其一生都在做事情。”谷正娟微微一笑。
<br/>
<br/>        “你爸爸真是个好人，外界都传闻谷家父子神龙不见首位，低调奢华大气！”闫欢笑着赞美了一下谷家。
<br/>
<br/>        “哈哈哈，我还是是觉得他们眼界不够宽，不像你们这些商人，能赶上时代的潮流，总是固步自封躲在自己的舒服圈子里不愿意走出来，其实说来说去，我们谷家这十几年来发展特别快还是因为谷强的原因，谷强能够大胆开拓，勇于创新。”
<br/>
<br/>        “你也不错啊，能够抓住机遇，利用自己的优势，敢于尝试新事物。”闫欢这句话说的是真心话，要是谷正娟不想和她合作，她还真的耗费一段时间，事情就不会像现在这样事半功倍了。
<br/>
<br/>        这一切都是因为谷正娟有一颗灵活的商业头脑，能够看到未来的发展，会摒弃个人的好恶。
<br/>
<br/>        “我只是一个女人，说真的，其实都是靠着谷强在支撑，所有的事情都是他在做，明的暗的全部都是他在操持，我只是一个看起来很光鲜的人。”谷正娟喝着茶悄悄的观察着闫欢的反应。
<br/>
<br/>        如果闫欢在甸村呆过的话肯定是知道谷强是干什么的，谷正娟想知道闫欢了解不了解谷强，知不知道谷家。
<br/>
<br/>        “明的暗的？哈哈哈，你哥哥还真是有意思，你们谷家不就你们兄妹俩个人吗？还有什么要背着你干的？”
<br/>
<br/>        “我毕竟是女孩子嘛，终究是要嫁人的。”
<br/>
<br/>        “你这话说得，我似乎看见了豪门恩怨。”
<br/>
<br/>        “闫总，你以前到过甸城吗？”
<br/>
<br/>        “没有，第一次来。”
<br/>
<br/>        “那你怎么会想到来甸城投资呢？”
<br/>
<br/>        “我妹妹不是在这里吗？她一个人孤孤单单的，我们反正是想在全国各地发展，她过年的时候回家了，我就想着到这里来投资，算是给她一个靠山吧，最起码不那么孤单！”
<br/>
<br/>        “你这个靠山不简单，用亿砸出来的。”
<br/>
<br/>        “我在这里也呆不了多久，这边弄好了我就得走，我要去别的地方拓展业务，最后这里可能就交给她来管理，到时候怕是要你辛苦多一些了，你比她了解很多事情，希望到时候你能够多承担一下。”
<br/>
<br/>        “你要走？”谷正娟吃惊的问。
<br/>
<br/>        “是啊，我不可能每个地方都会停留的啊，我们的模式就是投资，寻找合适的管理人员来管理，这里有你我就不用费尽心思来寻找管理人员了，直接就是合伙人。”
<br/>
<br/>        “你们在聊什么呢？”闫芈从外面走回来。
<br/>
<br/>        “我们在聊生意呢。你怎么现在就回来了？”闫欢怕谷正娟说多了，直接对闫芈说。
<br/>
<br/>        “下午没有太多的事情，我就提前下班了，正娟你怎么有时间过来？”
<br/>
<br/>        “我们的大老总在这里，我是来这里汇报工作呢。”谷正娟笑着说。
<br/>
<br/>        “你别说笑了，我姐姐在你们谷家眼里就是一粒小芝麻。”
<br/>
<br/>        “管它大西瓜小芝麻抓在手心里才是正道，我要和闫总一起开拓新的航线，要他们刮目相看！”谷正娟笑着说，闫芈一回来她感觉轻松了很多。
<br/>
<br/>        “你这是碰到知音了？”闫芈看了一眼闫欢笑眯眯的对谷正娟说。
